Pardus 2008.2 Release Notes

1. Welcome to Pardus 2008.2!

Welcome to Pardus version 2008.2, another great Pardus release from Pardus developers, translators, artists and documentation writers. This is the second minor release of Pardus 2008 branch.
We kindly invite you to visit http://www.pardus.org.tr/eng for more information.

1.1 Feedback

You can use http://bugs.pardus.org.tr, aka Pardus bug tracking system to send a bug, bug fix or any suggestion related to Pardus. We would like to thank our users and developers with their bug reports, related fixes, new package or feature suggestions for this release.

2. Basic Features

Since the latest minor release 2008.1, more than 120 bug fixes and 60 security fixes were commited to 663 packages which are included in the installation CD. In the same period, 26 new packages were added to the CD to support new featues and hardware, and a total of 177 (out of 663) packages were updated.

2.1. Updates in Pardus Technologies

With the release of Pardus 2008.2, the core technologies like PiSi and COMAR are updated to support more features and an updated YALI is used for our users to have a more smooth installation experience. While the fixes in Mudur subsytem provides a faster and more ruboust startup process, remote file system support is enchanced as well.
  • Disc Manager - Pardus 2008.2 features an complete new tool for fixed storage management.
  • Network Manager - Pardus' own tool for managing connections is updated and now includes a bunch of new features as well :
    - Flexible DNS settings,
    - Fixes on dhcp related timeout issues,
    - problems caused by weird ESSIDs are fixed,
    - support for script execution on connect and disconnect.
  • Display Manager - Pardus 2008.2 includes the latest drivrs for new generation Nvidia, ATI and Intel based display adapters. Pardus' own Display manager and COMAR based auto-configure subsytem zorg are updated to make use of this new drivers and features.
  • Package Manager - The Package Manager in Pardus 2008.2 includes a bunch of updates as well :
    - Improved package search function,
    - Support for handling of download errors and retries mostly caused by proxies and such,
    - GUI and translation updates,
    - Delta package and automatic updates of reverse dependencies - these features are planned to be supported after the release of 2008.2

2.2. KDE 3 and KDE 4 Desktop Enviroments

  • KDE 3 - KDE3, the default desktop enviroment of Pardus 2008.2, is updated and enchanced with Pardus spesific features as well. These updates are based on latest stable release of KDE 3, 3.5.10. Pardus 2008.2 Ä°nstallation CD's and KDE 3 based Live CD includes this KDE 3 platform as default.
  • KDE 4 - New member of KDE family, KDE 4, is the choice of many Pardus users for a while. Pardus 2008.2 inclueds latest stable release of KDE 4, 4.2 - which has been released a few days ago. Users who want use this new desktop enviroment can install and use KDE 4.2 in Pardus with the release of 2008.2. The KDE 4 based Live CD includes this new desktop enviroment and the KDE 4 ports of popular KDE applications as well.

2.3. Two Variants of Install CDs

Pardus is offered as two variants to support ever-increasing number of languages : one install CD with Turkish and English support only (and a selection of additional development tools and games), and an internationalized install CD with support for all 11 languages. As always, users can install and use additional language support packages from Pardus' repositories, regardless of which install CD they've used to install their systems.

2.4. Updated Components

Pardus 2008.2 includes a lot of package updates besides Pardus Technologies and KDE components. Some of the highlights are as follows:
  • Pardus Linux Kernel in 2008.2 is based on the latest stable release of linux-2.6.25, 2.6.25.20. Pardus kernel also includes a lot of hardware driver and subsytem updates, stability and security fixes. Some of these new features are :
    - Fixes and enchanced support for network devices based on RTL818x, Marvell Yukon 88E8040T and Ralink rt2x00 chipsets,
    - IR controller for bt8xx based TV cards,
    - Fixes and enchanced support for Syntek DC-1125 based webcams,
    - Fixes for Asus eeePC, Apple Macbook Pro and Gigabyte M9xx,
    - Support for Broadcom BCM4311, BCM4312, BCM4321 and BCM4322 based wireless devices,
    - Support for Intel WiFi Link 5000 based wireless devices,
    - Support for rtl8187 based wireless devices,
    - Support for RICOH Bay1Controller RICOH SD card readers,
    - ALSA 1.0.18a_20081203 containing support a bunch of new soundcards, bugfixes and new features,
    - Support for splash images in different resolutions,
    - Support for init images with virtio and xen modules.
  • Also with Mozilla Firefox 3.0.6, OpenOffice.org 2.4.1.10 and Mplayer 20090101, updates to popular applications like KDE PIM, vim, Gimp, Frozen Bubble, Pardus 2008.2 is more stable, secure and user friendly as ever.

For more informaiton about updates and new features on Pardus 2008.2, you can visit www.pardus.org.tr/changelog2008.2/ web site.

3. Installation requirements

You can run Pardus 2008.2 on an average hardware. We suggest to use at least 512 MB RAM (memory) and 1400 MHz processor to get an effective performance. Minimum and recommended hardware requirements for Pardus 2008.2 are mentioned below:
  • 384 MB memory (512 MB recommended)
  • 1000 MHz Intel or AMD processor (1400 MHz recommended)
  • At least 4 GB of free hard disk space (10 GB recommended)
    • * We recommend a system with a minimum of 1 GB memory for KDE 4 Based Live CD.
