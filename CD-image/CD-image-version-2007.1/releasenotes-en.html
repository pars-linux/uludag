Pardus 2007.1 release notes

1. Welcome to Pardus 2007.1

We would like to welcome you to Pardus version 2007.1, a maintanence release for Pardus 2007 featuring easy to install and use operating system with providing many useful, compatible and secure development over K Desktop Environment (KDE). With Pardus, you can connect to Internet, read your e-mail messages, securely communicate with your friends and exploit your computer's multimedia capability while listening to music and watching movies. We kindly invite you to visit www.pardus.org.tr/eng to discover how Pardus 2007.1 can answer your computing needs.

1.1 Feedback

You can use bugs.pardus.org.tr, aka Pardus bug tracking system to send a bug, bug fix or any suggestion related to Pardus. We would like to thank you for your kind interest and support with reporting a problem, suggesting a new packet or requesting a feature for the next release.

Also, you would use the feedback application that comes with Pardus 2007.1 to send your ideas about Pardus over Internet.

2. Basic features

This section describes the features Pardus 2007.1 introduces.

2.1 Desktop usability (KDE 3.5.6)

Pardus 2007.1 introduces KDE 3.5.6 for better stability, translations and eye candy for Pardus users.

2.2 WPA support for Network Manager

Network manager application now comes bundled with WPA support, showing the status of Wifi networks with a nice graphical interface. This was one of the most wanted features after Pardus 2007.

2.3 Improved package management system

2.4 More hardware recognition

2.5 Package updates

2.6 More localization support

Pardus 2007 had support for English, Turkish, German and Dutch from the installation through the desktop. Now, Pardus 2007.1 also has support for Brazilian Portuguese from ground up.

Support for more languages will follow after Pardus 2007.1

3. Installation requirements

You can run the Pardus on an average hardware. We suggest you at least 512 MB RAM (memory) and 1200 MHz processor to get an effective performance. The basic hardware components that provide the Pardus to run in a comfortable environment are mentioned below:
  • 256 MB memory (512 MB recommended)
  • 800 MHz Intel or AMD processor (1200 MHz recommended)
  • At least 4 GB of free hard disk space (10 GB recommended)
