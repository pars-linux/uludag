Pardus 2007.1 release notes

1. Welcome to Pardus 2007.1

We would like to welcome you to Pardus version 2007.1, a maintanence release for Pardus 2007 featuring easy to install and use operating system with providing many useful, compatible and secure development over K Desktop Environment (KDE). With Pardus, you can connect to Internet, read your e-mail messages, securely communicate with your friends and exploit your computer's multimedia capability while listening to music and watching movies. We kindly invite you to visit www.pardus.org.tr/eng to discover how Pardus 2007.1 can answer your computing needs.

1.1 Feedback

You can use bugs.pardus.org.tr, a.k.a Pardus bug tracking system to send a bug, bug fix or any suggestion related to Pardus. We would like to thank our users and developers with their bug reports, related fixes, new package or feature suggestions for this release.

Also, you may use the feedback application that comes with Pardus 2007.1 to send your ideas about Pardus over Internet.

2. Basic features

This section describes the features Pardus 2007.1 introduces. Pardus 2007.1 is mainly a bugfix release, translation update and minor extensions to Pardus 2007.

2.1 Desktop usability (KDE 3.5.6)

Pardus 2007.1 now introduces KDE 3.5.6 for better stability, translations and eye candy for Pardus users. This package is also available for Pardus 2007 users through Pardus package manager.

2.2 WPA support for Network Manager

Network manager application now comes bundled with WPA support, showing the status of Wifi networks with a nice graphical interface. This was one of the most wanted features after Pardus 2007.

2.3 Improved package management system

PISI, the Pardus package manager has reached version 1.1.1, introducing more effective searches and a 40% + speed improvement in update repository phase. Now users will benefit from the speed of repository updates and case insensitive search in package name, summary and description information.

2.4 Package updates

Lots of security updates in Pardus 2007 branch have been merged in 2007.1, including mplayer, spamasassin, clamav, ImageMagick, nexuiz, wireshark, kernel and others. In addition to this, more than 280 packages have been updated and corresponding updates have been included in 2007.1

2.5 More localization support

Pardus 2007 had support for English, Turkish, German and Dutch from the installation through the desktop. Now, Pardus 2007.1 has additional support for Brazilian Portuguese from ground up.

Italian and French language support is underway, and we kindly invite you to support Pardus for your language in upcoming versions.

3. Installation requirements

You can run the Pardus on an average hardware. We suggest you at least 512 MB RAM (memory) and 1200 MHz processor to get an effective performance. Minimum and recommended hardware requirements for Pardus 2007.1 are mentioned below:
  • 256 MB memory (512 MB recommended)
  • 800 MHz Intel or AMD processor (1200 MHz recommended)
  • At least 4 GB of free hard disk space (10 GB recommended)
