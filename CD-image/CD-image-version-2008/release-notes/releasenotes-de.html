Pardus 2008 Release-Information

1. Willkommen zu Pardus 2008

Willkommen zu Pardus Version 2008, eine weitere grossartigea Ausgabe von Pardus, zusammengestellt unter Mitwirkung von Entwicklern, Übersetzern, Designern und Autoren der technischen Beschreibungen. Dieses ist die erste Ausgabe der Serie Pardus 2008.

Für weitere Informationen besuchen Sie bitte http://www.pardus.org.tr/eng .

1.1 Feedback

Sie können http://bugs.pardus.org.tr, alias Fehler-Verfolgungs System (bug tracking) benutzen, um einen Fehler, eine Fehlerlösung oder Verbesserungsvorschläge im Zusammenhang mit Pardus zu melden. Wir bedanken uns an dieser Stelle bei den Benutzern und Entwicklern, für die ihre vielfältigen Meldungen, Lösungen und Vorschlägen die zu dieser aktuellen Ausgabe beigetragen haben.

Ausserdem steht Ihnen ein Feedback-Programm zur Verfügung, das Sie im Menu finden können, mit dem Sie Ihre Anregungen und Kommentare über Pardus per Internet abschicken können.

2. Haupteigenschaften

Dieser Abschnitt beschreibt die Eigenschaften von Pardus 2008.

2.1 Netzwerk-Manager aktualisisert

Der Netzwerk-Manager unterstützt die automatische Erkennung von drahtlosen Profilen und ihre Aktivierung seit der Version 2007.3 von Pardus. Ausserdem zeigt er mit einer Pop-up Nachricht Informationen und ist intuitiv und einfach zu bedienen. Mit Pardus 2008 können sie jetzt einen optimisierten und Fehler-bereinigten Netzwerk-Manager benutzen.

2.2 Verbessertes Paket-Management System

Der Paket-Manager ist weiter perfektioniert worden. Viele Fehler aus dem Bug-Tracker sind bereinigt worden. Delta-Depos werden für die Ausgabe Pardus 2008 zur Verfügung stehen, so dass sich Aktualisierungen schneller durchführen lassen.

2.3 YALI

YALI (yet another Linux installer) ist neu überarbeitet worden, mit der Entwicklungsbibliothek qt4 und einem polierten Äusserem. Jetzt können Sie die Einstellungen der Zeitzone, Konfigurierung von Auto-Login, und zusätzliche Informationen eingeben, bevor die eigentliche Installation startet. Sie sparen Ihre Zeit...

Erwähnenswert ist auch die Eigenschaft des Installationsprogrammes YALI, das automatisch erkennen kann, ob eine Festplatte oder Partition startbar ist (boot), und kann eigenständig entscheiden, auf welche der Festplatten eine Installation von Pardus am geeignetesten wäre. Pardus war die erste Distribution Linux mit dieser Funktion.

2.4 Aktualisierte Programme

KDE, der bevorzugte Fenstermanager von Pardus, wurde auf die Version 3.5.9 aktualisiert.
Die Büro-Suite OpenOffice.org ist jetzt in der Version 2.4.

3. Installationsanforderungen

Wir empfehlen einen Haupspeicher mit mindestens 512 MB RAM und Prozessor mit 1200 MHz für ein angenehmes Arbeiten.
Im folgenden die mindest- und empfohlenen Hardware-Daten für Pardus 2008:
  • Hauptspeicher: 256 MB (512 MB empfohlen)
  • Prozessor: 800 MHz Intel oder AMD (1200 MHz empfohlen)
  • Festplatte: Partition mit mindestens 4 GB (10 GB empfohlen)
