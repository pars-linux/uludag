Notas de Lanzamiento de Pardus 2007.3

1. Bienvenido a Pardus 2007.3

Bienvenido a Pardus versión 2007.3, otra estupenda versión por los desarrolladores de Pardus, traductores, artistas y editores de documentación. Esta versión contiene nuevas herramientas y software actualizado.

Le invitamos cordialmente a visitar www.pardus.org.tr/eng para más informaciones.

1.1 Notificación - Feedback

Puede utilizar la página web bugs.pardus.org.tr para enviar un reporte de falla, una corrección o cualquier sugerencia relacionada con Pardus. Nos gustaría agradecer a nuestros usuarios y desarrolladores por su envío de reportes de fallas, correcciones y sugerencias para esta versión.

También puede utilizar el aplicativo de feedback, disponible en el menú, para enviar sus ideas y opiniones sobre Pardus vía internet.

2. Características Principales

Esta sección describe las caracteristicas de la versión 2007.3.

2.1 Administrador de Red actualizado

Administrador de Red puede detectar los propios perfiles wireless y conmutar automaticamente a uno de ellos. También da información vía una ventana popup desde la barra de tareas, y por lo tanto es más intuitivo y amigable.

2.2 Sistema de administración de paquetes mejorado

Administrador de Paquetes está más pulido ahora. Muchos de los errores reportados fueron corregidos.

2.3 YALI

YALI (yet another Linux installer) ahora hace las preguntas al principio, y luego realiza la instalación. De esta forma el proceso resulta más flexible y consume menos tiempo. El usuario tambien puede decidir si quiere ingresar automáticamente al escritorio (autologin). También se incluyeron definiciones de nuevos discos rígidos, para simplificar el proceso de instalación.

Además YALI puede determinar si un disco es booteable o no y decidir en cual disco se va a instalar. Pardus es la primera distribución de Linux con esta facilidad.

2.4 Aplicaciones Actualizadas

KDE, el entorno de escritorio primario en Pardus 2007.3 fue actualizado a la versión 3.5.8. La suite de oficina OpenOffice.org está ahora en la versión 2.3. K3b, la aplicación para quemar CD y DVD fue actualizada a la versión 1.0.4.

3. Requerimientos de Instalación

Se puede ejecutar Pardus 2007.3 en un equipo con hardware promedio. Sugerimos al menos 512 MB de memoria RAM y procesador de 1200 MHz para un rendimiento aceptable. Requerimientos de hardware mínimos y recomendados para Pardus 2007.3 son:
  • Memoria RAM 256 MB (512MB recomendado)
  • Procesador 800 MHz Intel o AMD (1200 MHz recomendado)
  • Al menos 4 GB de espacio en disco rígido (10 GB recomendado)
