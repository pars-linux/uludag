Notas de lanzamiento de Pardus 2007.2

1. Bienvenido a Pardus 2007.2

Bienvenido a Pardus versión 2007.2, un lanzamiento de mantenimiento para Pardus 2007, luego de Pardus 2007.1, proveyendo un sistema operativo fácil de instalar y fácil de usar con muchos desarrollos útiles, seguros y compatibles para el entorno de escritorio K (KDE).
Aqui describimos las diferencias principales entre Pardus 2007.1 y 2007.2. Le invitamos cordialmente a visitar www.pardus.org.tr/eng por más informaciones.

1.1 Feedback

Puede utilizar la página web bugs.pardus.org.tr, alias sistema de reporte de fallas de Pardus, para enviar un reporte de falla, una corrección, o cualquier sugerencia relacionada con Pardus. Nos gustaría agradecer a nuestros usuarios y desarrolladores por su envío de reportes de fallas, correcciones relacionadas y sugerencias para funcionalidades del lanzamiento actual.

Además puede utilizar el aplicativo de feedback, disponible en el menú, para enviar sus ideas y opiniones sobre Pardus a través de la internet.

2. Características Principales

Esta sección describe las caracteristicas nuevas incluidos en Pardus 2007.2. Pardus 2007.2 es principalmente un lanzamiento de correcciónes, actualización de traducciones y extensiones menores de Pardus 2007.1.

2.1 Escritorio actualizado (KDE 3.5.7)

Pardus 2007.2 ahora introduce KDE 3.5.7 por su mejor estabilidad, traducción y efectos visuales para los usuarios de Pardus. Este paquete tambien está disponible para Pardus 2007 y Pardus 2007.1 mediante el administrador de paquetes de Pardus.

2.2 Soporte PEAP-MSCHAPv2 para Administrador de Red

Administrador de red ahora incluye soporte PEAP-MSCHAPv2. Clientes Pardus ahora pueden autenticarse con dispositivos inalambricos usando este protocolo, aprovechando las facilidades de encriptación fuerte.

2.3 Mejorado Administrador de Paquetes

Algunos usuarios reclamaron la lentidud de busqueda del administrador de paquetes. Ahora pisi está utilizando un método nuevo de búsqueda para buscar en los nombres y descripciones. Como resultado las busquedas de paquetes responden de manera más ágil y confiable.

2.4 Actualización de Paquetes

El repositorio de Pardus 2007.2 contiene una cantidad creciente de programas, con 1728 paquetes estables y probados para este lanzamiento. Además los usuarios de Pardus pueden descargar más de 650 paquetes del repositorio contrib.

2.5 Administrador de Arranque

Pardus 2007.2 introduce un nuevo administrador de arranque, con el cual se puede agregar, modificar o eliminar entradas al GRUB, usado para el proceso del arranque del ordenador.

2.6 Soporte autologin en YALI

Muchos usuarios solicitaron una manera rápida para iniciar el escritorio Pardus, y lo implementamos. YALI (Yet Another Linux Installer) ahora ofrece soporte de autologin durante la instalación. Quiere decir, durante la instalación se puede elegir al usuario que al iniciar el sistema se logea por defecto al escritorio KDE.

2.7 Actualizaciones y correcciones

Soporte Jmicron se agregó a Pardus 2007.2. Usuarios con controladores Jmicron podrán ahora instalar Pardus 2007.2 sin complicaciones.

3. Requerimientos de instalación

Se puede ejecutar Pardus 2007.2 en un equipo con hardware promedio. Sugerimos al menos 512 MB RAM (memoria) y 1200 MHz (procesador) para un rendimiento aceptable. Requerimientos de hardware mínimos y recomendados para Pardus 2007.2 son:
  • 256 MB memoria (512 MB recomendado)
  • Procesador 800 MHz Intel o AMD (1200 MHz recomendado)
  • Al menos 4 GB de espacio en disco rígido (10 GB recomendado)

4. Contacto

Por favor, visite la página web de Pardus (www.pardus.org.tr/eng) por más información.
