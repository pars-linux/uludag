Pardus 2007.2 Release-Informationen

1. Willkommen zu Pardus 2007.2

Wir begrüssen Sie herzlich zu der Pardus 2007.2 Version, ein Wartungsrelease-nach Pardus 2007.1-zu Pardus 2007, dass einfach zu installierendes, zu verwendbares Betriebssystem und mit vielen nützlichen, kompatiblen und sicheren Etwicklungen durch K Desktop Environment (KDE) ausgestattet ist. Dieses Dokument beschreibt die grundlegenden Unterschiede zwischen Pardus 2007.1 und 2007.2. Für weitere Informationen, bitten wir Sie die www.pardus.org.tr/eng Seite zu besuchen.

1.1 Feedback

Sie können bugs.pardus.org.tr alias Pardus Programmfehler-Meldungssystem verwenden, um uns bei evtl. auftretenden Fehlern zu benachrichtigen, bei Fehlerlösungen oder Anregungen&Vorschlägen zu Pardus mitzuteilen. Wir bedanken uns im Voraus bei unseren Benutzern und Entwicklern für Ihre Bugmeldungen, Lösungen, neuen Paketen oder zusätzlichen Vorschlägen für diese Release.

Sie können auch die Feedback-Anwendung, dass mit Pardus 2007.2 erscheint, verwenden, um Ihre Ideen über Pardus über das Internet senden.

2. Haupt-Eigenschaften

Dieser Abschnitt beschreibt die Eigenschaften von Pardus 2007.2. Pardus 2007.2 ist hauptsächlich ein Bugfix-Release, Übersetzungs-Update und mit geringen Systemerweiterungen im Vergleich zu Pardus 2007.1.

2.1 Aktualisiertes KDE 3.5.7)

Pardus 2007.2 führt nun KDE 3.5.7 für eine verbesserte Stabilität, Übersetzungen und Blickfängene für die Pardus-Benutzer. Dieses Paket ist auch für die Pardus 2007.1-Benutzer durch den Pardus Paket-Manager verfügbar.

2.2 PEAP-MSCHAPv2-Unterstützung für den Netzwerk-Manager

Netzwerk-Manager-Anwendung erscheint nun gebündelt mit einer PEAP-MSCHAPv2-Unterstützung. Pardus-clients können sich nun mit Drahtlosen-Geräten mittels dieses Protokolls authentifizieren, das vorteilhafte sichere Verschlüsselungs-Möglichkeiten hat.

2.3 Verbessertes Paketmanagementsystem

Some users have complained regarding the speed of search facility in packager manager. Now pisi benefits from a new search method for package names and descriptions. As a result, package searchs are more faster and reliable.

2.4 Paket-Updates

Pardus 2007.2 repository has an ever-increasing number of software, containing 1728 tested and stable packages in this release. Moreover, Pardus users can enjoy to download more than 650 packages by adding contrib repository to Package Manager.

2.5 Boot-Manager

Pardus 2007.2 führt einen neuen Boot-Manager ein, welcher zum hinzufügen, ändern oder zum löschen der GRUB-Einträge, die man während des Boot-Vorganges sieht, verwendet werden kann.

2.6 Auto-Login-Support bei YALI

Many users have been asking us a way to quickly start Pardus desktop, and we have implemented it. YALI (Yet Another Linux Installer) now includes autologin support during the installation. That means, you can choose a user which can automatically login to the KDE desktop as system boots.

2.7 Updates and fixes

Jmicron-Unterstützung has been added to Pardus 2007.2. Those having Jmicron controllers are now able to install Pardus 2007.2 without any hassle.

3.Installations-Anforderungen

Für die Installation und Verwendung wird ein Rechner mit mindestens 512 MB RAM (Arbeitsspeicher) und ein Prozessor mit 1200 MHz empfohlen. Minimale und empfohlene Hardwarekomponente für Pardus 2007.2 sind :
  • 256 MB Arbeitsspeicher (empfohlen wird 512 MB)
  • 800 MHz Intel oder AMD Prozessor (empfohlen wird mindestens 1200 MHz)
  • 4 GB freien Festplattenspeicher (10 GB wird empfohlen)
