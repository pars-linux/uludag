Netzwerk-Manager

Netzwerk-Manager kann verwendet werden, um die Netzwerkeinstellungen zu konfigurieren, damit mit dem Internet, lokalen-, kabellosen Netzwerk oder Dial-up Diensten verbunden werden kann. Er hat eine benutzerfreundliche Schnittstelle, der es ermöglicht Sie überall in Sekunden zu verbinden.

Wenn das Programm zum ersten mal ausgeführt wird, dann müssen Sie den Verbindungstyp auswählen und die Einstellungen konfigurieren. Nachfolgend können Sie die Schemen der Verbindungstypen sehen.

Verbindung mit einer Ethernet-Karte

Pardus-Linux sollte bereits Ihre Ethernet-Karte erkannt haben, damit mit dem Ethernet-Netzwerk verbunden werden kann. Führen Sie diese Schritte aus, damit Sie sich für dieses Verfahren verbinden können.

  1. Führen Sie den Netzwerk-Manager aus
  2. Klicken Sie auf "Erstellen"
  3. Wählen Sie das "Ethernet-Netzwerk"
  4. Klicken Sie auf "Verbindung erstellen"
  5. Geben Sie Ihren Verbindungsnamen ein und wählen Sie die Ethernet-Karte aus
  6. Wählen Sie DHCP oder geben Ihre IP-Konfiguration ein
  7. Klicken Sie auf "Anwenden" und verlassen Sie dieses Fenster
  8. Klicken Sie auf "Verbinden"
Sie werden sofort mit dem Netzwerk verbunden sein und ein grüner Ikon wird erscheinen.

Für Pardus-Benutzer mit ADSL oder Kabelmodem sollte es ausreichen DHCP auszuwählen.

Wenn Sie auf "Allgemeine Einstellungen" klicken, können Sie

  1. den Namen von Ihrem Computer oder
  2. einen DNS-Server auswählen.

In einer Netzwerkumgebung werden generell IP's automatisch vergeben (z.B. via DHCP). Es ist ausreichend DHCP auszuwählen, damit Computer ihre IP bekommen und mit dem Internet verbinden können.

Verbindung mit einer drahtlosen Karte

Die drahtlose Verbindung ist genauso wie Ethernet-Verbindung. Der bedeutendste Unterschied ist The most important difference der Name des drahtlosen Netzwerks, wie z.B. die ESSID. Manche Netzwerke erfordern diese Namen auszuwählen.

  1. Führen Sie den Netzwerk-Manager aus
  2. Klicken Sie auf "Erstellen"
  3. Klicken Sie auf "Drahtloses Netzwerk" und "Verbindung erstellen"
  4. Geben Sie Ihren Verbindungsnamen ein, wählen Sie die drahtlose Karte und ESSID aus.

Sie können ebenso auf "Suchen" klicken und eine Liste von drahtlosen Netzwerken kriegen, die derzeit von Ihrem System gesehen werden.

Verbindung mit einem Modem

Führen Sie diese Schritte aus, damit Sie sich mit einem Modem verbinden können. Bitte beachten Sie dass einige Modems speziell für andere Betriebssysteme konzipiert wurden und unter Linux nicht funktionieren könnten.

  1. Führen Sie den Netzwerk-Manager aus
  2. Klicken Sie auf "Erstellen"
  3. Klicken Sie auf "Drahtloses Netzwerk" und "Verbindung erstellen"
  4. Geben Sie den Verbindungsnamen, den von dem Modem verwendeten seriellen Port (in der Regel COM1) und die Telefonnummer von Ihrem Dienst Provider ein.
  5. Klicken Sie auf den Tab "Authentifizierung"
  6. Geben Sie Ihren Benutzernamen und Ihr Passwort ein

Klicken Sie nun auf "Verbinden" und Ihr Modem wird dadurch aktiviert
