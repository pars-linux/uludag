Kiadási megjegyzések

1. Welcome to Pardus 2011.1 Dama Dama

Welcome to Pardus 2011.1 Dama Dama, the latest maintenance release of Pardus, a powerful, KDE-focused Linux distribution.

1.1. Mi a Pardus?

A Pardus egy nyílt forrású, ingyenesen terjeszthető, GPL licenszelésű barátságod GNU/Linux disztribúció számos hardver kompatibilitási, stabilitási, biztonsági és felhasználhatósági fejlesztéssel.
A Pardus project támogatója a törökországi Informatikai Technológia Fejlesztési és Információs Biztonságtechnikai Kutatóközpont (BILGEM), a Törökországi Tudományos és Technológiai Kutatótanács (TÜBİTAK) tagja.
A Pardus célja, hogy az emberek számára egy stabil, megbízható, könnyen használható és funkciógazdag számítógép felhasználási élményt nyújtson nyílt forrású szoftverek segítségével. A fejlesztők jelentős erőfeszítést tesznek azért, hogy a Pardus a lehető legtöbb hardveren tudjon működni.
A prodzsekt nevét az Anatóliai leopárd latin nevéről kapta, Panthera pardus tulliana.

1.2. Hogyan próbálhatom ki a Pardus-t?

Kipróbálhatja a Pardust a számítógépére telepítve, de telepítés nélkül is lehetősége van rá a Live verzió használatával. Bootolhat optikai meghajtóról, vagy USB tárolóról mint a külső merevlemez vagy pendrive.
A telepíthető és live verziók mindegyike meghaladja egy hagyományos CD méretét, így optikai meghajtó választása esetén DVD lemezt kell használnia.
Figyeljen arra hogy a rendszere BIOS-ában a boot sorrend megfelelő legyen, az optikai meghajtó vagy az USB tároló legyen az első helyen. Újabb rendszereken a bootolási sorrend induláskor módosítható egy gomb lenyomásával (általában F8 az asztali rendszereken).

2. Áttekintés

A Pardus 2011.1 jelentős fejlesztésekkel gazdagodott a Pardus 2011 kiadása óta. A következő ismertetőből átfogó képhez juthat ezekről a változtatásokról.
  • Kernel - 2.6.37.6 Linux kernel provides updated hardware support together with a hundreth of bug fixes.
  • YALI - YALI, the installer of Pardus, gained Rescue Mode which provides easy password and bootloader recovery.
  • KDE SC - A Pardus 2011.1 a legfrissebb KDE-vel érkezik (KDE SC 4.6.5). Már az alapcsomagok is számtalan javítást és frissítést tartalmaznak, amitől a KDE környezet felhasználói élménye jelentősen javul.
  • LibreOffice - LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is updated to the much-awaited 3.4 version which brings huge improvements over the older version.
  • Firefox - Firefox 5.0, the latest stable version of the most popular Open Source web browser, is now shipped with Pardus 2011.1.

3. Mik az újdonságok?

3.1. Fejlesztések a Pardus Technológiában

3.1.1. YALI

  • YALI, the installer of Pardus, gained system rescue menu that provides easy bootloader and password recovery.

3.1.2. Pisi

  • Some of the core components are now parallellized to increase the utilization of multi-core machines which means faster package operations and better package management experience.

3.1.3. QuickFormat

  • QuickFormat is a removable device formatting tool that can integrate to Dolphin. Now, it's very easy to format USB flash drives and card readers.

3.2. Fejlesztések, frissítések az alap komponensekben

  • Kernel - The Linux kernel shipped with Pardus 2011.1 is 2.6.37.6.
  • Plymouth - The bootsplash technology used in Pardus 2011 is updated to its latest version for a much stable experience.
  • X.org - xorg-server is updated to 1.9.5.
  • TeX Live - TeX Live documentation stack is updated to latest 2011 version.

3.3. Felhasználói élmény

3.3.1. Változások a KDE asztali környezetben

A Pardus 2011.1 a legfrissebb KDE-vel érkezik (KDE SC 4.6.5). Már az alapcsomagok is számtalan javítást és frissítést tartalmaznak, amitől a KDE környezet felhasználói élménye jelentősen javul.
KDE Software Compilation, versioned 4.6.5, provide many new features in each of KDE's components including Plasma, Solid, Nepomuk, KWin and also introduces a brand new Bluetooth framework called BlueDevil allows easy device pairing and management.

3.3.2. Alapértelmezett alkalmazások

3.3.2.1. Mozilla Firefox
Pardus 2011.1 comes with Mozilla Firefox 5.0 as the default web browser application. This version of Firefox introduces CSS animations, Do-Not-Track, improved support for HTML5, MathML, and SMIL, improved network, memory and JavaScript performance and fixes for several stability and security issues.
3.3.2.2. Clementine
Clementine, a cross-platform music player based on Amarok 1.4 is updated to latest stability and maintenance version 0.7.1.
3.3.2.3. SMPlayer
SMplayer, az Mplayer Qt alapú felülete az alapértelmezett Video/VCD/DVD lejátszó alkalmazás, amely jól integrált a KDE-be, és számos videoformátumot támogat köszönhetően az Mplayernek.
3.3.2.4. KsCD
KsCD, az alapértelmezett audio CD lejátszó alkalmazás.

3.3.3. Böngészőprogramok

Habár a telepítő médiumok nem tartalmazzák, a csomagkezelőből böngészőprogramok széles választéka telepíthető, mint a Chromium, rekonq, Arora és Opera.

3.3.4. Rendszerbeállítások

The display configuration tool of the graphics card manufacturer is now recommended in the Display and Monitor module of the System Settings, if it suits better to your graphics card driver.

3.3.5. Irodai csomag

LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is the default Office Suite in Pardus 2011.1. 3.4 version of LibreOffice introduces; improved text rendering, configurable color charts, improved thumbnailed HTML export of presentations, faster Flat ODF import/export and better integration desktop theme integration.

3.3.6. Other

Most wanted packages like Skype, a P2P VoIP client that allows you to make voice calls, and Wine, a software that allows you to run Microsoft Windows applications, are now included in Pardus 2011.1 package repopsitories.

4.2. Rendszermemória (RAM)

A Pardus 2011.1 memóriaigényét erősen befolyásolja a felhasználó számítógépkezelési szokásai, ezért nehéz alsó és felső határértéket mondani. Mi mindenesetre, a zökkenőmentes használathoz minimum 1GB memóriát ajánlunk.

4.3. Merevlemez követelmények

4.3.1. Minimum követelmények

A Pardus 2011.1 sikeres telepítéshez legalább 4GB szabad merevlemez kapacitás szükséges. Mivel a telepítendő csomagok kiválasztása nem opció, ezért ez a minimum követelmény minden telepítésre vonatkozik.

4.3.2. Ajánlott rendszerkövetelmények

A Pardus 2011.1 hatékony használatához mi legalább 10GB szabad merevlemez területet javaslunk.

4.3.3. Megjegyzés a lapozó területhez

A lapozóterületet akkor használja az operációs rendszer, ha kifogy a fizikai memóriából. Ha Ön szerint a fizikai memória elegendő a mindennapi használati szokásaira, akkor a telepítés alatt kihagyhatja a lapozóterület létrehozását.
Megjegyzés: egy megfelelő méretű lapozóterület szükséges ha használni kívánja a Pardus hibernálási funkcióját. Ellenkező esetben ez a funkció tiltva lesz.
