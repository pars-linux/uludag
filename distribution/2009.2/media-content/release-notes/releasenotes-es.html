Pardus 2009.2 Release Notes

1. Welcome to Pardus 2009.2 !

Welcome to Pardus 2009.2, another great Pardus release from Pardus developers, translators, artists y documentation writers.

Le invitamos a visitar http://www.pardus.org.tr/eng por más información

1.1. Que es Pardus ?

Pardus es una distribución de GNU/Linux, producto del proyecto Pardus, realizado por el Instituto Nacional de Electrónica y Criptología de Turquía (UEKAE), bajo el Concicilio de Ciencias y Investigación Tecnológica de Turquía (TÜBİTAK) y además por contribuyentes voluntarios. Su nombre viene del nombre latín (y científico) del leopardo de Anatolia, "Panthera pardus tulliana".

Pardus consiste de un sistema operativo GNU/Linux, aplicaciones de código abierto y datos necesarios para entornos de escritorio y de servidores, para brindar a los usuarios una experiencia con fácilidad y mucha experiencia.

1.2. Como puedo utilizar Pardus

Pardus viene en 2 modalidades, Kurulan (instalador) o Çalışan (liveCD, que se ejecuta solamente desde el medio, sin modificar su computadora, pero es más lento de una versión instalada) y se puede arrancar desde CD / DVD / USB. Para iniciarse Pardus necesita insertar o conecta el medio a su computadora, rebootear, y setear el BIOS adecuadamente para que se bootee del medio deseado con Pardus.

1.3. Feedback - Retroalimentación

Puede utilizar http://bugs.pardus.org.tr, (sistema de seguimiento de bugs de Pardus) para reportar un error, enviar una corrección o enviar cualquier sugerencia relacionado a Pardus. Agradecemos a nuestros usuarios y desarrolladores por sus reportes de errores, correcciones y sugerencias para la presente versión.

2. Que hay de nuevo ?

2.1. Actualizaciones en tecnologías Pardus

With the release of Pardus 2009.2, core technologies like PiSi and COMAR are updated to provide new features. YALI is also updated to offer a smoother and more reliable installation experience. The enhancements in Mudur initialization system provide a faster and more robust startup process along with a better remote file system support.

Todas las herramientas administrativas de Pardus fueron rediseñados y optimizados para KDE4. Nuevas herramientas de sistema están disponibles en los repositorios de Pardus: Plasmoid de Red para un sencillo cambio de perfiles de red, Plasmoid de Servicios para una sencilla administración de servicios. Estos plasmoids se puede colocar en el escritorio o en cualquier barra de sistema para su fácil acceso.

Una nueva herramienta de administración Administrador de Sistema fue añadido para facilitar la configuración de parámetros globales del sistema, como locale predeterminada, distribución de teclado, zona horaria, etc.

Administrador de Paquetes and PiSi son más veloces que antes, gracias al nuevo mecanismo de cache. Muchos cambios se aplicaron al sistema de paquetes PiSi, resultando en una reducción de los paquetes PiSi por ca. 20%, además predeterminado será desde ahora en adelante el soporte de paquetes delta. La instalación y actualización de paquetes será así mucho más rápido y requiere menos ancho de banda. También el arbol de componentes (categorías de programas) fue rediseñado para mejorar la busqueda de los paquetes deseados.

Sistema de Instalación Yalı fue rediseñado y aprovecha las nuevas funcionalidades de PiSi, duplicando la velocidad de instalación del sistema Pardus. Pardus ahora instala todo en ca. 15 minutos en una computadora promedia de hoy. Una nueva modalidad de recuperación fue agregado a YALI, permitiendo retroceder actualizaciones de paquetes, restaurar el arranque etc.

2.2. Mejoras Visuales

Pardus 2009.2 comes with a lot of graphical enhancements. With this new release, Pardus users will meet the Milky icon theme of Pardus, consisting of about 1000 icons. We also have a wide variety of graphical enhancements, from wallpapers of nature photographs to 3D rendered images, splash images etc.

2.3. Entorno de Escritorio KDE 4

Nuevo miembro de la familia KDE KDE4 that has been the choice of many Pardus users for a while now, is now the default desktop environment. Pardus 2009.2 includes the latest stable release of KDE that has been enhanced and modified for Pardus. All the KDE programs in Pardus repositories have been updated to or replaced by their latest KDE 4 versions.

Con la nueva versión de KDE Pardus brinda una nueva experiencia para los usuarios. El escritorio con su sistema basado en plasmoides, que permite colocar cada componente en cualquier parte del escritorio, la búsuqeda de escritorio Strigi integrada completamente con el entorno de escritorio, efectos de escritorio como parte de KDE, el administrador de archivos Dolphin de la última generación con soporte para muchos protocolos de red son solamente algunos ejemplos de las nuevas características para nuestros usuarios.

2.4. Componentes Actualizados

Pardus 2009.2 includes a lot of package updates besides Pardus Technologies. Some of the highlights are as follows:

  • Linux Kernel 2.6.31.13
  • OpenOffice.org Office Suite 3.2.1.1
  • Firefox Web Browser 3.6.3
  • Gimp 2.6.8
  • Xorg 1.6.5
  • Python 2.6.5
  • Texlive 2008
  • GCC 4.3.3
  • GLIBC 2.9

2.5. Nuevos Sistemas de Archivos

Pardus viene como sistema de archivos predeteminado con el sistema de archivos Ext4, el sucesor del Ext3. Ext4 es mucho más rápido que Ext3 y tiene muchas nuevas características relacionado con capacidad y extension. Por ejemplo Ext4 soporta tamaños de particiones hasta 1 exabyte y tamaños de archivos hasta 16 terrabyte.

XFS, ReiserFS y el sistema experimental Btrfs también están disponibles en nuestro sistema de instalación, Yalı.

For more information about updates and new features on Pardus 2009.2 , you can take a look at the Sitio web de Pardus.

3. Requisitos de Instalación

You can run Pardus 2009.2 on an average hardware. We suggest to use at least 512 MB RAM (memory) and a 1400 MHz processor to get a good performance. Minimum and recommended hardware requirements for Pardus 2009.2 are mentioned below:
  • Memoria RAM 256 MB (recomendado 1GB o más)
  • Procesador 450 MHz Intel o AMD (recomendado 1000 MHz)
  • Espacio de 4 GB disponible en disco (recomendado 10 GB)
