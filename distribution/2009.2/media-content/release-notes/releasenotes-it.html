Pardus 2009.2 Release Notes

1. Welcome to Pardus 2009.2 !

Welcome to Pardus 2009.2, another great Pardus release from Pardus developers, translators, artists e documentation writers.

Ti inviatiamo a visitare http://www.pardus.org.tr/eng per ulteriori informazioni.

1.1. Cos'è Pardus?

Pardus è una distriuzione Linux facente parte del Progetto Pardus,sviluppata dall'Istituto di Ricerca Elettronica e Criptologica Nazionale Turco(UEKAE), che fa parte del Concilio Turco della Ricerca Scientifica e Tecnologica(TÜBİTAK). Viene inoltre sviluppata da volontari. Il nome deriva dal nome scientifico del leopardo antoliano, Panthera Pardus Tulliana.

Pardus è una distribuzione Linux, che include applicazioni Open Source ideali per ambiente desktop e server. Il fine di Pardus è quello di offrire ai suoi utenti un sistema semplice da usare e ricco di caratteristiche al tempo stesso.

1.2. Come Usare Pardus

Pardus è disponibile in due versioni. La Kurulan, che serve ad installare Pardus sul tuo computer, e la Çalışan, che è una versione live, in grado di avviare Pardus su dispositivo come CD, DVD, memorie USB ecc., che puoi usare senza bisogno di installazione, a patto di rinunciare ad un po' di velocità. Per avviare entrambe versioni devi inserire il dipositivo nel computer, riavviare e fare in modo che il BIOS avvii il dispositivo sul quale si trova Pardus.

1.3. Feedback

Puoi usare http://bugs.pardus.org.tr, il Bug tracking system di Pardus, per segnalare un bug, inviare dei bug fix o proporre dei cambiamenti relativi a Pardus. Intendiamo ringraziare i nostri utenti e i nostri sviluppatori per i loro report, bugfix e altri suggerimenti relativi a questa release.

2. Cosa c'è di nuovo?

2.1 Miglioramenti delle Tecnologie di Pardus

With the release of Pardus 2009.2, core technologies like PiSi and COMAR are updated to provide new features. YALI is also updated to offer a smoother and more reliable installation experience. The enhancements in Mudur initialization system provide a faster and more robust startup process along with a better remote file system support.

Tutti i programmi di gestione di Pardus sono stati migliorati e ottimizzati specificamente per KDE4. I nuovi strumenti di Pardus, il Plasmoide di Rete e il Plasmoide Servizi sono disponibili nei repository. Questi Plasmoidi possono essere posizionati ovunque sul desktop o sui pannelli.

Un nuovo Gestore Il Gestore di Sistema è stato aggiunto per fornire una configurazione semplice e veloce del sistema in generale, come la lingua, la tastiera, il fuso orario.

Il Gestore Pacchetti and PiSi adesso sono molto più veloci, grazie ad un nuovo meccanismo di caching. Il sistema di pacchettizzazione PiSi ha subito molte modifiche, rendendo i già leggerissimi pacchetti circa il 20% più piccoli, e il supporto per pacchetti delta d'ora in poi sarà attivato per default. L'installazione e l'aggiornamento di pacchetti sarà molto più veloce e richiederà meno banda. Inoltre l'albero dei componenti dei pacchetti è stato rifatto, in modo da rendere la ricerca di un pacchetto ancora più semplice.

Sistema di installazione Yali adesso è più veloce grazie alle nuove caratteristiche di PiSi, dimezzando i tempi di installazione. Adesso Yali è in grado di installare Pardus in circa 15 minuti su un computer di fascia media. Inoltre sono state aggiunte alcune caratteristiche alla modalità di ripristino.

2.2. Miglioramenti Grafici

Pardus 2009.2 comes with a lot of graphical enhancements. With this new release, Pardus users will meet the Milky icon theme of Pardus, consisting of about 1000 icons. We also have a wide variety of graphical enhancements, from wallpapers of nature photographs to 3D rendered images, splash images etc.

2.3. L'ambiente Desktop KDE4

Il nuovo membro della famiglia KDE KDE4 that has been the choice of many Pardus users for a while now, is now the default desktop environment. Pardus 2009.2 includes the latest stable release of KDE that has been enhanced and modified for Pardus. All the KDE programs in Pardus repositories have been updated to or replaced by their latest KDE 4 versions.

With the new version KDE, Pardus provides a new look and feel for users. Plasmoid based desktop system, where every component can be placed on anywhere on the desktop, Strigi desktop search fully integrated to desktop environment, desktop effects coming as a part of KDE, Dolphin the next generation file manager with many remote protocol support are just a few examples of the new desktop experience for our users.

2.4. Componenti Aggiornati

Pardus 2009.2 includes a lot of package updates besides Pardus Technologies. Some of the highlights are as follows:
  • KDE 4.4.3
  • Kernel Linux 2.6.31.13
  • Suite OpenOffice.org 3.2.1.1
  • Browser Firefox 3.6.3
  • Gimp 2.6.8
  • Xorg 1.6.5
  • Python 2.6.5
  • Texlive 2008
  • GCC 4.3.3
  • GLIBC 2.9

2.5 Nuovi Filesystem

Pardus now comes with Ext4 filesystem, successor of Ext3, as the default filesystem. Ext4 is a lot faster than Ext3 and has many new features in terms of capacity and extensions. For example, Ext4 supports partition sizes up to 1 exabyte and file sizes up to 16 terrabyte.

XFS, ReiserFS and the experimental Btrfs are also available in our installation system, Yalı.

For more information about updates and new features on Pardus 2009.2 , you can take a look at the Pardus website.

3. Requisiti di Installazione

You can run Pardus 2009.2 on an average hardware. We suggest to use at least 512 MB RAM (memory) and a 1400 MHz processor to get a good performance. Minimum and recommended hardware requirements for Pardus 2009.2 are mentioned below:
  • 256 MB di RAM (almeno 1GB consigliata)
  • Processore Intel o AMD 450 MHz (1GHz consigliata)
  • Almeno 4 GB di spazio sul disco (10 GB consigliati)
