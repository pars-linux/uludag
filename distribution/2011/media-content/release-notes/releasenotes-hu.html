Release Notes

1. Welcome to Pardus 2011

Welcome to Pardus 2011, the latest release of Pardus, a powerful, KDE-focused Linux distribution.

1.1. What is Pardus?

Pardus is an open-source, freely distributable, GPL (General Public License) licensed user-friendly GNU/Linux distribution with many usability, hardware compatibility, stability and security enhancements.
The Pardus project is sponsored by the National Research Institute of Electronics and Cryptology (UEKAE), an affiliate of the Scientific and Technological Research Council of Turkey (TÜBİTAK).
The aim of the Pardus distribution is to offer people a stable, reliable, easy-to-use and feature-rich computer experience by providing modern and popular open-source software. The developers are also putting an additional effort to make Pardus run on a wide range of hardware.
The project is named after the latin (and scientific) name for the Anatolian leopard, Panthera pardus tulliana.

1.2. How Can I Try Pardus?

You can try Pardus either by installing it on your computer or running the Live version without touching your hard disk. You can boot Pardus from an optical drive as well as from USB mass storage devices like external hard drives and USB flash drives.
Both the installation and live version images exceeds the size of a regular CD so if you are willing to use an optical drive for booting Pardus, you currently have to use DVD media to burn the images.
Note that the boot order in your system's BIOS software should be set with the optical drive or USB mass storage device as the first device in order to boot from the installation or the live medium. On newer systems, the boot order can also be changed by pressing a key (generally F8 on desktop systems) while booting.

2. Overview

Pardus 2011 contains major improvements over the previous Pardus release 2009.2. The following sections provide a brief overview of those changes.

2.1. Hardware Overview

Pardus 2011 now supports 64-bit processor architecture which is supported by many new Intel and AMD processors.

2.2. Software Overview

  • Kernel - The latest Linux kernel 2.6.37 provides an up-to-date hardware support together with a thousand of bug fixes.
  • Plymouth - The bootsplash technology used in Pardus 2009.2 is completely dropped and replaced by the new Plymouth engine.
  • YALI - YALI, the installer of Pardus, gained LVM/RAID and UUID support.
  • KDE SC - Pardus 2011 comes with the latest KDE Software Component, KDE SC 4.5.5. The base packages also contains numerous backports and fixes which will improve the stability of your desktop experience significantly.
  • Kaptan - Kaptan, the desktop customization tool of Pardus, now optionally captures your picture and sets it as your avatar in KDE.
  • NetworkManager - GNOME NetworkManager 0.8.2 is now the default networking backend in Pardus 2011. Users are now able to set up their HSPA/CDMA/VPN networks together with the already supported Ethernet and 802.11 WLAN networks.
  • GTK Oxygen style - All GTK applications are now rendered with Oxygen style thanks to the oxygen-gtk project. This brings a huge improvement to the user interface consistency.
  • LibreOffice - LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is now the default Office Suite in Pardus 2011.

3. What is New?

3.1. Updates in Pardus Technologies

3.1.1. YALI

  • YALI, the installer of Pardus, gained LVM/RAID and UUID support.
  • The user interface is completely redesigned to improve the usability.
  • Users are now able to select between Open Source and proprietary drivers for their nVidia video adapters.

3.1.2. First Boot

  • Some of the settings which were asked during the installation in Pardus 2009.2 are now moved into a different application which shows up once during the first boot after the installation.

3.1.3. Kaptan

  • Kaptan, the desktop customization tool of Pardus, now optionally captures your picture and sets it as your avatar in KDE.
  • Users are now able to select their preferred icon theme which will be used in their KDE desktop environment.
  • The user interface is completely redesigned to improve the usability.

3.1.4. Pisi

  • Pisi now internally uses XZ as the default compression algorithm.
  • Binary package names now reflects the architecture and the distribution release for which the package is built.
  • Various speed enhancements has been done in the core components for a better package management experience.

3.1.5. Package Manager

  • Users are now able to vote for a package and see the screenshots of graphical applications shipped within packages.
  • The graphical user interface is redesigned to improve the usability. Various speed enhancements has been done for a better graphical package management experience.
  • A lot of KDE integration work has been done including the usage of KDE system tray.

3.2. Updates in Core Components

  • Kernel - The Linux kernel shipped with Pardus 2011 is 2.6.37 which is the latest stable kernel released on 2011/01/05.
  • Plymouth - The bootsplash technology used in Pardus 2009.2 is completely dropped and replaced by the new Plymouth engine.
  • Python - Python is updated to 2.7.1.
  • Perl - Perl is updated to 5.12.2.
  • X.org - xorg-server is updated to 1.9.4 RC1, improvements to the automatic driver configuration mechanism has been done by our developers.
  • Texlive - Texlive documentation stack is updated to 2009.

3.3. Desktop Experience

3.3.1. Changes in KDE Desktop Environment

Pardus 2011 comes with the latest KDE Software Component, KDE SC 4.5.5. The base packages also contains numerous backports and fixes which will improve the stability of your desktop experience significantly.
KDE SC 4.5.5 brings more performance and stability to the desktop experience. Support for WebKit, improved user experience for netbooks, a new high-speed cache allowing fast access to resources stored on disk are among the new features of this version.

3.3.2. Visual Enhancements

All GTK applications are now rendered with Oxygen style thanks to the oxygen-gtk project. This brings a huge improvement to the user interface consistency.
Milky icon theme especially designed for Pardus is shipped within the installation image. Users can set this icon theme through Kaptan or System Settings.

3.3.3. Network Management

Besides switching to GNOME NetworkManager as the default networking backend, Pardus 2011 comes with a heavily improved and modified KDE Network Management tool and plasmoid to give their users a much more consistent networking experience.
With this new networking backend, users are now able to easily set up their Ethernet (With support for 802.1x authentication), 802.11 Wireless (With support for WEP/WPA/WPA2/WPA-EAP and 802.1x authentication), EDGE/3G/CDMA and VPN networks.

3.3.4. Default Applications

3.3.4.1. Mozilla Firefox
Pardus 2011 comes with Mozilla Firefox 4.0 Beta9 as the default web browser application. New features of this Firefox release include Firefox Panorama, application tabs, a redesigned extension manager, Jetpack extensions support, integration with Firefox Sync, and support for multitouch displays.
3.3.4.2. Clementine
Clementine, a cross-platform music player based on Amarok 1.4 is set as the default audio playing application as it is well maintained and lightweight.
3.3.4.3. SMPlayer
SMplayer, a Qt based frontend for MPlayer is set as the default Video/VCD/DVD playing application as it is well integrated with KDE desktop environment in terms of usability and graphical user interface and supports playing a wide range of video file types thanks to the MPlayer backend.
3.3.4.4. KsCD
KsCD, is set as the default Audio CD playing application.

3.3.5. Web Browsers

Although not included in the installation images, a wide range of web browsers including Chromium browser, rekonq, Arora and Opera are available for installation through the package manager.

3.3.6. System Settings

Panda (Pardus alternative driver administration) is a new System Settings module which allows users to change the driver between Open Source and proprietary one for nVidia and ATI display adapters.
Users are now able to set up their Bluetooth devices, Wacom tablets and Synaptics touchpads through System Settings.
The default language used throughout the whole system can now be changed through the locale settings section.

3.3.7. Office Suite

LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is now the default Office Suite in Pardus 2011.

3.3.8. Games

Pardus 2011 continues the tradition of giving importance to the quality of games offered to their users. The default installation now includes the board and logic games provided by the kdegames package.

3.4. Other Changes

Firefox gets new stable release rapidly. 4.0 Beta9 release presents many new features like the new interface of the browser as well as plugin management new interface, tab grouping, bookmarks between computers, passwords and history matching, a new plug-in interface. HTML5, CSS3 and WebM technology supports are also most important features for the developers.

4. System Requirements

4.1. Processor (CPU)

You may want to install the 64-bit version of Pardus 2011 if your system is equipped with one of the processors listed in the sections below.

4.1.1. Intel processors which support 64-bit architecture

  • Newer versions of Pentium 4, Celeron D, Xeon and Pentium Dual-Core
  • Atom D5xx, N4xx and N5xx
  • All versions of Pentium D, Pentium Extreme Edition, Core 2, Core i7, Core i5 and Core i3 processors
  • Probably newer processors manufactured after the time this document was originally written.

4.1.2. AMD processors which support 64-bit architecture

  • AMD Athlon 64, 64 X2, 64 FX
  • AMD Athlon II
  • AMD Opteron
  • AMD Turion 64
  • AMD Turion 64 X2
  • AMD Sempron
  • AMD Phenom
  • AMD Phenom II
  • Probably newer processors manufactured after the time this document was originally written.

4.2. System memory (RAM)

System memory (RAM) requirements of Pardus 2011 depends heavily on your daily workflow so it is very hard to define upper and lower bounds, but we recommend at least 1GB of memory for a smooth desktop experience.

4.3. Hard disk space requirements

4.3.1. Minimum requirements

Pardus 2011 requires at least 4GB of free hard disk space for a successful installation. As we don't offer custom package selection during the installation, this minimum size constraint applies to all installations.

4.3.2. Recommended requirements

In order to effectively use Pardus 2011 on your system without running low on disk space during your daily workflow, we recommend at least 10GB of free hard disk space.

4.3.3. Note on Swap space

Swap space is used whenever your operating system runs out of physical memory. If you think that your physical memory is big enough for your daily usage, you may skip creating a swap space during installation.
Note that a disk partition reserved for swap space is required if you want to hibernate (Suspend to disk) Pardus. Otherwise the functionality will be disabled.
