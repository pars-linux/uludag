Sürüm Notları

1. Welcome to Pardus Corporate 2

Welcome to Pardus Corporate 2, the first release of Pardus Corporate series offering a mature enterprise desktop experience to the users.

1.1. Pardus Nedir?

Pardus açık kaynak kodlu, özgürce dağıtılan, GPL (Genel Kamu Lisansı) ile lisanslanmış; kullanışlılık, donanım uyumluluğu, kararlılık ve güvenlik konularında geniş olanaklar sunan bir GNU/Linux dağıtımıdır.
Pardus projesi, Türkiye Bilimsel ve Teknolojik Araştırma Kurumu (TÜBİTAK)'na bağlı Bilişim ve Bilgi Güvenliği İleri Teknolojiler Araştırmalar Merkezi (BİLGEM) bünyesinde geliştirilmektedir.
Pardus, kullanıcılarına tanınmış özgür yazılım ürünleri kullanarak kararlı, güvenilir, kolay kullanılır ve bol özellik içeren bir bilgisayar deneyimi sunmayı hedefler. Bunun yanında, geliştirici ekip bu olanakların geniş bir donanım yelpazesinde çalışması için çalışmalar yürütmektedir.
Projenin ismi Anadolu parsının latince (bilimsel) isminden, Panthera pardus tulliana'dan gelmektedir.

1.2. Pardus'u Nasıl Deneyebilirim?

Pardus'u bilgisayarınıza kurarak ya da Çalışan sürümüyle sabit diskinize hiç dokunmadan deneyebilirsiniz. Pardus hem optik diskler üzerinde hem de USB bellek ve disklerde çalışabilir.
Hem kurulan hem de çalışan sürümler standart CD'lere sığamayacak kadar büyük olduğu için optik sürücü ile kullanmak istediğiniz takdirde DVD kullanmanız gerekli.
Sisteminizin BIOS yazılımında belirlenen açılış sırasını Pardus kullanacağınız optik sürücü ya da USB bağlantısı önce olacak şekilde ayarlamanız gereklidir. Yeni bilgisayarların çoğu açılış sırasında bir tuşla (bir çok masaüstü bilgisayarda F8 bu işi görür) bu sırayı değiştirebilmenizi sağlar.

1.3. Daha Fazla Bilgi

Daha çok bilgi için lütfen http://www.pardus.org.tr adresinden resmi web sitemizi ziyaret edin.

1.4. Geri bildirim

Karşılaştığınız hataları bildirmek, yeni bir özellik ya da entegre edilmesini istediğiniz bir uygulamayı önermek için lütfen http://hata.pardus.org.tr adresini ziyaret edin. Pardus, özgür yazılım ve Linux topluluklarının tüm üyelerine ve kullanıcılarına değerli katkıları için teşekkür ederiz.

2. Genel Bakış

The following sections provides a brief overview of the features offered by Pardus Corporate 2.

2.1. Donanım

Pardus Corporate 2 supports both 32-bit and 64-bit processor architectures manufactured by Intel and AMD.

2.2. Yazılım

  • Kernel - The Linux kernel shipped with Pardus Pardus Corporate 2 is 2.6.35 which is the latest kernel branch which will be supported for a long term period.
  • Plymouth - The bootsplash technology used in Pardus Corporate 2 is the brand new Plymouth engine.
  • YALI - Kurulum aracı YALI, LVM/RAID ve UUID desteği kazandı.
  • KDE - Pardus Corporate 2 comes with KDE 3.5.10, the old KDE stack which has proven its stability and reliability. The base packages also contains numerous fixes which will improve the stability of your desktop experience significantly.
  • NetworkManager - GNOME NetworkManager 0.8.2 is the default networking backend in Pardus Corporate 2. Users are now able to set up their HSPA/CDMA/VPN networks together with the already supported Ethernet and 802.11 WLAN networks.
  • LibreOffice - LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is the default Office Suite in Pardus Corporate 2.

3. Yenilikler

3.1. Pardus Teknolojilerinde Yenilikler

3.1.1. YALI

  • Pardus kurulum aracı YALI, LVM/RAID ve UUID desteği sunuyor.
  • Kullanışlılığı artırmak için tüm arayüz yeniden tasarlandı.

3.1.2. Pisi

  • Pisi XZ sıkıştırma algoritmasını öntanımlı olarak kullanıyor.
  • İkili paketlerin isimleri artık hangi sürüm ve işlemci mimarisine göre derlendikleri bilgisini gösterir şekilde hazırlanıyor.
  • Paket yönetiminde daha rahat kullanım için temel bileşenlerde bir çok iyileştirme yapıldı.

3.1.3. Paket Yöneticisi

  • The graphical user interface is redesigned to improve the usability.
  • Various speed enhancements has been done for a better graphical package management experience.

3.2. Temel Bileşenlerde Güncellemeler

  • Kernel - The Linux kernel shipped with Pardus Pardus Corporate 2 is 2.6.35 which is the latest kernel branch which will be supported for a long term period.
  • Plymouth - The bootsplash technology used in Pardus Corporate 2 is the brand new Plymouth engine.
  • Python - Python 2.6.5 is the default Python interpreter in Pardus Corporate 2.
  • Perl - Perl 5.10.1 is the default Perl interpreter in Pardus Corporate 2.
  • X.org - The shipped xorg-server version is 1.7.7, improvements to the automatic driver configuration mechanism has been done by our developers.

3.3. Updates in Server Components

  • Web servers - Pardus Corporate 2 ships both Apache 2.2.17 and NGINX 0.8.54 HTTP daemon alternatives.
  • Databases - MySQL 5.5.8, PostgreSQL 9.0.3, BerkeleyDB 4.6.21 and SqLite 3.7.3 are among the DBMS alternatives offered in Pardus Corporate 2.
  • Mail Servers - Postfix 2.6.5 and Dovecot 1.2.16 are the mail server daemon softwares offered within Pardus Corporate 2.

3.4. Masaüstü Deneyimi

3.4.1. Görsel İyileştirmeler

The default icon theme of Pardus Corporate 2 is BC Tango which is a Tango derivative specially customized for KDE desktop environment.

3.4.2. Ağ Yönetimi

Pardus Corporate 2 uses GNOME NetworkManager as the default networking backend.
With the GNOME networking backend, users are able to easily set up their Ethernet (With support for 802.1x authentication), 802.11 Wireless (With support for WEP/WPA/WPA2/WPA-EAP and 802.1x authentication), EDGE/3G/CDMA and VPN networks.

3.4.3. Ofis Paketi

LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is the default Office Suite in Pardus Corporate 2.

4. Sistem Gereksinimleri

4.1. İşlemci (CPU)

You may want to install the 64-bit version of Pardus Corporate 2 if your system is equipped with one of the processors listed in the sections below.

4.1.1. 64-bit mimarisini destekleyen Intel işlemciler

  • Pentium 4, Celeron D, Xeon ve Pentium Dual-Core işlemcilerinin yeni sürümleri
  • Atom D5xx, N4xx ve N5xx
  • Pentium D, Pentium Extreme Edition, Core 2, Core i7, Core i5 ve Core i3 ailelerinin hepsi
  • Bu belge yazıldıktan sonra üretilen yeni işlemcilerin büyük bir kısmı.

4.1.2. 64-bit mimarisini destekleyen AMD işlemciler

  • AMD Athlon 64, 64 X2, 64 FX
  • AMD Athlon II
  • AMD Opteron
  • AMD Turion 64
  • AMD Turion 64 X2
  • AMD Sempron
  • AMD Phenom
  • AMD Phenom II
  • Bu belge yazıldıktan sonra üretilen yeni işlemcilerin büyük bir kısmı.

4.2. Sistem Belleği (RAM)

System memory (RAM) requirements of Pardus Corporate 2 depends heavily on your daily workflow so it is very hard to define upper and lower bounds, but we recommend at least 512MB of memory for a smooth desktop experience.

4.3. Sabit disk

4.3.1. Minimum gereksinimler

Pardus Corporate 2 requires at least 4GB of free hard disk space for a successful installation. As we don't offer custom package selection during the installation, this minimum size constraint applies to all installations.

4.3.2. Önerilen gereksinimler

In order to effectively use Pardus Corporate 2 on your system without running low on disk space during your daily workflow, we recommend at least 10GB of free hard disk space.

4.3.3. Takas alanı hakkında

Takas alanı, fiziksel belleğin yetersiz kaldığı durumlarda kullanılır. Eğer fiziksel belleğinizin gündelik kullanımınız için yeterli olduğundan eminseniz kurulum sırasında takas alanı oluşturmamayı tercih edebilirsiniz.
Lütfen Pardus'un kış uykusuna (disk kullanarak askıya alma) yatmasını istiyorsanız takas alanı kullanmanız gerektiğini unutmayın. Takas alanı olmadığında bu özellik devre dışı kalacaktır.
