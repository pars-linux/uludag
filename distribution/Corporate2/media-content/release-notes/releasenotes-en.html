Pardus Corporate 2 Alpha Release Notes

1. Welcome to Pardus Corporate 2 Alpha

Welcome to Pardus Corporate 2 Alpha, another greate Pardus release from Pardus developers, translators, artists and documentation writers.


We kindly invite you to visit http://www.pardus.org.tr/eng for more information.

1.1. What is Pardus?

Pardus is a Linux distribution developed as a product of the Pardus Project, by Turkish National Research Institute of Electronics and Cryptology (UEKAE), which is under the Scientific and Technological Research Council of Turkey (TÜBİTAK) and by volunteered contributors. It is named after the Latin (and scientific) name for the Anatolian leopard, Panthera pardus tulliana.

Pardus consists of a Linux based operating system, open source applications and related data files necessary for desktop and server environments, aiming to give its users an easy to use and feature rich experience.

2. What is new?

2.1. Updates in Pardus Technologies

With the release of Pardus Corporate 2 Alpha, core technologies like PiSi and COMAR are updated to provide new features. YALI is also updated to offer a smoother and more reliable installation experience. The enhancements in Mudur initialization system provide a faster and more robust startup process along with a better remote file system support.

Package Manager and PiSi are now a lot faster than before, thanks to the new caching mechanism. Many changes have been made to the packaging system of PiSi, making PiSi packages about 20% smaller, and also delta package support will be on by default from now on. Package installing and updating will be a lot faster and will require much less bandwidth. Package component tree has also been refactored, letting users find their preferred packages even more easily.

Installation system YALI has a boost with refactoring and new PiSi features, doubling the installation speed of Pardus. Pardus now installs everything in about 15 minutes on an average computer of today. A new feature rescue mode support is also added to YALI, which can be used to take back previous package updates, restore bootloader etc.

2.2. Graphical Enhancements

Pardus Corporate 2 Alpha comes with a lot of graphical enhancements. The default icon theme is a Tango flavour polished for KDE called BCTango. We also have a wide variety of graphical enhancements, from wallpapers of nature photographs to 3D rendered images, splash images etc.

2.3. KDE Desktop Environment

Pardus Corporate 2 Alpha uses KDE Desktop Environment 3.5.10 which fully satisfies the needs of enterprise users in terms of performance and stability.

2.4. Updated Components

Pardus Corporate 2 Alpha includes a lot of package updates besides Pardus Technologies. Some of the highlights are as follows:
  • KDE 3.5.10
  • Linux Kernel 2.6.32.13
  • Openoffice Office Suite 3.1.1.5
  • Firefox Web Browser 3.6.3
  • Gimp 2.6.8
  • Xorg 1.7.7
  • Python 2.6.5
  • Texlive 2008
  • GCC 4.3.3
  • GLIBC 2.9

2.4.1 Base System

  • PolicyKit is upgraded to 0.96, polkit-gnome is now the default authentication agent,
  • D-Bus is upgraded to the latest non-permissive 1.2.24 version which includes various important bug fixes,
  • procps is adjusted so that its tools provides kernel control groups related informations,
  • curl now gracefully handles anonymous login attempts using a less generic identity,
  • dhcpcd is updated to the latest stable 5.2 branch,
  • The default kernel and the PAE flavour are all updated to 2.6.32.13,
  • libcgroup is packaged for further systemd integration attempts

2.4.2 Hardware

  • hplip is updated to 3.10.5 to support more printers,
  • ntfs_3g is updated to the latest stable version,
  • All ALSA components are updated to a recent snapshot for better hardware support and stability,
  • Smart card support is improved, serious problems with OpenCT and pcsc-lite are fixed,
  • CUPS is updated to 1.4.3, PHP binding for the libcups is enabled,
  • bluez is updated to 4.63

2.4.3 Networking

  • GNOME NetworkManager, 4 VPN plugins, network-manager-applet and ModemManager are now available in the repository. Users can switch to NetworkManager backend by tweaking the /et/conf.d/NetworkManager file,
  • OpenAFS client/server and kernel modules are now available in the repository,
  • PPP is updated to 2.4.5 which includes serious DNS related bug fixes

2.4.4 Desktop

  • Mozilla Firefox is updated to 3.6.3,
  • Mozilla Thunderbird 3.0.4 is now bundled within the installation image,
  • notification-daemon now uses the nodoka theme by default,
  • Qt is updated to 4.6.2, Gtk+ to 2.20.1,
  • Package Manager is refactored and polished with many new features and improvements,
  • The default BCTango icon theme is improved,
  • evolution as an alternative e-mail client is available in the repository

2.4.5 Server

  • mit-kerberos is compiled with LDAP support,
  • ntp is updated to 4.2.6_p1 and the package is splitted into client and server sub packages,
  • Various openldap server startup issues has been fixed
For more information about updates and new features on Pardus Corporate 2 Alpha, you can take a look at the Pardus web site.

3. Installation requirements

You can run Pardus Corporate 2 Alpha on an average hardware. We suggest to use at least 512 MB RAM (memory) and a 1400 MHz processor to get a good performance. Minimum and recommended hardware requirements for Pardus Corporate 2 Alpha are mentioned below:
  • 256 MB memory (1GB or more recommended)
  • 1000 MHz Intel or AMD processor (1400 MHz recommended)
  • At least 4 GB of free hard disk space (10 GB recommended)
