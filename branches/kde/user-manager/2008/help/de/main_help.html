Benutzer-Manager

Benutzer-Manager kann verwendet werden, um Benutzer auf Pardus hinzuzufügen, zu löschen oder zu bearbeiten. Nachdem die Applikation gestartet ist, zeigen zwei Tabs die Benutzer und Gruppen an. Um einen neuen Benutzer hinzuzufügen, gehen Sie wie folgt vor:

  1. Klicken Sie auf "Hinzufügen".
  2. Geben Sie den Benutzernamen (z.B. John Doe) und das Passwort ein.
  3. Definieren Sie den Benutzernamen (z. B. john).
  4. Wählen Sie aus dem rechten Abschnitt die Aktivitäten aus, die dieser Benutzer ausführen darf.
  5. Klicken Sie auf "Hinzufügen"-der Benutzer wird zum System hinzugefügt.

Um einen Benutzer zu löschen, klicken Sie einfach nach der Auswahl des Benutzers aus der Liste auf "Löschen".
