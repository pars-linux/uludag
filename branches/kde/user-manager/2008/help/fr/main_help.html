Gestionnaire d'utilisateurs

Le gestionnaire d'utilisateur peut être utilisé pour ajouter, supprimer ou modifier un utilisateur sur Pardus. Après le démarrage de l'application, deux onglets montrent les utilisateurs et les groups. Afin d'ajouter un nouvel utilisateur, suivre la procédure suivante :

  1. Cliquer sur "Ajouter".
  2. Entrer le nom de l'utilisateur (comme Pierre Dupont) et le mot de passe.
  3. Définir le nom d'utilisateur (comme pierre).
  4. Sélectionner les actions que l'utilisateur sera habilité à réaliser sur le panneau de droite.
  5. Cliquer sur "Ajouter" - L'utilisateur sera ajouté au système.

Pour supprimer un utilisateur, il suffit de cliquer sur "Supprimer" aprés l'avoir choisi dans la liste.
