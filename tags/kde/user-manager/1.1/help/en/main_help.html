User Manager

User manager can be used to add, delete or modify a user on Pardus. After the application starts, two tabs show the users and groups. In order to add a new user, do the following:

  1. Click on "Add".
  2. Enter the user name (i.e John Doe) and password.
  3. Define the username (i.e john).
  4. Select what actions this user will be allowed to do on the right pane.
  5. Click on "Add" - the user will be added to the system.

In order to delete a user, just click on "Delete" after choosing the user from the list.
