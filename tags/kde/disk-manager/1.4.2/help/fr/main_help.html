Gestionnaire de disques

Information générale

Le gestionnaire de disques inclus des paramétrages concernant les disques et leur partitions de votre système. Le gestionnaire de disque inclus également des configurations supplémentaires basées sur le type de système de fichier (Ext2, Ext3, ReiserFS, XFS, NTFS ou Fat16/32), le répertoire (chemin) pour monter le disque afin de l'utiliser au sein du système, et le type de disque. Vous devez ouvrir le gestionnaire de disque en Mode Administrateur afin de pouvoir appliquer des changements disques.

Utilisation

Trouver les disques automatiquement

Si vous n'ête pas sûrs de ce que vous faites, cliquez simplement sur le bouton Trouver les disques automatiquement en haut à gauche de la fenêtre. Cette fonctionnalité permet au Gestionnaire de Disques de détecter et d'effectuer automatiquement les réglages nécessaires concernant les partitions de disques sur votre système.

Détails

Si vous voulez configurer le paramétrage d'une partition, vous devez tout d'abord sélectionner le disque concerner dans la liste. Effectuer des changements à propos d'une partition n'est pas suffisant pour les rendre permanents. Vous devrez presser le bouton "Sauvegarder" afin de sauvegarder les changements effectués sur les partitions. Après une sauvegarde, les partitions du disque sont automatiquement montées sur le système et vous devriez voir les partition dans votre fenêtre système. Remarquez qu'après avoir effectués des changements sur une partition disque que le disque sera démonté de son ancien point de montage et sera monté dans son nouveau point avec les nouvelles préférences. Toutefois si la partition disque est en cours d'utilisation, le démontage ne fonctionnera pas et les préférences ne seront valide qu'au prochain démarrage. Dans une telle situation, vous devriez redémarrer votre ordinateur afin de valider vos nouveaux paramètres.
