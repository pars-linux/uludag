Network manager

Network manager can be used to configure network settings, in order to connect to internet, local area network, wireless network or dial-up services. It has a easy-to-use interface with the ability to connect you everywhere in seconds.

When the program is run the first time, you need to choose the type of connection and configure the settings. Below, you can see each type of connection scheme.

Connecting with an ethernet card

Pardus Linux should already recognize your ethernet card to connect to an ethernet network. Follow these steps to connect using this method.

  1. Run Network Manager
  2. Click on "Create".
  3. Choose "Ethernet network"
  4. Click on "Create connection"
  5. Enter your connection name and choose the ethernet card
  6. Choose DHCP or enter your IP configuration
  7. Click on "Use" and exit from this window
  8. Click on "Connect".
You will immediately be connected to the network and a green icon will show up.

It should be sufficient to choose DHCP for Pardus users having an ADSL or cable modem.

If you click on "General settings", you can

  1. Choose the name of your computer, or
  2. Select a DNS server

Usually, in a network environment where IP's are distributed automatically (i.e, via DHCP), it's sufficient to choose DHCP to let computers have their IP and connect to internet.

Connecting with a wireless card

A wireless card connection looks like the same as an ethernet connection. The most important difference is the name of the wireless network, i.e the ESSID. Some networks require you to choose this name.

  1. Run Network Manager
  2. Click on "Create"
  3. Click on "Wireless Network" and "Create Connection"
  4. Enter the connection name, choose the wireless card and ESSID.

You may also click on "Scan" and get a list of wireless networks which can be currently seen by your system.

Connecting with a modem

Follow these steps to connect with a modem. Note that some modems specifically designed for other operating systems may not work under Linux.

  1. Run Network Manager
  2. Click on "Create"
  3. Click on "Dialup network" and "Create connection"
  4. Enter the connection name, the serial port modem uses (usually COM1) and phone number of your service provider.
  5. Click on "Authentication" tab
  6. Enter your user name and password

Now click on "Connect", and your modem will activate
