Administrador de Usuarios

Con el Administrador de Usuarios se pueden agregar, eliminar o modificar usuarios en Pardus. La aplicación tiene dos pestañas que muestran los usuarios y los grupos. Para agregar un usuario nuevo proceda de la siguiente manera:

  1. Apriete el botón "Agregar"
  2. Ingrese el nombre completo del usuario (Ejemplo: Juan Pérez) y la contraseña
  3. Ingrese el nombre de usuario del sistema (Ejemplo: juan)
  4. Marque en la ventana derecha las tareas que se va a permitir al usuario
  5. Apriete el botón "Agregar" para agregar el usuario al sistema

El nuevo usuario aparece desde este instante en la lista de usuarios.

Para eliminar un usuario necesita solamente marcar el usuario a a eliminar y apretar el botón "Eliminar".
