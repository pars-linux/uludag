Administrador de Red

Con el Administrador de Red se pueden configurar las preferencias de la red, para conectar a internet, a una red local, a una red inalambrica o a servicios dial-up. Tiene una interfaz amigable con la facilidad de establecer la conexión en segundos.

Cuando se ejecuta el program, inicialmente se debe elegir el tipo de conexión a configurar. Abajo están listados los esquemas de conexión según tipo de conexión.

Conexión con tarjeta ethernet

Pardus Linux ya debería haber reconocido la tarjeta de red para conectar a una red local. Sigua los pasos para conectar este tipo de conexión.

  1. Ejecute el Administrador de Red
  2. Pinche "Crear".
  3. Seleccione "Ethernet network"
  4. Pinche "Crear connexión"
  5. Ingrese el nombre de la conexión y eliga la tarjeta ethernet
  6. Elija DHCP o ingrese la configuración IP 
  7. Pinche "Usar" y salga de esta ventana
  8. Pinche "Conectar".
Inmediatamente se establecerá la conexión a la red y aparecerá in ícono verde.

Para usuarios de Pardus con modem ADSL o de modem analógico debería ser suficiente usar DHCP.

Pinche la opción "Preferencias generales", para

  1. Determinar el nombre de la computadora
  2. Indicar un servidor DNS

Comunmente, en una red con asignación de IP automática (i.e, via DHCP), es suficiente con elegir DHCP para que las computadoras puedan obtener su IP y conectarse a internet.

Conexión con tarjeta wireless

Una conexión con tarjeta inalambrica se parece mucho a una conexión con tarjeta ethernet. La diferencia más importante es el nombre de la red inalambrica, i.e. el ESSID. Algunas redes requieren indicar este nombre.

  1. Ejecute Administrador de Red
  2. Pinche "Crear"
  3. Pinche "Red Inalambrica" y "Crear Conexión"
  4. Ingrese un nombre de conexión, eliga la tarjeta inalambrica y un ESSID.

Tambien puede pinchar "Scan" y obtener una lista de redes inalambricas que el sistema puede ver en este instante.

Conexión con modem

Siga los pasos para una conexión con un modem. Observe que algunos modems diseñados específicamente para otros sistemas operativos pueden no funcionar en Linux.

  1. Ejecute Administrador de Red
  2. Pinche "Crear"
  3. Pinche "Red Dialup" y "Crear conexión"
  4. Ingrese el nombre de conexión, el puerto serial que está utilizando el modem (generalmente COM1) y el número telefónico del proveedor de servicio.
  5. Pinche la pestaña "Autenticación" 
  6. Ingrese nombre de usuario y contraseña

Ahora puede pinchar "Conectar", y se activará el modem.
