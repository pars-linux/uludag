User Manager

User manager can be used to add, delete or modify a user on Pardus. After the application starts, two tabs show the users and groups. In order to add a new user, do the following:

  1. "Ekle" düğmesine tıklayın
  2. Karşınıza gelen pencerede kullanıcı adı (örnek: Ahmet Kızılbörek) ve parolasını girin
  3. Giriş yapacak kullanıcı adını tanımlayın (örnek: ahmet)
  4. Bu kullanıcının yapabileceği işlemleri sağdaki pencereden sçin
  5. "Ekle" düğmesine tıklayarak kullanıcıyı ekleyin

Bu aşamada eklenen kullanıcı listede görüntülenecektir.

Bir kullanıcıyı silmek için kullanıcı üzerine tıkladıktan sonra "Sil" düğmesine tıklamak yeterlidir.
