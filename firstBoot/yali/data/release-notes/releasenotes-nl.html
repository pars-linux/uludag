Pardus 2007.3 release notes

1. Welkom bij Pardus 2007.3

Welkom bij Pardus 2007.3, alweer een fantastische release van Pardus ontwikkelaars, vertalers, artiesten en documentatie schrijvers. Deze release bevat een set van nieuwe gereedschappen in combinatie met samen met bijgewerkte software na de eerdere Pardus 2007.2 release.

Wij nodigen u van harte uit om ook een bezoek te brengen aan http://www.pardus.org.tr/eng of http://www.pardus-linux.nl voor meer informatie.

1.1 Feedback

U kunt http://bugs.pardus.org.tr, ook wel bekend als het Pardus Bug Tracking systeem gebruiken om een bug, bug fix of elke andere sugestie aangaande Pardus door te geven. Wij willen graag onze gebruikers en ontwikkelaars danken voor hun harde werk met bugs rapporteren, repareren, en suggereren van nieuwe pakketten of toekomstig gebruik van deze nieuwe release.

Overigens kunt u ook gewoon de "feedback" optie in het Pardus gebruikers menu gebruiken om uw op-of aanmerkingen door te geven via internet.

2. Basis extras

Dit gedeelte omschrijft de Pardus 2007.3 extras

2.1 Netwerk manager is bijgewerkt

De netwerk manager detecteert zelf uw eigen draadloze profielen en schakelt automatisch, indien gewenst, over. Het geeft u tevens informatie via een popup in de taakbalk, wat de netwerk manager nog gebruikers vriendelijker maakt.

2.2 Verbeterd pakketbeheer systeem

Het pakketbeheer is verder opgepoetst. Vele bugs zijn via onze bug tracker hersteld.

2.3 YALI

YALI (yet another Linux installer) stelt u eerst enkele vragen waarna het systeem pas geinstalleerd wordt. Dit biedt u een flexibelere en meer tijdbesparende procedure. Gebruikers kunnen beslissen om al dan niet automatisch in te loggen bij opstarten. Tenslotte zijn er meerdere herkennings mogelijkheden voor state-of-the-art harde schijven toegevoegd om het installatie proces verder te vergemakkelijken.

Maar er is meer, YALI herkent of een harde schijf bootable (opstartbaar) is of niet en kiest zo de harde schijf waarop Pardus geinstalleerd zal worden. Pardus is de eerste Linux distributie met deze optie.

2.4 Bijgewerkte applicaties

KDE, de standaard desktop omgeving voor Pardus 2007.3, is bijgewerkt tot versie 3.5.8. Het alles-in-1 kantoorpakket Openoffice is bijgewerkt tot versie 2.3. K3b, de cd/dvd brand applicatie is bijgewerkt tot versie 1.0.4.

3. Installatie - Systeem vereisten

U kunt Pardus 2007.3 installeren op gemiddelde hardware. Wij raden aan om minimaal 512 MB RAM (geheugen) en een 1200 Mhz processor te gebruiken voor een effectieve werkomgeving.

De minimale en aanbevolen vereisten voor Pardus 2007.3 staan hieronder vermeld:
  • 256 MB geheugen (512 MB aanbevolen)
  • 800 MHz Intel of AMD processor (1200 MHz aanbevolen)
  • Mininaal 4 GB vrije hard disk ruimte (10 GB aanbevolen)
