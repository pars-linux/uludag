Administrador de Servicios

Pardus ejecuta algunos servicios durante el arranque del sistema. El usuario del sistema puede controlar este proceso, de manera que se pueden deshabilitar algunos servicios del arranque, y pueden ser habilitados de nuevo. Se puede utilizar el Administrador de Servicios para la configuración de esta tarea.

Un ícono indicador a la iquierda del nombre de servicio muestra el estado del servicio correspondiente. Si está rojo, el servicio no se está ejecutando. Si está verde, en este instante el servicio está activo.

Para iniciar un servicio inactivo, basta con marcar el servicio y presionar luego abajo el botón "Start". De manera similar, para detener un servicio, basta con marcar el servicio y presionar luego abajo el botón "Stop".
