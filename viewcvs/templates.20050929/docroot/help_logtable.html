The table based log view must be enabled in the site wide ViewCVS configuration file.

It displays the revision history of the selected source file in a table containing the following columns:

  • Revision: The revision number.
  • Tasks: This column contains several links to view a revision.
  • Diffs: Several links to view the changes between this and other revisions.
  • Branches and Tags: Displays the branch and a pulldown menu button displaying all tags bound to that revision
  • Age:The revision's check in-time
  • Author:The revision's author
  • Log:Log message describing the changes made in the revision

At the bottom of such a page you will find a form which allows to request diffs between arbitrary revisions.
