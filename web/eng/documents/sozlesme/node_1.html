Project Main Contract

2  Project Definition

2.1  Target

Creating a useful, stable, developable operating system with merging classical distribution concept named as HD and distribution infrastructure that named as Impulsive Technologies (DT).

2.2  Limits

  • Updates and all development on packages will be performed on existing distributions' work.
  • Central infrastructure that installer system, package management system and configuration managers use commonly will be prepared.
  • Before connective layer will be ready; installer, package manager and configuration tools will be developed without that layer or will be chosen from existing free softwares.
  • Distribution will be prepared for Turkish users first.

2.3  Resources

  • Free softwares
  • Existing Linux distributions
  • Open standart committees
  • Other projects that produces Impulsive Technologies
