About Project


What is the Uludağ Project?

Ulusal Dağıtım Project was started and still continued by TÜBİTAK - UEKAE. This project aims to develop a GNU/Linux, plans on answering the expectations and basic desktop requirements of the people who has informatics literacy by using excellent ways of Linux distributions, code or architecture with building tools that will make easy to use and setup operations.

Do you have a relation with Uludağ University?

No :)

What is the aim of Uludağ Project?

The aims of project are
  • to develop a widely-used
  • to form an organization that will continue this distribution.
  • to contribute the original technology by creating a eco-system using this distribution and open source software around the distribution.

Where the name "Pardus" comes from? Why is the name of distribution different from the name of project?

The name "Pardus" comes from Anatolian Leopard, Panthera Pardus Tulliana.

Anatolian Leopard is one of the last members of Leopard subspecies. Anatolian Leopard's length reaches up to 200-250 cm, females are 35-50 kg and males 45-70 kg in weight. Their lifetime is about 20 years. Strong and agile Anatolian Leopard feeds on wild ungulates, chamois, deer, boar and birds. Anadolu Leopards lived in forest and hill regions of Aegean,West Mediterranean, East Mediterranean and East Anatolia. Scarcity of hunting resources directed them to settling areas, because of this they were usually killed.

The last official record about Anatolian Leopard was in 1974-1-17. This record was about killing a Anatolian Leopard because of Leopard's attack to a woman Bağözü Village, 5 km far from Beypazarı. Most of the views are about to die out, but it is suggested that there is still 10-15 Anatolian Leopard in Turkey. In 2001, around Dandi region in East Mediterranean, around Müsikli Brook in East Blacksea; in 2004, in Pokut Plateau in East Blacksea region, it has been seen. Volunteers' works still continues to prove assets of Anatolian Leopards and to protect them.

Uludağ is the name of the project and Pardus is the name of one of the products of Uludağ Project.

Why Linux? Why don't you build all from scratch?

Both Linux kernel and the applications we use like Firefox Web Browser, OpenOffice.org, KDE, etc ... are free software and they have been developed after a long time by lots of people. Doing this work doesn't provide us more than spending money and time.

Because of said software being free, when we need to develop something about Turkish support or do cleaning for last-users, we can do it easily by editing the software. Submitting these software again to free software world makes us having a voice in free software eco-system and directing this eco-system to takes advantages.

Linux has been springboard for us to catch and to pass IT World's current level by putting original contributions on it.

Why National(Ulusal) Distribution(Dağıtım)? What makes you national?

Uludağ Project and its product Pardus will form quantum leap in Turkey to make Linux become widespread. Uludağ Project will do this by giving full and right Turkish support, leading to create a free software eco-system for projects and products. So we will have usage facilities,especially re-distribution and changing software, that are cited in licenses. By using Pardus and other Linux distributions, we will have software, data, knowledge and our technology by forming a road map for Pardus and determining priorities. Uludağ is the one leaders for Linux's being protected at nation scales, so it is "national (ulusal)".

Who develops this project? Is it a voluntary work?

Uludağ Project is being developed under Turkey Scientific and Techical Researck Assocation (TÜBİTAK) - National Electronic and Cryptology Research Institute (UEKAE) structure. All of the members of Uludağ Project are employees of TUBİTAK. Nonetheless, volunteers ,the count of them increases, helps for different parts of project. We plan that the number of contributors will increase and they will have a voice in project so a new distribution will be formed by them around the TUBİTAK's employees'.

How long will this project continue? What if TÜBİTAK leaves supporting?

Uludağ Project will continue until the aims, forming a free software eco-system and forming a organization to continue project, are completed. TÜBİTAK will support all works in accordance with these aims.

If TUBİTAK decides ,surprisingly, not to support Uludağ Project, Pardus will continue as a free software. Anyone or any company can continue to develop Pardus through GPL. TUBİTAK has rights for only PARDUS brand. The software owners are whoever uses it.

Where is Pardus developed? What should I do if I want to meet you?

UEKAE is in TUBİTAK Gebze Campus. Uludağ Project team is also there. Due to UEKAE's security checks, you have to ask an appointment for meeting.


Using Pardus

What is Pardus Live CD?

Pardus Live CD is a product of Uludağ Project that contains Linux kernel, software that most of them are free and developed by Uludağ team . It fully works on CD and doesn't need to be installed to hard drive. Pardus Live CD has been developed to test Pardus Operating System without installing to hard drive. You can use Pardus without damaging the other operating systems installed to your hard drive. You should only set BIOS to boot from CD.

For more information about Pardus Live Cd and usage, visit products page.

Some files on your web page have .md5sum extension. What is it?

MD5SUM files are used to control whether if the downloaded ISO files are damaged while downloading. When you open MD5SUM files with an editor, you will see a string and a file name. You can check the MD5SUM of ISO file with md5summer.exe application. If the result of MD5SUM of ISO file and the string in the MD5SUM file aren't the same, it means ISO file was damaged while downloading. In that you have to download the ISO file again. Linux users can do this with "md5sum" application. In console, use "md5sum -c file_name.md5sum" command to check if the ISO file was damaged or not.

How can I download Pardus?

To download Pardus Live CD, you can use following links;

ISO: ftp://ftp.uludag.org.tr/pub/pardus/calisan/1.0/pardus_calisan_cd-1.0.iso
MD5SUM: ftp://ftp.uludag.org.tr/pub/pardus/calisan_cd/pardus-calisan_cd-1.0.md5sum

After downloading these files, please check md5sum as told above.

What should I attend to while writing Pardus to cd?

If there is no problem with md5 summary, you can write ISO files to CD. ISO is a file system type for CDs so you shouldn't extract it with archive managers or copy the ISO file to CD. Otherwise, you won't have a bootable CD and be able to start Pardus. You should follow the instructions in help menu of the application , you use to write CD, to write ISO images (generally, specified as "writing an iso image" or "creating a bootable cd").

I want to use Pardus in my corporation. Do you give corporate support?

Uludağ Project doesn't have a infrastructure to give support. So, we are planning to solve this through business partners. In this case, corporate support may be given as a new project in UEKAE or from a structure which base is UEKAE. Business model its channels hasn't been formed yet so you won't be able to find ready solutions. But if you contact to bilgi at uludag.org.tr to be a business partner, we can form a new project together.

Can I use Pardus on my Macintosh?

Pardus was prepared and compiled for Intel and Intel-compatible processors. We don't have a plan for making a version for PowerPC processor, used in Macintosh computers.


License and Distribution

What is the license of Pardus?

The Software that prepared for Pardus by Uludağ Project are GPL. Most of software used in distribution are distributed under GPL or other free licenses. However, some of the hardware drivers which doesn't have free license may be distributed in Pardus for end users. The Software that prepared for Pardus by Uludağ Project will always be distributed under GPL.

How can I get more information about GPL license?

You can find GPL(General Public License)'s original text from http://www.gnu.org/licenses/gpl.html and Turkish translation of text from here.

Does this license permit me to copy and sell Pardus?

Yes, absolutely yes. You can copy, change and re-distribute(with cost or not) the software distributed under GPL. But you have to distribute the changes ,which you made, again free. Although the software under GPL can be distributed, because of Pardus and Pardus logos' being registered trademark, changed products shouldn't be distributed with the same name and logo.

Where can I find the source codes of applications in Pardus?

All software in development process are distributed through Subversion version control system. svn.uludag.org.tr/uludag stores software made by Uludağ Project and svn.uludag.org.tr/paketler stores free software used in Pardus and their patches.

Does Pardus have a road map? What will you do next?

Yes and no :). Pardus has a road map. But it doesn't have a published one. Works to be done are discussed in e-mail lists and presented to contributors. The contributors which wants to play role in road map give support to project.

Is there any probability for TÜBİTAK to close the source codes of Pardus?

Source code distributed under GPL can't be closed. But if there is no contribution to the next version of any software, it can be chose not to distribute it under GPL. If there is contribution, all contributors' approval must be received. For all we know, TÜBİTAK doesn't have intent to close the source codes but, if TÜBİTAK does, all software made until now will always be distributed under GPL.

Technical/Others

Can Pardus answer all my expectations?

In first step, Pardus will answer basic desktop expectations for users as described informatics literacy. After required infrastructure is completed and achieved the aims, it will developed to answer different expectations for different users.

Does Pardus have plugins like Java and Flash?

Plugins were added to Pardus as their licenses permitted. There are setup tools that answer expectations for the plugins that can't be added to distributions. For example; flash plugin which can't be added to distribution can be installed automatically by Firefox browser without a setting when a web page that contains flash is opened.

Where can I find visual materials for Pardus?

You can find visual materials and screen shots from Pardus Visual Store.
Can I use Pardus visual materials anywhere?

Pardus visual materials are registered and TÜBİTAK-UEKAE has all rights. However, you can use visual materials by getting permission.

Does Pardus support only Turkish Language? Can I use it in English?

Pardus's language is Turkish by default. But if you want to use in English, you can use it by changing the setting in "Country/Region and Language" setting under "Regional and Accessibility" menu which is in Pardus Configuration Center.

What are the hardware requirements to use Pardus?

For Pardus Live CD 1.0, the hardware requirements are;

Intel Pentium 2 or better (or compatible) processor
128MB RAM (256MB recommend)
CD-ROM (4x or better)
Graphic Card

How is the performance of Pardus?

Pardus works well on any PC contains minimum hardware requirements. But because of the number of applications you use simultaneously and the memory requirements of the applications you use; Pardus is faster as strong as your hardware is. For example; if you want to use office and multimedia tools simultaneously, you are recommend to use 256MB or better RAM.

Can I access to my documents in other operating systems with Pardus?

When you click "Devices" icon on the desktop, you see hard drive 's partitions and you can access to your documents on any partition by clicking on any partition icon.

To use the hard drive 's partitions, the must be mounted. The green arrow on partitions' icons shows mounted partitions. If the partitions are not mounted, you can mount them by clicking the mount option which comes when you right click on the partition icon.

Can I use my office documents in other other operating systems with Pardus?

Pardus contains OpenOffice package for office applications. You can use your documents that are in other operating systems by using OpenOffice software.

Can I read my e-mails, surf on Internet?

Pardus contains applications for users described as user who have informatics literacy to answer expectations for Internet. You can surf on internet(Firefox, Konqueror), read e-mails(Kmail, Thunderbird), follow RSS feedbacks(Akregator), connect to ftp servers(Konqueror), connect to ICQ, Jabber, MSN, AIM, Gadu-Gadu, YahooIM, IRC (Kopete, Konversation) in Pardus by using various applications so you can use all basic Internet services.

Can I play DVD/VCD/MP3 in Pardus?

Lots of software comes with distribution like Kaffeine, MPlayer, Juk, XMMS, Noatun, Kaboodle. By using these applications you can play audio files that have WAV, MP2/MP3, OGG/VORBIS, AC3, WMA extensions. You can play video files that have AVI, MPEG/MPG, MOV, WMV, QT, DAT extensions and watch live broadcasts which uses MMS, RTSP, HTTP protocols. Furthermore, again with these applications you can watch and listen video, audio and images in various disc formats like DVD, DLDVD, VCD, SVCD. Necessary codec files and libraries comes ready to use with Pardus.

Can I install Pardus Live CD to my computer?

While Live Cd was developed, its system files and settings were set to detect hardware automatically and use low memory. An expert user can create an installed system by copying the system files to hard drive and changing the settings but it is not recommended. When the installation cd of Pardus will be prepared, we will have a more easy to install and set so system so we suggest users to wait for installation cd.

Does Pardus damage files in my computer?

While Live Cd is running, it doesn't write to hard drives so it was designed not to damage files in your computer. However, if user wants to change files while using Pardus as in other distributions, for example editing a file or deleting it from hard drive, Pardus will enforce the order.

Reporting Bugs / Help

I found a bug, what should I do now?

When you find a bug in Pardus, you can report it to Pardus Bug Tracking System (http://bugs.uludag.org.tr/).

What is Uluzilla? How can I use it?

Uluzilla is the name of Pardus Bug Tracking System. The application called Bugzilla works on the base. After logging in with your Uluzilla account, you can report bug, ask for a request and write comments for other bugs, vote, track the bugs and requests by subscribing. Before reporting or asking a request , please search if the bug was reported or the request was asked from Bug Query page. If there is not a bug as yours, you can report it from New Bug Reporting page by selecting the category and filling the form. The informations about the bug you reported or the request you asked will be mailed to your e-mail address which you use while logging.

Should I know programming to help you?

Certainly no :) Ones that don't know programming can help as the ones that know programming.

I want to help. Where is the information about help?

We have a project team for the ones who wants to help : Quality Team. This team provides them to adapt the sub-projects and to help starting projects that will be developed by them. You can find information about Quality Team on Quality Team web page. You can contribute the projects specified in the Quality Team web page or direct the Quality Team with new ideas.

I wrote an application. Can you add it to Pardus?

There are some criteria to add an application to Pardus. First of all, it must be distributed under a free license and source codes must be accessed by public. It must call upon the users who have informatics literacy , be easy-to-use, be stable and the most important one; it must not be an alternative for the applications in Pardus (for example; a file manager instead of Konqueror won't be added to Pardus). You can use Uluzilla to ask the developers to add the software that obtains the criteria above.
