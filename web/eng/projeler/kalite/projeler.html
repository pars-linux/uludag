:: HCL - Hardware Compatibility List ::

With this application is expected to provide a user-interface and database, where users will be able to check whether any of their hardware is compatible with Pardus prior of use.

When the application is finalized it will be available at http://www.uludag.org.tr.


- Status: Design Phase.
- SVN: http://svn.uludag.org.tr/uludag/trunk/pardul/
- Responsible: Emre Saglam (design, management).
- Responsible: Faruk Eskicioglu (implementation).

 

:: UDSS - Uludag Enhancement Proposals System ::

The idea of setting up a system like GLEP (Gentoo Linux Enchancement Proposals) or PEP (Python Enhancement Proposals) was suggested in the Uludag list before. This application is expected to meet the requirements of an infrastructure where ideas and suggestions related with the project are stored as concrete, coherent and complete documents which can also be viewed by people so that these ideas do not get lost.
