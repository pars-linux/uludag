Packages Project

This project particularly deals with choosing the packages, fixing the bugs, notifying the upstream and handling localization problems. Packagers are expected to read the this guide (in Turkish) before applying for a SVN account.

Application selection, error correction, resolving problems peculiar to Turkish language, gathering information needed by PiSi and ÇOMAR and packaging in those information are all in scope of this project.
