Pardus Live CD 1.1

Pardus Live CD was prepared by Ulusal Dağıtım Project, for getting computer users ideas in Turkey about the Project and telling their critics and opinions in the time.

Live CD can be used without installation and doesn't harm data on your hard disc.

MD5SUM MD5SUM summary file is used for controlling the downloaded iso file's entirety and being sure of file doesn't harm during the downloading proccess. Please be sure that ISO file's summary is same with MD5SUM file's content, with this program: ftp://ftp.uludag.org.tr/pub/md5summer.exe If summaries are not same, this means that the file in your computer was damaged because of a connection problem during the ISO file's downloading process. Then you shouldn't write the damaged ISO file to CD and you should download the ISO file again. Linux users can do process of getting MD5SUM summary with "md5sum" named application.

If there is no problem with summaries, now you can write the ISO file via your CD-Writer. Iso is a knd of CD file System . So that you shouldn't open it with archive tools or you shouldn't copy it into a CD. Otherwise your CD will not be a bootable disc and Pardus can not start. Before To Start Working With Pardus

For opening your computer with Pardus, putting the Pardus Live CD to your CD-ROM Drive and restarting the computer is sufficent.

At this time if your computer starts with operating system which was on your hard disc, normally this shows that you can't set your BIOS or you can't write the CD correctly.

At this situation you should set your BIOS for starting from CD-ROM drive. For this in your mainboard's BIOS starting up settings, CD-ROM drive must be chosen as first. On the other hand, laptop users may need to select starting up devices via special key combinations which is determined by producers. If you tried but couldn't prosper or if you don't know how to do, you can get information from your mainboard's manual.

Starting Pardus

After your computer begin starting up with Pardus in the CD-ROM drive, you will see a manu which contains choices that you can move with direction keys and select with enter key:

  • Pardus Linux
  • Pardus Linux (HT or minimum 2 CPU systems)
  • Pardus Linux (minimum settings [vesa=standart])
  • Pardus Linux (minimum settings [acpi=off, apm=off, vesa=standart])
First choice is default and probably suitable for you. Second choice is which have more than one cpu. Third or fourth choices are for old computers or laptops which may have some problems and for some desktop systems. When you select one of these choices, Linux Kernel begin to be loaded to your memory. After a while you will see a progress bar, which shows that how much part of process have done, with green coloured Pardus screen. If your monitor gets dark and there is no signal for a long time, you should restart your computer with RESET button and choose 3rd or 4th choices from the menu. Although you try all choices, your computer can not be started up with Pardus because of hardware problems.

At the last part of starting, green coloured Pardus screen shuts down and desktop environment begin to start up. At this time if your screen gets dark or if you see "Signal Out Of Range" message, you can decrease the default resolution with CTRL, ALT and - key combination. After your graffical environment was started up, you can set the screen resolution again with configuration manager which is at the rigth bottom of the screen.

Later, desktop environment's starting up screen will appear and finally desktop will completely start up.

Working with Pardus

Along with starting up the desktop environment completely, Kaptan Masaüstü Desktop Manager will welcome you for setting the wallpaper and mouse.

You encounter with Starting Up, Devices, My Documents, Pardus Desktop Sharing and Trash icons on Pardus Desktop, wh,ch is completely started up with your wallpaper.Devices icon shows your storage devices on your computer and supplies to reach to your files on these devices. So you can display the files on your disc or you can save the files that you don't want to lose while you are working with Pardus.With Pardus icon, a local web page which contains purposes of Ulusal Dağıtım Project and project developers' communication information.

At the right side of toolbar there are several small applications that you can set volume level, screen resolution and keyboard layout with clicking the mouse. And you can reach to the starting up menuwith clicking Pardus icon at the left side of toolbar.

Starting up menu that you reach by clicking the Pardus icon at the left bottom of the screen, is a special menu for Pardus. At this menu, from Programs section, you can reach to the applications in Pardus. Also you can change the default configuration choices in Pardus, by reaching to the configuration center TASMA from menu.

"uludag" is the default username of Pardus Live CD. Also default and root user passwords are "uludag". When you use console by using "run command" or when you lock the screen, you will need to this password.
