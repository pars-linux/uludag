Why Lyx?

Gürer Özen <gurer@uludag.org.tr>

We prepare documents with lyx in Uludağ project. Reasons that let us to make this decision can summarize with points below.
  1. Lyx (http://www.lyx.org) files are plain text files, so, we can easily keep them in svn/cvs repositories. In big files, there is no problem about two people to edit documents' different parts at same time.
  2. Lyx files in fact a similar and easy to parse copy of TeX/LATEX files. To convert files without installing Lyx, there is a perl script at http://www-hep.colorado.edu/~fperez/lyxport/ address.
  3. It uses existing document formats, so, format problems can't make slower the production.
  4. Users didn't need to fight with Lyx or LATEX, Lyx files can be converted to lot of format with high quality.
    • LATEX provides higher print quality than other softwares.
    • We can convert it to HTML and publish it on web.
    • We can convert it to PDF and publish as one file.
  5. Lyx is easy to learn, no need to know LATEX.
  6. Lyx has ports to OS/2, Windows and OS X excluding Unix.
  7. We have been started to write with Lyx while OpenOffice.org is starting.
