Gestionnaire Réseau

Le Gestionnaire réseau peut être utiliser pour configurer les paramètre réseaux pour, vous connecter à internet, votre réseau local, le réseau sans fil ou connection par modem. C'est une interface simple à utiliser capable de vous connecter partout en quelques secondes.

Lorsque le programme est lancé pour la première fois, vous devez choisir le type de connection et configurer les paramètres. Ci-dessous, vous pouvez voir les différents types de schémas de connection.

Connection via une carte réseau ethernet

Pardus Linux devrait déjà reconnaître votre carte ethernet pour vous connecter à un réseau ethernet. Suivez ces instructions pour vous connecter de cette manière.

  1. Lancer le Gestionnaire Réseau
  2. Cliquer sur "Créer".
  3. Choisir "Connection Ethernet"
  4. Cliquer sur "Créer la connection"
  5. Entrer le nom de la connection et choisissez une carte ehternet
  6. Choisissez DHCP ou entrer votre adresse IP
  7. Cliquer sur "Utiliser" et sorter de la fenêtre
  8. Cliquer sur "Connecter".
Vous serez alors immédiatement connecté au réseau et une icône verte va apparaître.

Pour les utilisateurs ayant un modem ADSL ou cable, choisir DHCP devrait suffir.

En cliquant sur "Paramètres généraux, vous pouvez :

  1. Choisir un nom d'hôte pour votre ordinateur, ou
  2. Sélectionner un serveur DNS

Habituellement, dans les environnements réseaux où les adresses IP sont distribuées automatiquement (via DHCP), choisir DHCP devrait suffir pour permettre aux ordinateurs d'obtenir leur adresse IP et de se connecter à internet.

Connection via une carte sans fil (wifi)

Une connection via carte sans fil apparaît exactement comme une connection ethernet. La différence la plus notable étant le nom du réseau sans fil, c'est à dire le ESSID. Certain réseaux nécessitent de choisir ce nom pour fonctionner.

  1. Lancer le Gestionnaire Réseau
  2. Cliquer sur "Créer"
  3. Cliquer sur "Réseau sans fil" puis "Créer une connection"
  4. Entre le nom de la connection, choisissez la carte réseau sans fil puis le ESSID.

Vous pouvez également cliquer sur "Scanner" et obtenir la liste des réseaux sans fil qui sont vus par votre système.

Connection par modem

Suivez les étapes suivantes pour vous connecter par modem. Notez que certains modems sont spécifiquement conçus pour d'autres systèmes d'exploitation et peuvent ne pas fonctionner sous Linux.

  1. Lancer le Gestionnaire Réseau
  2. Cliquer sur "Créer"
  3. Cliquer sur "Réseau Téléphonique" puis "Créer une connection"
  4. Entrer le nom de la connection, le port série utilisé par le modem (habituellement COM1) puis le numéro de téléphone de votre fournisseur d'accès.
  5. Cliquer sur l'onglet "Identification"
  6. Entrer votre nom d'utilisateur et votre mot de passe

Maintenant cliquer sur "Connection" devrait activer votre modem.
