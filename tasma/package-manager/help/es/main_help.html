Administrador de Paquetes

Administrador de Paquetes se utiliza para buscar, instalar o actualizar paquetes del repositorio de paquetes de Pardus. Es una herramienta práctica y útil para administrar los paquetes de manera fácil.

Instalando software nuevo

Para instalar un nuevo paquete, pinche la pestaña "Mostrar paquetes nuevos" y ubique el software que le interesa. Pinchar la pestaña "instalar paquetes" mostrará los paquetes y dependencias que se instalarán.

El tiempo de instalación dependerá de su ancho de banda del acceso a internet. Se mostrará el progreso durante la instalación de los paquetes.

Remover un paquete

Para remover un paquete pinche la pestaña "mostrar paquetes instalados". Seleccione el paquete a remover y luego pinche "Remover paquetes". Se removerán los paquetes seleccionados del sistema.

Mostrar actualizaciones

Por omisión se ve en la barra de tareas las actualizacioes disponibles. Al abrir el administrador de paquetes pinche "Mostrar actualizaciones" para ver las actualizaciones del sistema disponibles. Pinche "Actualizar paquetes" para actualizar el sistema.

Buscar paquete

Para buscar un paquete, simplemente escriba su nombre en la barra de busqueda. Se mostrará una lista de paquetes que coinciden con el término de busqueda.
