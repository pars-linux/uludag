Pardus 2007 release notes

1. Welkom bij Pardus 2007

Graag willen we u welkom heten bij Pardus versie 2007, een eenvoudig te installeren en te gebruiken operating system waarin zich vele bruikbare, compatible en veilige ontwikkelingen voor de K Desktop Environment (KDE) bevinden. Met Pardus, kunt u op het Internet, uw e-mail lezen, veilig communiceren met vrienden en de computer's multimedia capaciteiten gebruiken door naar muziek te luisten of films te bekijken. We nodigen u graag uit een bezoek te brengen aan www.pardus.org.tr/eng of www.pardus-linux.nl om te ontdekkken hoe Pardus kan beantwoorden aan uw informatica behoeften.

1.1 Feedback

U kunt bugs.pardus.org.tr, oftewel het Pardus bug tracking system gebruiken om een bug in te sturen, een bug fix of elke suggestie aangaande Pardus. We willen u graag bedanken voor uw warme belangstelling, uw bijdrage door, het raporteren van een probleemm een suggestie voor een nieuw pakket of verzoek voor een feature voor de volgende release.

Tevens kunt u de feedback applicatie gebruiken die in Pardus 2007 zit om uw ideeen te versturen over Pardus via het Internet.

2. Standaard eigenschappen

Deze alinea beschrijft de features die komen met Pardus 2007.

2.1 Desktop gebruik (KDE 3.5.5)

De K Desktop omgeving, versie 3.5.5, die geleverd wordt bij Pardus 2007, voldoet aan de noodzakelijheden en verwachtingen van de verwende computergebruikers, samen met nieuwe kleuren schema' s, menu ontwerpen en taalsupport in meer dan 65 talen, snelle multimedia toepassingen en eenvoudige toegang tot het Internet en nog veel meer. Ongeveer 1500 grote en minder grote verbeteringen zij gemaakt bij deze versie van KDE. De belangrijkste is de 20% snellere installatie. Verder zij er nieuwe features toegevoegd aan de componenten KMail, Akregator, Kopete en KPDF.

2.2 Communicatie en compatibiliteit met Windows

Pardus 2007 is compatibel om samen te werken met andere besturingssystemen in een network. Het kan hard disks, printers en andere drives, gedeeld in een windows omgeving, gebruiken. De hard disks die gebruikt worden door Pardus 2007 kunnen op dezelfde wijze worden gebruikt in een windows omgeving.

2.3 Java support

De Sun Microsystems Java development omgeving wordt meegeleverd in de Pardus 2007. Op Java gebaseerde progamma's, platform onafhankelijke applicaties, werken in deze onafhankelijke omgeving. Tevens is er een directe toegang tot grote Java library' s in de wereld.

2.4 Games en amusement

De vele games die werden meegeleverd bij Pardus 2007 geven u de gelegeheid uw tijd leuk door te brengen. Tevens kunt u door een paar klikken spellen downloaden uit de Pardus depots.

2.5 Netwerk gebruik

Pardus kan worden gebruikt als service provider voor de genoemde network services hieronder.
  • File server (Met Samba software)
  • Print server (Met Cups software)
  • Web server (Met Apache software)
  • E-mail server (Met Postfix software)
  • FTP server (Met Vsftpd software)
  • OpenLDAP server (Met Openldap software)

3. Installatie benodigdheden

Pardus zal over het algemeen werken op gemiddelde hardware. Minimaal 512 MB RAM (geheugen) en een 1200 MHz processor zijn nodig voor een redelijke prestatie. De basis hardware onderdelen die er voor zorgen dat Pardus doorgaans prima werkt worden hier onder opgesomt.
  • 256 MB intern geheugen (beter is 512 MB)
  • 800 MHz Intel of AMD processor (advies minimaal 1200 MHz)
  • 4 GB vrije hard disk ruimte

4. Verbeteringen in deze versie

De Pardus 2007 versie ziet het licht na een ontwikkelperiode van meer dan 6 maanden. Deze versie bevat de verbeteringen, hieronder opgesomd, gemaakt naar aanleiding van feedback van eindgebruikers.

4.1 Pardus firewall

De Pardus firewall eenvoudig in het gebruik, met als doel aanvallen van buiten af te slaan. Het draait constant op het system en beschermd uw computer tegen aanvallen. Met de Pardus firewall, kunt u verbindingen met uw machine via een specifieke poort desgewenst uitschakelen of de verbindingen bekijken.

4.2 Verbeterd package management system

PiSi gebruikt nu het LZMA algoritme, wat als resultaat heeft een 30% snelheidsverbetering en u kunt meer programma's gebruiken. Met deze verbetering, PiSi packages en de gedownloade software door eindgebruikers, verkleind in grootte, is het mogelijk meer software te leveren samen met Pardus 2007.

4.3 De nieuwe software management interface

De PISI package management interface wordt vervangt door de makkelijk te gebruiken package manager. Deze nieuwe software voorziet in installatie, de-installatie en het eenvoudig en effectief updaten van programma's.

4.4 Verbeteringen in het YALI installatie systeem

In YALI zijn diverse bugs gefixed en het is nu geconfigureerd om samen te werken met COMAR en PISI. Verder is de YALI interface gecallibreerd en voorzien van nieuwe kleuren.

4.5 Het nieuwe en verbeterde Pardus startup system

Het activeren van de kernel start wanneer het login scherm verschijnt aan de gebruiker, gemanaged door het opstartsystem. Dit bevat het mounting systeem, herkennen van de hardware en installeren van hun drivers, de basis network configuratie en de opstartsysteem service. Het is ook verantwoordelijk voor het stoppen van de diensten, het unmounten van de filesystemen met een juiste afsluiting wanneer de computer stopt.

De broncode van het Pardus startup systeem is herschreven van de bodem af omdat het startup systeem in Pardus 1.0 te traag was en slecht in het herkennen van de hardware. Het nieuwe systeem begint met met "mudur" (wat "manager" betekent in het Engels) dat is een startup script geschreven in Python en "muavin" (wat "assistent" betekent, watkijkt naar de hardware modules en deze installeerdook geschreven in Python. Scripts van systeem services worden gemanaged door Comar en hebben het voordeel van ' parallel processing' en een gemakklijk gebruik. Met deze verbetering:
  • Een belangrijke tijdsbesparing in het boot en shutting down proces,
  • Pardus 2007 heeft een veel meer flexibele en makkelijk te onderhouden distributie door de kracht van Python.
  • Services zijn ontworpen en geprogrameerd met de mogelijkheid om beheerst te worden via de command prompt of heel gemakkelijk met TASMA.

4.6 Meer hardware identificatie

Het nieuwe hardware identificatie systeem, specifiek voor Pardus, maakt het mogelijk om nagenoeg alle hardware types op de markt te herkennen en mee samen te werken. Omdat het handmatige herkenning van hardware minimaliseerd, blijven er nauwelijks vragen over wat betreft drivers. Pardus kan de hardware automatisch identificeren en installeren tijdens het boot proces van het systeem.

Het hardware identificatie systeem Muavin voorkomt het onnodig laden van modules wat resulteerd in het besparen van werkgeheugen. En verder geen gebruik maken van het hotplug systeem wat oude technieken zijn, de nieuwere en meer flexibe technology genaamd udev wordt gebruikt voor dynamische 'device naming'.

4.7 Meer dan 1200 bugs gerepareerd

Meer dan 1200 bug reports gezonden naar het bug tracking system van Pardus zijn bekeken en nauwgezet verholpen door de ontwikkelaars zodat de release van 1.0 met een sterk en stabiel gebruiksvriendelijk systeem opleverde voor de gebruiker.

4.8 Nieuwe 2.6.18 kernel

De meeste hardware problemen zijn opgelost bij de 2.6.18 kernel die geleverd wordt bij Pardus. Ondersteuning voor USB ADSL modems die geleverd wordt met deze kernel, geeft de mogelijkheid om meer USB ADSL modems te indentificeren dan de oudere releases. Verder is het boot probleem opgelost, wat veroorzaakt werd door APIC/ACPI bij oudere machines. De kernel bootup tijd is gereduceerd met 3 seconden door de patch voor identificering van de IDE harddisks.

Een paar van de nieuw ondersteunde harware :
  • Nieuwe SATA devices
  • USB ADSL modems
  • Verbeterde wireless network devices support
  • ondersteuning voor meer cameras, scanners ed printers

4.9 Nieuwe iconen

Een van de meest gewardeerde en karakteriserende items was de Pardus 1.0 "Tulliana" icon set. Deze sets zijn verbeterd in Pardus 2007 en extra icons zijn toegevoegd. Het is nu genoemd Tulliana 2.0 icon set, het bevat meer uniformiteit met de Pardus desktop en andere applicaties en geeft je meer plezier met je desktop.

4.10 Vernieuwd gebruiker management programma

Van af 2007, is "user-kga" veranderd in "user manager" wat kan worden gebruikt om nieuwe gebruilkers toe te voegen in Pardus. Dit nieuwe gereedschap geeft een eenvoudige manier om gebruikersrechten te beheren, zoals het gebruik van de CD drive, het luisten naar muziek en het spelen van spelletjes.

4.11 Nieuwe applicatie: Service manager

Een service manage programma wordt meegeleverd 2007 om gemakkelijk de server en de services te heren van en bij het booten. Met deze manager, worden de services opgestart tijdens het booten en verder wordt elke service gestart en gestopt als nodig.

4.12 Software updates (xorg, gcc, ..)

Met Pardus 2007, zijn veel bugs gefixed die verband hebben met Xorg en belangrijke updates gemaakt voor videokaarten. de laatste stabiele release van de 3.4 serie van de GNU C compiler (gcc) is inbegrepen bij Pardus 2007. Deze release bevat goede eigenschappen voor C++ software programmeurs. Glibc en binutils packages zijn gecompileerd met de nieuwste patches (-Bdirect, -hashvals, -dynsort), daarom wordt er een startup performance versnelling verwacht voor de programa's.

4.13 Verbeteringen in Skype, Kopete en andere communicatie tools

Skype is een complete IP telefoon oplossing dat gebruikt maakt van het H.323 protocol. Je kunt bellen met met Skype dat meegeleverd wordt met Pardus. Verder kan je een camera die ondersteund wordt bevestigen en een video conferencie houden met andere gebruikers van en via Kopete.

4.14 Meer localizatie support voor de Nederlandse taal

Er zijn veel verbeteringen gemaakt in de Nederlandse localizatie support in OpenOffice.org, Firefox, Thunderbird, GNOME en KDE applicaties. Alle desktop applicaties kunnen in het Nederlands via deze verbeteringen. We willen graag KDE Nederland bedanken, OpenOffice.org, GNOME en de Mozilla werk groep voor hun buitengewone inspanningen.

Nb: Pardus 2007 is voorzien van support voor zowel in het Nederlands, Turks als het Engels.

4.15 Niewe release OpenOffice.org 2.0.4

Deze nieuwe release van OpenOffice.org bevat vele fixes en ook geeft het een 23% performance verbetering met de Calc applicatie. Het levert een verbeterde mogelijkeheid om met MS Office file types te werken. In OpenOffice.org 2.0.4, tevens zijn er meer mogelijkheden om met PDF file te bewerken alvorens deze te printen. Een andere optie in deze release is de mogelijkheid om MS Basic spreadsheet en read-only MS Access compatibility in de database module.

4.16 Niewe build farm

Met Pardus 2007, worden alle voorbereide PISI packages klaargemaakt in de "farm hence" zodat ze kunnen worden gedownload in een kortere tijd. De '"Build farm" bestaat uit twee sterke machines en voorziet in processor power met een open toegang naar alle Pardus developers.
