Welcome to Pardus

Pardus is an open source, freely distributable, GPL (General Public License) licensed user-friendly GNU/Linux distribution with many usability, hardware compatibility, stability and security enhancements. It's named after the Anatolian Leopard.

Pardus officially supports 11 languages: Turkish, English, Dutch, French, German, Spanish, Catalan, Italian, Swedish, Polish and Brazilian Portuguese.

The Pardus project is sponsored by the Turkish National Research Institute of Electronics and Cryptology (UEKAE), an affiliate of the Scientific and Technological Research Council of Turkey (TÜBİTAK).

This site is aims to help people who would like to help improve Pardus in various ways. For further information about Pardus, please visit http://www.pardus.org.tr/eng

How To Contribute

Bug Triage
If you say you don't have the time to spare for development or to look after packages, but you have technical skills and will to help us, then we will be glad if you help us with these despicable bugs.

Testing
We are continuously improving Pardus, but we are still human. Maybe you can help us see our mistakes, by trying out our latest products, before we release them to the wild.

Projects
If you are willing to help us improve Pardus and/or help with the packages, this is the place.

Packaging
Packaging is easy and a fun job to do. Especially if you are a new programmer. However, you should be aware that packages do need some love time to time :)

Translation
Pardus already speaks 11 different languages, but we always need help to maintain the support for these languages. Maybe you can help Pardus to speak one more language?

Documentation

Graphics and Multimedia
Do you have special skills on graphical design or multimedia? We would like to see your ideas come true and become a part of Pardus experience.

Browsing Pardus Repositories

You can browse our Subversion repository through Pardus SVN Web Interface or you can use SVN to checkout it:

Individual technologies 
  svn co https://svn.pardus.org.tr/uludag/trunk/PROJECT_NAME
Source package repository 
  svn co https://svn.pardus.org.tr/pardus/RELEASE_NAME
