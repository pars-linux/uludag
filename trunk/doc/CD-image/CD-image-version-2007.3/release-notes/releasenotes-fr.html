Notes de livraison de Pardus 2007.3

1. Bienvenue sur Pardus 2007.3

Bienvenue sur la version 2007.3 de Pardus, une nouvelle version géniale de Pardus de la part de vos développeurs, traducteurs, artistes et chargés de documentations Pardus. Cette livraison comprend un ensemble d'outils nouveaux couplés à toutes les mises à jour depuis la sortie de Pardus 2007.2.

Nous vous invitons cordialement à visiter www.pardus.org.tr/eng pour plus d'informations.

1.1 Feedback

Vous pouvez utiliser bugs.pardus.org.tr, alias le système de suivi de bug Pardus afin de signaler tout bug, correction de bug ou suggestion relative à Pardus. Nous voulons également remercier nos utilisateurs ainsi que nos développeurs pour leurs contributions diverses à cette version.

Vous pouvez également utiliser l'application de feedback disponible depuis votre menu afin d'envoyer vos idées à propos de Pardus via Internet.

2. Fonctionalités de base

Ce chapitre décrit les fonctionnalités de Pardus 2007.3

2.1 Un gestionnaire de réseau (alias Network Manager) remis au goût du jour

Network manager permet de détecter vos profils sans-fil et automatiquement basculer de l'un à l'autre. Il vous tient informé par popup depuis votre barre des tâches ce qui le rends plus intuitif et plus facile à appréhender.

2.2 Un Gestionnaire de paquet amélioré

Package manager (alias gestionnaire de paquet) est maintenant plus abouti. De nombreux bugs signalés ont été corrigés.

2.3 YALI

YALI (yet another Linux installer -- un installeur de Linux de plus) pose d'abord les questions puis installe le système. La procédure d'installation est plus flexible et plus rapide que jamais. L'utilisateur peut également choisir de lancer la session automatiquement ou non. Cerise sur le gateau, YALI inclus les définitions pour les disques durs les plus récents afin de simplifier le processus d'installation.

Ajoutons également -et c'est une première ! Que YALI sait détecter quel disque est amorçable et lequel ne l'est pas, et décider sur quel disque installer Pardus.

2.4 Des logiciels mis à jour

KDE, l'environnement bureautique principal de Pardus 2007.3 est maintenant aligné sur la 3.5.8. La suite bureautique complète OpenOffice.org est servie en version 2.3 alors que K3b, l'application de gravage de CD et DVD est désormais à la version 1.0.4

3. Configuration requise

Vous pouvez utiliser Pardus 2007.3 sur des configurations moyennes. Nous vous suggérons toutefois au moins 512 Mo de Ram (mémoire) ainsi qu'un processeur cadencé à 1200 MHz pour avoir des performances visibles. Les configurations minimales et conseillées pour Pardus 2007.3 sont indiquées ci-dessous :
  • 256 Mo de mémoire (512 MB recommandés)
  • Processeur Intel ou Amd cadencé à 800 MHz (1200 MHz recommandés)
  • Au moins 4 Go d'espace disque disponible (10 Go recommandés)

4. Contact

Veuillez visiter le site web Pardus (www.pardus.org.tr/eng) pour de plus amples informations.
