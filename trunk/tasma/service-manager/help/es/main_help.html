Administrador de Servicios

Pardus ejecuta algunos servicios durante el arranque del sistema. El usuario del sistema puede controlar este proceso, quiere decir que se pueden deshabilitar algunos servicios del arranque, y pueden ser habilitados de nuevo. Se puede utilizar el Administrador de Servicios para la configuración de esta tarea.

Un LED indicador a la iquierda del nombre de servicio muestra el estado del servicio correspondiente. Si está rojo, quiere decir que el servicio no se está ejecutando. Si está verde, quiere decir que en este instante el servicio está activo.

Para ejecutar un servicio parado, basta con marcar el servicio y apretar después abajo el botón "Start". De manera similar, para parar un servicio, basta con marcar el servicio y apretar luego abajo el botón "Stop".
