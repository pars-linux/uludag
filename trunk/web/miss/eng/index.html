What's New in Pardus 2007.1 Felis chaus

Pardus speaks Brezilian Portugese // It is possible to install Pardus in Brezilian Portugese besides Turkish, English, Spanish, German or Dutch now...

Up-to-date applications // Felis chaus users will enjoy KDE 3.5.6 with the latest release of more than 300 softwares...

Package manager // PiSi is now 40% faster for reposity processes and the available number of packages are increasing...

Better network managing // WPA authentication, status monitoring and improvements have been done with network-manager, wireless networks are easier now...

And what remains the same

Pardus, comes with internet tools, office suite, multimedia (picture, music, video, etc) players, games and numerous applications to satisfy all the needs of desktop users, all squezed in a single CD...
