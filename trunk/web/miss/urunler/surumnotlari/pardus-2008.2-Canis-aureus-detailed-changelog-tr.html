  2. Yeni paketler ^^^

    • babl

      • babl is a library that allows converting between different methods of storing pixels known as pixel formats that have with different bitdepths and other data representations, color models and component permutations.

    • br2684ctl

      • br2684ctl is a utility for configuring RFC 2684 ATM/Ethernet bridging. ATM bridging is a way to extend Ethernet over an ATM network and is mainly used for DSL connections. This package contains the user space utility needed to configure the kernel driver. This package is needed if you own an USB DSL modem and your connection uses one of these protocols: RFC 1483 bridged (RFC 2684 bridged), PPP over Ethernet (PPPoE).

    • broadcom-wl

      • This package contains Broadcom's IEEE 802.11a/b/g/n hybrid Linux device driver for use with Broadcom's BCM4311, BCM4312, BCM4321, and BCM4322 based hardware. You must read the LICENSE.txt file in the docs directory before using this software.

    • ccid

      • This package provides the source code for a generic USB CCID (Chip/Smart Card Interface Devices) driver and ICCD (Integrated Circuit(s) Card Devices).

    • disk-manager

      • Disk Manager is an application used to administer disks on Pardus.

    • eciadsl

      • Linux driver for ECI HiFocus ADSL USB and Globespan based modems. EciAdsl is a free Linux driver for connecting your computer to the internet, if you have a Globespan based ADSL USB modem.

    • gegl

      • gegl (Generic Graphics Library) provides infrastructure to do demand based cached non destructive image editing on larger than RAM buffers. Through babl it provides support for a wide range of color models and pixel storage formats for input and output.

    • hcfpcimodem

      • Linuxant's modem driver for Connexant HCF chipset (PCI)

    • hunspell

      • Hunspell spell checker is an improved replacement for myspell in OOo. It is also used in OOo 2, Firefox 3, Thunderbird and Google Chrome as spell checker.

    • iwlwifi5000-ucode

      • Contains firmware data files for the Intel WiFi Link 5000 series wireless adapters.

    • libopenraw

      • libopenraw is an ongoing project to provide a free software implementation for camera RAw files decoding. One of the main reason is that dcraw is not suited for easy integration into applications, and there is a need for an easy to use API to build free software digital image processing application.

    • libshout

      • Libshout is a library for communicating with and sending data to an icecast server. It handles the socket connection, the timing of the data, and prevents bad data from getting to the icecast server.

    • libssh2

      • Library implementing the SSH2 protocol as defined by Internet Drafts: SECSH-TRANS, SECSH-USERAUTH, SECSH-CONNECTION, SECSH-ARCH, SECSH-FILEXFER, SECSH-DHGEX, SECSH-NUMBERS, and SECSH-PUBLICKEY.

    • linux-atm

      • Contains tools for Asynchronous Transfer Mode. Supports raw ATM connections (PVCs and SVCs), IP over ATM, LAN emulation, MPOA, Arequipa, and some others.

    • lua

      • Lua is a powerful light-weight programming language with all the fancy tools

    • microcode-data

      • Microcode data for Intel IA32 CPU's

    • p7zip

      • p7zip is a port of 7-Zip archiver for Unix. Supports packing-unpacking 7z, ZIP, GZIP, BZIP2 and TAR archives and unpacking RAR, CAB, ISO, ARJ, LZH, CHM, Z, CPIO, RPM, DEB and NSIS arhives.

    • pcsc-lite

      • pcsc-lite is middleware to access a smart card using SCard API (PC/SC).

    • rp-pppoe

      • rp-pppoe is a PPPoE client and server suite. It is fully RFC-compliant and supports cookies, relay-IDs, and multiple simultaneous PPPoE discovery phases.

    • rtl8187se

      • rtl8187se supports wireless network controllers of Asus eeepc-701SD, MSI Wind and Datron netbook.

    • sdricoh_cs

      • Linux kernel driver for some Ricoh SecureDigital Cardreaders that announce themselves as RICOH Bay1Controller pcmcia cards

    • syntekdriver

      • Driver for webcams with Syntek DC-1125 chipsets

    • tcpdump

      • Tcpdump is a tool to print out (or save to file) the contents of packets on a network interface. It also allows limiting the number of packets collected using expressions.
