Mailing Lists

-= Pardus-kullanicilari =-

General discussion list of Pardus. You can ask any kind of technical questions in this list.

You can visit http://liste.pardus.org.tr/mailman/listinfo/uludag for subscriber actions, and http://liste.pardus.org.tr/uludag/ to read list archives.

-= Turkish =-

Localization and Turkish workings goes on this list. At this list there is different workings like solving Turkish problems, finding equivalent Turkish words of terms, generating collective dictionary database.

You can visit http://liste.pardus.org.tr/mailman/listinfo/turkce for subscriber actions, and http://liste.pardus.org.tr/turkce/ to read list archives.

Jabber Server

It's Pardus instant messaging service. To use this service only thing is having a installed jabber client (can be downloaded from jabber.org) and connect to jabber.pardus.org.tr. You can also get developers' Jabber addresses from about page.

Jabber server address is jabber.pardus.org.tr.

Bugzilla

Popular bug tracking system Bugzilla is (also) choice of Pardus Project. Enhancements and bugs about Pardus services, development tools and development products can be tracked from here, will be updated or new ones added. Developers keep informed about every change.

You can access Pardus bugzilla at http://bugs.pardus.org.tr.
