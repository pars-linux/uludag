Contact

E-Mail Lists

-= Uludag =-

General discussion list of Uludag Project. You can share your questions, comments, critiques and suggestions about project. By tracking this list you will get information about project status.

You can visit http://liste.uludag.org.tr/cgi-bin/mailman/listinfo/uludag for subscriber actions, and http://liste.uludag.org.tr/uludag/ to read list archives.

-= Pardus Technical =-

Technical discussion list of Pardus, only technical issues will be discussed in this list. You can ask questions about problems with applications in Pardus.

You can visit http://liste.uludag.org.tr/cgi-bin/mailman/listinfo/kalite for subscriber actions, and http://liste.uludag.org.tr/kalite/ to read list archives.

-= Quality =-

Uludag Project Quality Team's workings goes on this list. This list aims that create a starting point for all people at any level who wants help and make faster projects mechanism. Before getting a member of this list you must read "How can I help? (Quality)" document located on documents page.

You can visit http://liste.uludag.org.tr/cgi-bin/mailman/listinfo/kalite for subscriber actions, and http://liste.uludag.org.tr/kalite/ to read list archives.

-= Turkish =-

Localization and Turkish workings goes on this list. At this list there is different workings like solving Turkish problems, finding equivalent Turkish words of terms, generating collective dictionary database.

You can visit http://liste.uludag.org.tr/cgi-bin/mailman/listinfo/turkce for subscriber actions, and http://liste.uludag.org.tr/turkce/ to read list archives.


Jabber Server

It's Uludag's instant messaging service. To use this service only thing is having a installed jabber client (can be downloaded from jabber.org) and connect to jabber.uludag.org.tr. Otherwise you can get developers jabber addresses from about page.

Server address is jabber.uludag.org.tr.

Bugzilla

Popular bug tracking system Bugzilla is (also) choice of Uludag Project. Enhancements and bugs about Uludag Project services, development tools and development products can be tracked from here, will be updated or new ones added. Developers keep informed about every change.

You can access Uludag Bugzilla at http://bugs.uludag.org.tr.
