Project Main Contract

1  Introduction

1.1  Vision

Uludag is GNU/Linux distribution developed by UEKAE according to computer literates basic desktop needs; uses existing distributions' dominant parts as concept, architecture or code; provides easy use, configuration, installation with configuration environment and tools that can be converted to an autonomous system.

1.2  Supporter

TÜBİTAK/UEKAE

1.3  Project Leader

Erkan Tekman, Project Manager

1.4  Suppositions

  • Project will be GPL'd free software project.
  • Project will be public, compatible with free software philosophy, use contributors' work.
  • Project can transfer other Linux distributions' architecture and components, will use open source (distribution independent) projects directly.
  • Localization works for free softwares will be used directly.

2  Project Definition

2.1  Target

Creating a useful, stable, developable operating system with merging classical distribution concept named as HD and distribution infrastructure that named as Impulsive Technologies (DT).

2.2  Limits

  • Updates and all development on packages will be performed on existing distributions' work.
  • Central infrastructure that installer system, package management system and configuration managers use commonly will be prepared.
  • Before connective layer will be ready; installer, package manager and configuration tools will be developed without that layer or will be chosen from existing free softwares.
  • Distribution will be prepared for Turkish users first.
