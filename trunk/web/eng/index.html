  Pardus 1.0 will be out today!

In accord with the Project Main Contract that has been issued in October 2004, and complying with the Pardus 1.0 Roadmap that has been published in October 2005, Pardus 1.0 operating system has been readied for final release and the CD image will be available for download today.

You may easily install and try Pardus 1.0 using the tools like YALI (Yet Another Linux Installer), an efficent package manager with a small footprint ideal for a wide range of applications (PiSi), and the latest release of innovative configuration manager (ÇOMAR) as well as Turkish NLP library (Zemberek).

Currently, Web page translation work has begun for English language. We are planning to finish this work as soon as possible. Please try again and thank you for your understanding.

Pardus is an open source operating system and freely distributed under the terms of GNU General Public License.
