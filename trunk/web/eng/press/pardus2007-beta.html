Pardus codename "ATA" released

Pardus has reached an important level of improvement with its latest beta version, with code name “ATA”. New version, with a renewed interface and improved attributes, also carries the significance of a preview of what Pardus final release will look like.

Pardus Linux, code name “ATA” incorporates a range of features enhancing the usefulness for personal computing. Some highlights are OpenOffice.org 2.0.4, KDE 3.5.3, YALI installer 1.1 beta7 and Firefox 1.5.0.7 together with lots of entertaining games, educational software, desktop publishing packages, digital camera software and painting programs freely downloadable with one click from Internet. Pardus “ATA” is geared towards desktop users, developers, Linux enthusiasts and hobbiests.

The start up framework of Pardus now uses COMAR and helps to speed up its opening process, being comparatively faster than the previous versions.

Thanks to the developments that are done to the package manager of Pardus, i.e. PiSi, it now uses less memory and runs faster. PiSi’s user friendly and pleasant graphic interface makes it easier both to update and install the software.

The new beta version’s visual elements are updated, too. The next version will feature Tulliana 2, a new icons set developed specifically for Pardus. Pardus “ATA”, comes as a single CD which Dutch, English, German and Turkish support on the desktop, internet tools, office software and multimedia players (photo, music, video etc.), games and countless of other productivity software.

“ATA” can be downloaded from Pardus servers: Developers are also working on a server version, which is planned to be released during 2007 Q2.
