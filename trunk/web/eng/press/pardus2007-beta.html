New Pardus beta, codenamed "ATA" has been released

Pardus operating system's latest beta version, codenamed “ATA”, is out for download and testing. This beta version will be followed by the stable version Pardus 2007, to be released on December 18th, 2006. Parud is a Linux based operating system, developed under the auspices of TUBITAK (The Scientific and Technological Research Council of Turkey) UEKAE (National Electronic and Cryptography Research Institute).

Pardus “ATA” incorporates a range of features enhancing the productivity and usability for personal desktop computing. Some highlights are OpenOffice.org 2.0.4, KDE 3.5.3, YALI installer 1.1 beta7 and Firefox 1.5.0.7; together with lots of entertaining games, educational software, desktop publishing packages, digital photography software and painting applications. Pardus “ATA” is geared towards desktop users, developers, Linux enthusiasts and hobbiests; and is freely downloadable with over the internet.

The boot-up system of Pardus now uses COMAR and shortens the start up time, compared to both the previous versions, most of the other Linux distributions, and competing operating systems.

PiSi, the package manager of Pardus, now uses less memory and runs faster. With the user friendly and pleasant graphic interface of PiSi installing and updating software is much easier and more fun.

The beta version has brand new visual elements as well. The Tulliana 2 icon set developed in house, solely for Pardus.

Pardus “ATA”, comes as a single CD which Dutch, English, German and Turkish support on the desktop, internet tools, office software and multimedia players (photo, music, video etc.), games and countless of other productivity software. Italian and French support will follow shortly. A server version of Pardus is under development and testing, and will released in Q1 2007.

“ATA” may be downloaded from Pardus ftp server.
