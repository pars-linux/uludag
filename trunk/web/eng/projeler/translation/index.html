Translating Pardus

Pardus can support your language, and it's easy, yet time consuming to let Pardus speak any language. These are the items you should consider before giving your language support to Pardus. Most of the user interface and help files are in po format, so an editor which can read and write po files (i.e Kbabel or Poedit) will be sufficient.

After installing Pardus, downloading kdesdk package will install Kbabel and place an entry in Pardus menu.

Supported languages

Pardus 1.0 comes with 2 languages (English and Turkish). The following language supports are available for Pardus 1.1, starting from Pardus 1.1 alpha3.

  • English (Maintainer: Görkem Çetin)
  • Turkish (Maintainer: Görkem Çetin)
  • Dutch (Maintainer: Willem Gielen)
