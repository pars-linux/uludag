YALI

YALI (Yet Another Linux Installer) is the first Pardus software a user encounters. Basically, it recognizes the hardware and installs Pardus software from the installation media (i.e CD) to a hard disk partition user selects. It has a high performance compared to other installers and we believe that YALI is the fastest graphical Linux installer on earth, thanks to PiSi package management system.
