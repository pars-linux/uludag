ÇOMAR

ÇOMAR (pronounced chow-mar), is the COnfiguration MAnageR that helps the installed software operate in a consistent manner. ÇOMAR knows the tasks that can be provided by each application, together with the functionality they depend on and other information. Different applications may adapt themselves according to the presence and capability of their peers.

ÇOMAR also includes other projects like Mudur, a fast and effective init system for Pardus which is written in Python. Mudur is currently in a working state starting from Pardus 1.1 alpha. Interested and experienced developers are more than welcome to build new enhancements on top of this model.

Click here to read what mudur is, and new infrastructure it brings to the init model of Linux.
