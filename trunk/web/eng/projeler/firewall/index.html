Pardus Firewall

Pardus Firewall is a free firewall application developed for Pardus Linux. It uses ÇOMAR as a backend to serve all the configuration needs. Pardus firewall aims to provide an easy to use, yet powerful firewall software for Pardus, serving desktop users.

Main features of Pardus firewall are the following:

  • Easy to use graphical interface with tabbed browsing
  • Tightly integrated with COMAR, the configuration manager of Pardus
  • Ability to show firewall log
  • Ability to define the inbound access policy
