Pardus 1.0

Pardus 1.0 (tam sürüm) (26/12/2005)

Yukarıdaki bağlantıya tıklayarak Pardus 1.0 sürümünü indirebilirsiniz. Yardım için, kurulum belgesi'ne başvurabilir, ayrıca 1.0 özellikleri hakkında bilgi almak için sürüm notları belgesini okuyabilirsiniz.

Pardus 1.0'ı çeşitli yansı adreslerinden de indirebilirsiniz. Aşağıda, mevcut yansı adreslerinin bir listesi bulunmaktadır:

  • Çanakkale 18 Mart Üniversitesi
  • Hacettepe Üniversitesi
  • Linux Kullanıcıları Derneği
  • Eskişehir Yazılım Üssü

  • Pardus Çalışan CD 2007 beta (10.11.2006)

  • Pardus Çalışan CD 1.0 (tam sürüm) (04/03/2005)
