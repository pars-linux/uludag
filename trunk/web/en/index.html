Uludag project aims at developing an operating system distribution based on the Linux kernel and the accompanying open source/free software applications. The distribution targets the desktop user in the first stage.

Goals of the Uludag project

  • To develop a widely used operating system distribution,
  • To establish a sustainable organization to develop this distribution,
  • To contribute to the open source/free software community through this distribution and the surrounding development ecosystem.

The Uludag operating system

  • Will provide full and sound Turkish support,
  • Will be easy to install and use,
  • Will have a human-centric task-based modular structure.
