Pardus 2009 Beta Release Notes

1. Willkommen zu Pardus 2009 Beta!

Willkommen zu Pardus 2009 Beta, ein weiteres großartiges Parduslinux der Pardusentwickler , Übersetzer, artists und Dokumentare. Das ist die zweite Testveröffentlichung des Pardus2009-zweiges.
Wir laden Sie höflich ein unsere Hompage zu besuchen http://www.pardus.org.tr/eng um mehr über Pardus zu erfahren.

1.1 Feedback

Um Feedback zu geben, nutzen Sie http://bugs.pardus.org.tr, (Pardus bug tracking system) falls die einen Programmierfehler (bug) entdeckt haben, oder senden Sie uns einen Bug-fix oder teilen Sie uns andere Verbesserungen u.ä. was für Pardus relevant sein kann mit. Wir danken unseren Nutzern und Entwicklern für deren Bug-berichte, Bug-fixes, neuen Paketen oder den Verbesserungsvorschlägen die in Pardus2009 einfließen konnten.

2. Was ist neu ?

2.1. Updates in den Pardus Technologien

Mit der Veröffentlichung von Pardus 2009 Beta, wurden grundsätzliche Technologien wie PiSi und COMAR verbessert. YALI (YetAnotherLinuxInstaller) wurde ebenfalls verbessert um eine flüssigere und möglichst angenehme Installation zu ermöglichen. Die Überarbeitung von Mudur dem Pardus Initialisationssystem hat einen schnelleren Bootvorgang, sowie einen robusteren Support für  Netzfreigaben zur Folge.

Ein neuer SERVICE System Manager wurde in Pardus 2009 implementiert. Der neue System Manager ermöglicht eine spielerisch leichte Konfiguration sämtlicher Systemeinstellungen. Alle Pardus eigenen Konfigurationshilfen wurden für KDE4 neu geschrieben und optimiert. Des weiteren sind auch die neuen KDE-Plasmoids der Pardus-Services über die Pardus-Repositories installierbar.

2.2. KDE 4 Desktop Enviroment

New member of the KDE family KDE 4 that has been the choice of many Pardus users for a while now, is now the default desktop environment. Pardus 2009 Beta includes the latest stable release of KDE, 4.2.4 that has been enhanced and modified for Pardus. KDE programs in Pardus repositories have been updated to thier latest KDE 4 versions.

2.3. Updated Components

Pardus 2009 Beta includes a lot of package updates besides Pardus Technologies. Some of the highlights are as follows:
  • KDE 4.2.4
  • Linux kernel 2.6.30.1
  • Openoffice office suite 3.1
  • Firefox Web browser 3.5
  • Gimp 2.6.6
  • Xorg 1.6.2pre
  • Python 2.6.2
  • GCC 4.3.3
  • GLIBC 2.9
Für weitere Informationen über Updates und neuen features bei Pardus 2009, besuchen Sie die Pardus website und das lokale User-Forum.

3. Installations- und Hardwareanforderungen

Pardus 2009 Beta läuft auf allen gängigen PC-systemen. Wir empfehlen mindestens 512 MB RAM (Speicher) und einen 1400 MHz Prozessor um eine gute Performance zu erzielen. Minimal empfohlene Hardwareanforderungen für Pardus 2009 Beta sind:
  • 256 MB RAM-Speicher (1GB oder mehr empfohlen)
  • 1000 MHz Intel oder AMD processor (1400 MHz empfohlen)
  • Mindestens 4 GB freien Festplattenspeicher (10 GB empfohlen)
