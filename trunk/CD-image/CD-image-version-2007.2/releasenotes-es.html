Notas de Release Pardus 2007.1

1. Bienvenido a Pardus 2007.1

Le damos la bienvenida a Pardus versión 2007.1, un lanzamiento de mantenimiento para Pardus 2007, siendo un sistema operativo fácil de instalar y usar, que provee una serie de desarrollos útiles, seguros y compatibles con el entorno de escritorio K (KDE). Con Pardus, puede conectarse a Internet, leer mensajes de e-mail, comunicarse de modo seguro con sus amigos y aprovechar las facilidades multimedia de su computadora, escuchando musica, y observando peliculas. Le invitamos cordialmente a visitar www.pardus.org.tr/eng para descubrir como 2007.1 puede responder a sus necesidades en computación.

1.1 Feedback

Puede utilizar la página web bugs.pardus.org.tr, alias sistema de reporte de fallas de Pardus, para enviar un reporte de falla, una corrección, o cualquier sugerencia relacionado con Pardus. Nos gustaría agradecer a nuestros usuarios y desarrolladores por su envío de reportes de fallas, correcciones relacionados y sugerencias para funcionalidades, y que de esta manera aportaron para la release actual.

Además puede utilizar el aplicativo de feedback, que viene con Pardus 2007.1 para enviar sus ideas sobre Pardus por la Internet.

2. Características Principales

Esta sección describe las características que está introduciendo Pardus 2007.1. Pardus 2007.1 es principalemente una release de corrección de errores (bugfix), actulización de traducciones y extensiones menores de Pardus 2007.

2.1 Usabilidad del Escritorio (KDE 3.5.6)

Pardus 2007.1 ahora introduce KDE 3.5.6 por su mejor estabilidad, traducción y efectos visuales para los usuarios de Pardus. Este paquete tambien es disponible para usuarios de Pardus 2007 mediante el administrador de paquetes de Pardus.

2.2 Soporte WPA para Administrador de Redes

El Administrador de Red ahora viene con soporte WPA incorporado, mostrando el estado de redes Wifi con una linda interfáz gráfica. Esto fue una de las características más requeridas luego de Pardus 2007.

2.3 Sistema de Administración de Paquetes mejorado

PISI, el administrador de paquetes de Pardus llegó a la version 1.1.1, introduciendo busquedas más eficientes y una mejora de 40% + en velocidad para la fase de actualización de repositorios. Ahora usuarios estarán beneficiados de la mejora en velocicad en actualizaciones de repositorios y busquedas insensitivas a mayusculas/minúsculas para nombres de paquetes, y información de resumen y descripción.

2.4 Actualizaciones de Paquetes

Muchas actualizaciones de seguridad de Pardus 2007 fueron incluidos en 2007.1, incluyendo mplayer, spamasassin, clamav, ImageMagick, nexuiz, wireshark, kernel y otros. Adicionalmente, más de 280 paquetes fueron actualizados y están incluidos en 2007.1

2.5 Más soporte de localización

Pardus 2007 incluye soporte para Inglés, Turco, Alemán, Holandés y Español desde la instalación hasta el escritorio. Ahora en Pardus 2007.1 hay soporte adicional para Portugués Brasiliano desde el inicio.

Soporte para Italiano y Francés está en preparación, y le invitamos a apoyar a Pardus con su idioma para incluirlos también en las proximas versiones de Pardus.

3. Requerimientos de Installación

Pardus puede ser ejecutado en un hardware promedio. Sugerimos al menos 512 MB RAM y procesador de 1200 MHz para un rendimiento aceptable. Requerimientos de hardware mínimos y recomendados para Pardus 2007.1 son:
  • 256 MB memoria (512 MB recomendado)
  • Procesador 800 MHz, Intel o AMD (1200 MHz recomendado)
  • Espacio en disco rígido al menos 4 GB (10 GB recomendado)

4. Contacto

Por favor, visite la página web de Pardus (www.pardus.org.tr/eng) por más información.
