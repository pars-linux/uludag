Pardus 2007.3 release notes

1. Welcome to Pardus 2007.3

Welcome to Pardus version 2007.3, another great Pardus release from Pardus developers, translators, artists and documentation writers. This release features a set of new tools together with updated software after the release of Pardus 2007.2.

We kindly invite you to visit www.pardus.org.tr/eng for more information.

1.1 Feedback

You can use bugs.pardus.org.tr, a.k.a Pardus bug tracking system to send a bug, bug fix or any suggestion related to Pardus. We would like to thank our users and developers with their bug reports, related fixes, new package or feature suggestions for this release.

Also, you may use the feedback application available via menu to send your ideas about Pardus over Internet.

2. Basic features

This section describes the Pardus 2007.3 features.

2.1 Updated network manager

Network manager now remembers the last connection and tries to connect to known connections. It also gives information via a popup from taskbar which makes network manager more intuitive and user friendly.

2.2 Improved package management system

Package manager is more polished now. Many bugs in bug tracker have been fixed.

2.3 YALI

YALI (yet another Linux installer) now asks questions first, and then installs the system. This provides a more flexible and time saving procedure. User also can decide whether to login to desktop automatically or not. Last but not least, it includes definitions for state-of-the-art hard disks, in order to ease installation process.

Moreover, YALI can understand whether a disk is bootable or not, and decide which disk to install Pardus on. This feature is first introduced in Pardus distribution.

2.4 Updated applications

KDE, which is used as primary desktop environment for Pardus 2007.3 is updated to 3.5.8. All-in-one office suite OpenOffice.org is now version 2.3. K3b, CD/DVD burner application version is upgraded to 1.0.4.

3. Installation requirements

You can run Pardus 2007.3 on an average hardware. We suggest to use at least 512 MB RAM (memory) and 1200 MHz processor to get an effective performance. Minimum and recommended hardware requirements for Pardus 2007.3 are mentioned below:
  • 256 MB memory (512 MB recommended)
  • 800 MHz Intel or AMD processor (1200 MHz recommended)
  • At least 4 GB of free hard disk space (10 GB recommended)
