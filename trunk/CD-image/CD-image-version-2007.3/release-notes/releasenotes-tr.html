Pardus 2007.3 sürüm notları

1. Pardus 2007.3'e hoşgeldiniz

Pardus geliştiricileri, çevirmenleri, belge yazarları ve sanatçılarının desteği ile hazırlanan Pardus 2007.3'e hoşgeldiniz. 2007 serisinin bu son sürümü pek çok güncelleme ve hata gidermeleri içermektedir. Pardus 2007.3 hakkında daha fazla bilgiye www.pardus.org.tr sayfasından ulaşabilirsiniz.

1.1 İstek ve önerileriniz

Pardus 2007.3'e ilişkin hata bildirimlerinizi, istek ve önerilerinizi hata.pardus.org.tr adresinde bulunan hata bildirim sistemine iletebilirsiniz. Göndereceğiniz mesajlar Pardus geliştiricileri tarafından incelenecek ve Pardus'un gelecek sürümlerinde yapılacak iyileştirmeler için kullanılacaktır. Şimdiden destek olan tüm Pardus severlere teşekkür ederiz.

Hata bildirim sistemi yanında, Pardus menüsünden ulaşılabilen "Geribildirim aracı" yardımıyla bir kaç adımda isteklerinizi iletmeniz mümkündür. Bu işlem için bilgisayarınızın internete bağlı olması gereklidir.

2. Temel özellikler

Bu bölüm Pardus 2007.2'den sonra yapılan ve Pardus 2007.3'e aktarılan değişiklikleri içermektedir.

2.1 Update network manager

Network manager now remembers the last connection and tries to connect to known connections. It also gives information via a popup from taskbar which makes network manager more intuitive and user friendly.

2.2 Improved package management system

Package manager is more polished now. Many bugs in bug tracker have been fixed.

2.3 YALI

YALI (yet another Linux installer) now asks questions first, and then installs the system. This provides a more flexible and time saving procedure. User also can decide whether to login to desktop automatically or not. Last but not least, it includes definitions for state-of-the-art hard disks, in order to ease installation process.

Moreover, YALI can understand whether a disk is bootable or not, and decide which disk to install Pardus on. This feature is first introduced in Pardus distribution.

2.4 Updated applications

KDE, which is used as primary desktop environment for Pardus 2007.3 is updated to 3.5.8. All-in-one office suite OpenOffice.org is now version 2.3. K3b, CD/DVD burner application version is upgraded to 1.0.4.

3. Kurulum gereksinimleri

Pardus 2007.3'ü ortalama bir donanımda çalıştırabilirsiniz. Etkili bir performans için en az 512 MB RAM (bellek) ve 1200 MHz işlemci öneriyoruz. Pardus 2007.3 için asgari ve tavsiye edilen donanım gereksinimleri aşağıdadır:
  • 256 MB bellek (512 MB tavsiye edilir)
  • 800 MHz Intel ya da AMD işlemci (1200 MHz tavsiye edilir)
  • En az 4 GB boş disk alanı (10 GB tavsiye edilir)
