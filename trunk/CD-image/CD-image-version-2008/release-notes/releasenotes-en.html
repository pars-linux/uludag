Pardus 2008 release notes

1. Welcome to Pardus 2008

Welcome to Pardus version 2008, another great Pardus release from Pardus developers, translators, artists and documentation writers. This is the first release of Pardus 2008 branch.

We kindly invite you to visit http://www.pardus.org.tr/eng for more information.

1.1 Feedback

You can use http://bugs.pardus.org.tr , a.k.a Pardus bug tracking system to send a bug, bug fix or any suggestion related to Pardus. We would like to thank our users and developers with their bug reports, related fixes, new package or feature suggestions for this release.

Also, you may use the feedback application available via menu to send your ideas about Pardus over Internet.

2. Basic features

This section describes the Pardus 2008 features.

2.1 Updated network manager

Network manager supports auto-detection of your wireless profiles and switch to one of them since 2007.3. It also gives information via a popup from taskbar which makes network manager more intuitive and user friendly. With Pardus 2008 you may use an optimized and bugfixed network-manager.

2.2 Improved package management system

Package manager is more polished now. Many bugs in bug tracker have been fixed. Delta repo will be available for 2008 release, so updates will be quicker and faster.

2.3 YALI

YALI (yet another Linux installer) is refactored, using qt4 with a polished artwork. You can now adjust your time-zone, auto-login preferences and necessary information before installing which saves time...

We want to remind that, YALI can understand whether a disk is bootable or not, and decide which disk to install Pardus on. This feature is first introduced by Pardus.

2.4 Updated applications

KDE, which is used as primary desktop environment for Pardus 2008 is updated to 3.5.9. All-in-one office suite OpenOffice.org is now version 2.4.

3. Installation requirements

You can run Pardus 2008 on an average hardware. We suggest to use at least 512 MB RAM (memory) and 1200 MHz processor to get an effective performance. Minimum and recommended hardware requirements for Pardus 2008 are mentioned below:
  • 256 MB memory (512 MB recommended)
  • 800 MHz Intel or AMD processor (1200 MHz recommended)
  • At least 4 GB of free hard disk space (10 GB recommended)
