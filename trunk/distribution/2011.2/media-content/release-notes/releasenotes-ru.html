Примечания к выпуску

1. Welcome to Pardus 2011.2 Cervus elaphus

Welcome to Pardus 2011.2 Cervus elaphus, the latest maintenance release of Pardus, a powerful, KDE-focused Linux distribution.

1.1. Что такое Pardus?

Pardus - это полностью открытый, свободно распространяемый, GPL (General Public License) лицензированный, простой в использовании GNU/Linux дистрибутив с множеством усовершенствований в плане удобства использования, совместимости с аппаратным обеспечением, стабильности и безопасности.
Проект Pardus спонсируется Центром исследования новых технологий информатики и информационной безопасности (BILGEM), филиалом Научно-технологического исследовательского совета Турции (TUBITAK).
Цель дистрибутива Pardus в том, чтобы предоставить людям стабильную, надежную, легкую в использовании и многофункциональную систему для работы с компьютером, представленную современным и популярным открытым программным обеспечением. В дополнение ко всему, разработчики прикладывают большие усилия, чтобы Pardus поддерживал широкий спектр оборудования.
Проект назван в честь латинского (и научного) имени Анатолийского леопарда, Panthera pardus tulliana.

1.2. Как я могу попробовать Pardus?

Вы можете попробовать Pardus либо путем установки его на свой компьютер, любо запустив Live-версию, не внося изменения на свой жесткий диск. Вы можете загрузить Pardus с оптического привода, а также с USB-устройств хранения данных, таких как внешние жесткие диски и USB флэш-диски.
В любом случае, установочный и live-образ дистрибутива превышают размер обычного CD-диска, поэтому если вы хотите использовать оптический привод для загрузки Pardus, вы должны использовать для записи образа более емкий DVD-диск.
Обратите внимание, что порядок загрузки в вашей системе (BIOS) должен быть установлен с оптическим приводом или USB-устройством, как первым устройством для загрузки. В новых системах, порядок загрузки можно также изменить, нажав специальную клавишу (как правило, это F8 на настольных системах) во время загрузки.

2. Обзор

Pardus 2011.2 содержит важные улучшения со времен предыдущего выпуска Pardus 2011. В следующих разделах приведен краткий обзор этих изменений.
  • Kernel - 2.6.37.6 Linux kernel provides updated hardware support together with a hundred of bug fixes.
  • YALI - YALI, the installer of Pardus, gained Rescue Mode which provides easy password and bootloader recovery.
  • KDE SC - Pardus 2011.2 поставляется с последней версией KDE Software Compilation, KDE SC 4.6.5. Базовые пакеты так же содержат многочисленные исправления и улучшения, что значительно увеличивает стабильность вашей системы.
  • LibreOffice - LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is updated to the much-awaited 3.4 version which brings huge improvements over the older version.
  • Firefox - Firefox 5.0, the latest stable version of the most popular Open Source web browser, is now shipped with Pardus 2011.2.

3. Что нового?

3.1. Обновления в технологиях Pardus

3.1.1. YALI

  • YALI, the installer of Pardus, gained system rescue menu that provides easy bootloader and password recovery.

3.1.2. Pisi

  • Some of the core components are now parallellized to increase the utilization of multi-core machines which means faster package operations and better package management experience.

3.1.3. QuickFormat

  • QuickFormat is a removable device formatting tool that can integrate to Dolphin. Now, it's very easy to format USB flash drives and card readers.

3.2. Обновления в основных компанентах системы

  • Kernel - The Linux kernel shipped with Pardus 2011.2 is 2.6.37.6.
  • Plymouth - The bootsplash technology used in Pardus 2011 is updated to its latest version for a much stable experience.
  • X.org - xorg-server is updated to 1.9.5.
  • TeX Live - TeX Live documentation stack is updated to latest 2011 version.

3.3. Окружение рабочего стола

3.3.1. Изменения в KDE Desktop Environment

KDE SC - Pardus 2011.2 поставляется с последней версией KDE Software Compilation, KDE SC 4.6.5. Базовые пакеты так же содержат многочисленные исправления и улучшения, что значительно увеличивает стабильность вашей системы.
KDE Software Compilation, versioned 4.6.5, provide many new features in each of KDE's components including Plasma, Solid, Nepomuk, KWin and also introduces a brand new Bluetooth framework called BlueDevil allows easy device pairing and management.

3.3.2. Стандартные приложения

3.3.2.1. Mozilla Firefox
Pardus 2011.2 comes with Mozilla Firefox 5.0 as the default web browser application. This version of Firefox introduces CSS animations, Do-Not-Track, improved support for HTML5, MathML, and SMIL, improved network, memory and JavaScript performance and fixes for several stability and security issues.
3.3.2.2. Clementine
Clementine, a cross-platform music player based on Amarok 1.4 is updated to latest stability and maintenance version 0.7.1.
3.3.2.3. SMPlayer
SMplayer, Qt-фронтенд для MPlayer, установлен по-умолчанию как Video/VCD/DVD плеер, отлично интегрируется в окружение рабочего стола KDE, имеет удобный графический интерфейс и поддерживает воспроизведение широкого спектра типов файлов, благодаря MPlayer.
3.3.2.4. KsCD
KsCD, проигрыватель Audio CD, используемый по-умолчанию.

3.3.3. Веб-браузеры

Помимо содержимого установочного образа, имеется большой спектр поддерживаемых браузеров, включая Chromium, rekonq, Arora и Opera. Они доступны для установки через менеджер пакетов.

3.3.4. Настройки системы

The display configuration tool of the graphics card manufacturer is now recommended in the Display and Monitor module of the System Settings, if it suits better to your graphics card driver.

3.3.5. Набор офисных приложений

LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is the default Office Suite in Pardus 2011.2. 3.4 version of LibreOffice introduces; improved text rendering, configurable color charts, improved thumbnailed HTML export of presentations, faster Flat ODF import/export and better desktop theme integration.

3.3.6. Other

Most wanted packages like Skype, a P2P VoIP client that allows you to make voice calls, and Wine, a software that allows you to run Microsoft Windows applications, are now included in Pardus 2011.2 package repopsitories.

4.2. Оперативная память (RAM)

Требуемый объем оперативной памяти (RAM) для Pardus 2011.2 сильно зависит от вашей ежедневной нагруженности системы, поэтому очень трудно определить нижнюю и верхнюю границу значений, но мы рекомендуем по крайней мере 1GB оперативной памяти для хорошей работы системы и рабочего стола.Требуемый объем оперативной памяти (RAM) для Pardus 2011.2 сильно зависит от вашей ежедневной нагруженности системы, поэтому очень трудно определить нижнюю и верхнюю границу значений, но мы рекомендуем по крайней мере 1GB оперативной памяти для хорошей работы системы и рабочего стола.

4.3. Требования к дисковому пространству

4.3.1. Минимальные системные требования

Pardus 2011.2 требует, по крайней мере 4GB свободного дискового пространства для успешной установки. Так как мы не предлагаем выбор устанавливаемых пакетов в процессе инсталляции, то это ограничение актуально для всех установок.

4.3.2. Рекомендуемые системные требования

С целью эффективного использования Pardus 2011.2 на вашей системе без нехватки свободного дискового пространства, во время вашей ежедневной работы за компьютером, мы рекомендуем как минимум 10GB свободного места на жестком диске.

4.3.3. О разделе подкачки (swap)

Раздел подкачки используется всякий раз, когда вашей операционной системе не хватает физической памяти. Если вы считаете, что размер физической памяти на вашем компьютере достаточно велик для ежедневной работы, то можете пропустить шаг создания раздела подкачки во время установки.
Обратите внимание, что раздел диска, отведенный под раздел подкачки необходим,если вы хотите использовать спящий режим в Pardus. В противном случае, данная функция будет отключена.
