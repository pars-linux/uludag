Bilješke o izadanju

1. Welcome to Pardus 2011.2 Cervus elaphus

Welcome to Pardus 2011.2 Cervus elaphus, the latest maintenance release of Pardus, a powerful, KDE-focused Linux distribution.

1.1. Što je Pardus?

Pardus is an open-source, freely distributable, GPL (General Public License) licensed user-friendly GNU/Linux distribution with many usability, hardware compatibility, stability and security enhancements.
Pardus projekt sponzorira Center of Research for Advanced Technologies of Informatics and Information Security (BILGEM), kojeg je ustanovio Scientific and Technological Research Council of Turkey (TUBITAK).
Cilj Pardus distribucije je ponuditi ljudima stabilano,pouzdano,lagano za korištenje i osobinama bogato računalno iskustvo donoseći moderan i popularan software otvorenog koda.Razvijatelji su uložili dodatni napor da bi se Pardus pokretaqo na širokom spektru hardware-a.
Projakt je nazvan po latinskom (i znanstvenom) nazivu za Anatolijskog leoparda, Panthera pardus tulliana.

1.2. Kako mogu isprobati Pardus?

You can try Pardus either by installing it on your computer or running the Live version without touching your hard disk. You can boot Pardus from an optical drive as well as from USB mass storage devices like external hard drives and USB flash drives.
Instalacijaka i Live verzija premašuju veličinu običnog CD-a,pa ako želite koristiti optički pogon zas podizanje Pardus,trenutno morate koristiti DVDF medij da biste snimili slike.
Note that the boot order in your system's BIOS software should be set with the optical drive or USB mass storage device as the first device in order to boot from the installation or the live medium. On newer systems, the boot order can also be changed by pressing a key (generally F8 on desktop systems) while booting.

2. Pregled

Pardus 2011.2 sadrži značajna poboljšanja u odnosu na prethodna Pardus izadanja 2011. slijedeće sekcije donose kratki pregled tih promjena.
  • Kernel - 2.6.37.6 Linux kernel provides updated hardware support together with a hundred of bug fixes.
  • YALI - YALI, the installer of Pardus, gained Rescue Mode which provides easy password and bootloader recovery.
  • KDE SC - Pardus 2011.2 comes with the latest KDE Software Compilation, KDE SC 4.6.5. The base packages also contains numerous backports and fixes which will improve the stability of your desktop experience significantly.
  • LibreOffice - LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is updated to the much-awaited 3.4 version which brings huge improvements over the older version.
  • Firefox - Firefox 5.0, the latest stable version of the most popular Open Source web browser, is now shipped with Pardus 2011.2.

3. Što je novo?

3.1. Dopune u Pardus tehnologiji

3.1.1. YALI

  • YALI, the installer of Pardus, gained system rescue menu that provides easy bootloader and password recovery.

3.1.2. Pisi

  • Some of the core components are now parallellized to increase the utilization of multi-core machines which means faster package operations and better package management experience.

3.1.3. QuickFormat

  • QuickFormat is a removable device formatting tool that can integrate to Dolphin. Now, it's very easy to format USB flash drives and card readers.

3.2. Dopune u komponentama jezgre

  • Kernel - The Linux kernel shipped with Pardus 2011.2 is 2.6.37.6.
  • Plymouth - The bootsplash technology used in Pardus 2011 is updated to its latest version for a much stable experience.
  • X.org - xorg-server is updated to 1.9.5.
  • TeX Live - TeX Live documentation stack is updated to latest 2011 version.

3.3. Iskustvo radne površine

3.3.1. Promjene u KDE radnom okruženju

Pardus 2011.2 comes with the latest KDE Software Compilation, KDE SC 4.6.5. The base packages also contains numerous backports and fixes which will improve the stability of your desktop experience significantly.
KDE Software Compilation, versioned 4.6.5, provide many new features in each of KDE's components including Plasma, Solid, Nepomuk, KWin and also introduces a brand new Bluetooth framework called BlueDevil allows easy device pairing and management.

3.3.2. Zadani programi

3.3.2.1. Mozilla Firefox
Pardus 2011.2 comes with Mozilla Firefox 5.0 as the default web browser application. This version of Firefox introduces CSS animations, Do-Not-Track, improved support for HTML5, MathML, and SMIL, improved network, memory and JavaScript performance and fixes for several stability and security issues.
3.3.2.2. Clementine
Clementine, a cross-platform music player based on Amarok 1.4 is updated to latest stability and maintenance version 0.7.1.
3.3.2.3. SMPlayer
SMplayer, a Qt based frontend for MPlayer is set as the default Video/VCD/DVD playing application as it is well integrated with KDE desktop environment in terms of usability and graphical user interface and supports playing a wide range of video file types thanks to the MPlayer backend.
3.3.2.4. KsCD
KsCD, is set as the default Audio CD playing application.

3.3.3. Web preglednici

Iako nisu uključeni u insatalacijske slike,širok raspon web preglednika uključujući Chromium preglednik,Arora i Opera dostupni su za instalaciju kroz upravitelja paketa.

3.3.4. Postavke sustava

The display configuration tool of the graphics card manufacturer is now recommended in the Display and Monitor module of the System Settings, if it suits better to your graphics card driver.

3.3.5. Office Suite

LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is the default Office Suite in Pardus 2011.2. 3.4 version of LibreOffice introduces; improved text rendering, configurable color charts, improved thumbnailed HTML export of presentations, faster Flat ODF import/export and better desktop theme integration.

3.3.6. Other

Most wanted packages like Skype, a P2P VoIP client that allows you to make voice calls, and Wine, a software that allows you to run Microsoft Windows applications, are now included in Pardus 2011.2 package repopsitories.

4.2. Memorija sustava (RAM)

Zahtjevi memorije sustava (RAM) za Pardus 2011.2 ovise o vašem dnevnom radu pa je teško odrediti gornju i donju granicu,ali mi preporučujemo najmanje 1GB memorije za ugodno iskustvo radne površine.

4.3. Zahtjevi prostora tvrdoga diska

4.3.1. Minimalni zahtjevi

Pardus 2011.2 zahtjeva najmanje 4GB slobodnog prostora tvrdog diska za uspješnu instalaciju.Zato jer ne nudimo prolagođeni odabir paketa tijekom instalacije,ova minimalna veličina sa primjenjuje na sve instalacije.

4.3.2. Preporučeni zahtjevi

Da biste učinkovito koristili Pardus 2011.2 da vam ne ponestane prostora na disku tijekom vašeg dnevnog rada,preporučujemo najmanje 10GB slobodnog prostora tvrdog diska.

4.3.3. Bilješka o swap prostoru

swap prostor se koristi uvijek kada vašem operativnom sustavu ponestane fizičke memorije.Ako mislite da je vaša fizička memorija dovoljno velikaza vašu dnevnu uporabu možete preskočiti stvaranje swap prostora tijekom instalacije.
Napominjemo particija diska rezervirana za swap prostor je potrebna ako želite hibernirati (Suspend to disk) Pardus.U suprotnom ta će finkcija biti onemogućena.
