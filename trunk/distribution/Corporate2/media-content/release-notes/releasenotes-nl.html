1. Welcome to Pardus Kurumsal 2

Welcome to Pardus Kurumsal 2, the first release of Pardus Corporate series offering a mature enterprise desktop experience to users.

1.1. Wat is Pardus?

Pardus is een open-source, vrij beschikbare, onder GPL-licentie (General Public License) vallende, gebruikersvriendelijke GNU/Linux-distributie met veel gebruikersgemak, hardware-compatibiliteit, stabiliteit en veilige mogelijkheden.
Het Pardus-project wordt ondersteund door de Center of Research for Advanced Technologies of Informatics and Information Security (BILGEM), een afdeling van de Scientific and Technological Research Council of Turkey (TUBITAK).
Het doel van de Pardus-distributie is het aanbieden van een stabiele, betrouwbare, gemakkelijk-in-gebruik en vele-mogelijkheden computerervaring door gebruik te maken van moderne en populaire open-source software. De ontwikkelaars hebben tevens additionele inspanningen gedaan om Pardus op zoveel mogelijk hardware mogelijk te maken.
Het project is genoemd naar de Latijnse (en wetenschappelijke) naam van de Anatolische luipaard, Panthera pardus tulliana.

1.2. Hoe kan ik Pardus proberen?

Het is mogelijk om Pardus te proberen door het op de computer te installeren of door het starten van de live-versie en daardoor de inhoud van de harde schijf niet te veranderen. Pardus kan opgestart worden vanaf een optisch medium en ook vanaf een USB-medium zoals externe harde schijf en USB-geheugen/stick.
Zowel de installatie- en ook de live-uitgave passen niet op een normale CD en dus is het gebruik van een DVD noodzakelijk om Pardus via een optisch medium te gebruiken.
Let op dat de volgorde van opstarten in de BIOS van de computer ingesteld dient te worden met het optisch - of USB-station als eerste opstartmedium. Bij de meeste nieuwere computers kan deze volgorde ook geselecteerd worden door het indrukken van een toets (meestal F8 of F11) tijdens het opstarten.

2. Overzicht

The following sections provides a brief overview of the features offered by Pardus Kurumsal 2.

2.1. Hardware-overzicht

Pardus Kurumsal 2 supports both 32-bit and 64-bit processor architectures manufactured by Intel and AMD.

2.2. Software-overzicht

  • Kernel - The Linux kernel shipped with Pardus Kurumsal 2 is 2.6.35 which is the latest kernel branch which will be supported for a long term period.
  • Plymouth - The bootsplash technology used in Pardus Kurumsal 2 is the brand new Plymouth engine.
  • YALI - YALI, het installeergereedschap van Pardus, is uitgebreid met ondersteuning voor LVM/RAID en UUID.
  • KDE - Pardus Kurumsal 2 comes with KDE 3.5.10, the old KDE stack which has proven its stability and reliability. The base packages also contains numerous fixes which will improve the stability of your desktop experience significantly.
  • NetworkManager - GNOME NetworkManager 0.8.2 is the default networking backend in Pardus Kurumsal 2. Users are now able to set up their HSPA/CDMA/VPN networks together with the already supported Ethernet and 802.11 WLAN networks.
  • LibreOffice - LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is the default Office Suite in Pardus Kurumsal 2.

3. Wat is er nieuw ?

3.1. Vernieuwingen in de technologie van Pardus

3.1.1. YALI

  • YALI, het installeergereedschap van Pardus, is uitgebreid met ondersteuning voor LVM/RAID en UUID.
  • De gebruikersinterfaceis helemaal opnieuw ontworpen om gemakkelijker te zijn.

3.1.2. Pisi

  • Pisi gebruikt intern nu XZ als het standaard compressie-algoritme.
  • Binaire pakketnamen geven nu de architectuur en de distributievrijgave weer waarvoor het pakket gebouwd is.
  • Verschillende snelheidsverbeteringen zijn in de hoofdcomponenten gedaan voor een verbeterde pakketbeheerervaring.

3.1.3. Pakketbeheer

  • The graphical user interface is redesigned to improve the usability.
  • Various speed enhancements has been done for a better graphical package management experience.

3.2. Verbeteringen in de hoofdcomponenten

  • Kernel - The Linux kernel shipped with Pardus Kurumsal 2 is 2.6.35 which is the latest kernel branch which will be supported for a long term period.
  • Plymouth - The bootsplash technology used in Pardus Kurumsal 2 is the brand new Plymouth engine.
  • Python - Python 2.6.5 is the default Python interpreter in Pardus Kurumsal 2.
  • Perl - Perl 5.10.1 is the default Perl interpreter in Pardus Kurumsal 2.
  • X.org - The shipped xorg-server version is 1.7.7, improvements to the automatic driver configuration mechanism has been done by our developers.

3.3. Updates in Server Components

  • Web servers - Pardus Kurumsal 2 ships both Apache 2.2.17 and NGINX 0.8.54 HTTP daemon alternatives.
  • Databases - MySQL 5.5.8, PostgreSQL 9.0.3, BerkeleyDB 4.6.21 and SqLite 3.7.3 are among the DBMS alternatives offered in Pardus Kurumsal 2.
  • Mail Servers - Postfix 2.6.5 and Dovecot 1.2.16 are the mail server daemon softwares offered within Pardus Kurumsal 2.

3.4. Bureaubladervaring

3.4.1. Visuele verbeteringen

The default icon theme of Pardus Kurumsal 2 is BC Tango which is a Tango derivative specially customized for KDE desktop environment.

3.4.2. Netwerkbeheer

Pardus Kurumsal 2 uses GNOME NetworkManager as the default networking backend.
With the GNOME networking backend, users are able to easily set up their Ethernet (With support for 802.1x authentication), 802.11 Wireless (With support for WEP/WPA/WPA2/WPA-EAP and 802.1x authentication), EDGE/3G/CDMA and VPN networks.

3.4.3. Kantoortoepassingen

LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is the default Office Suite in Pardus Kurumsal 2.

4. Systeemvereisten

4.1. Processor (CPU)

You may want to install the 64-bit version of Pardus Kurumsal 2 if your system is equipped with one of the processors listed in the sections below.

4.1.1. Intel-processoren met ondersteuning voor 64-bit architectuur

  • Nieuwere versies van Pentium 4, Celeron D, Xeon en Pentium Dual-Core
  • Atom D5xx, N4xx en N5xx
  • Alle versies van Pentium D, Pentium Extreme Edition, Core 2, Core i7, Core i5 en Core i3 processoren
  • Hoogstwaarschijnlijk ook nieuwere versies die vrijgegeven worden na het schrijven van dit document.

4.1.2. AMD-processoren met ondersteuning voor 64-bit architectuur

  • AMD Athlon 64, 64 X2, 64 FX
  • AMD Athlon II
  • AMD Opteron
  • AMD Turion 64
  • AMD Turion 64 X2
  • AMD Sempron
  • AMD Phenom
  • AMD Phenom II
  • Hoogstwaarschijnlijk ook nieuwere versies die vrijgegeven worden na het schrijven van dit document.

4.2. Systeemgeheugen (RAM)

System memory (RAM) requirements of Pardus Kurumsal 2 depends heavily on your daily workflow so it is very hard to define upper and lower bounds, but we recommend at least 512MB of memory for a smooth desktop experience.

4.3. Eisen voor harde schijfruimte

4.3.1. Minimum eisen

Pardus Kurumsal 2 requires at least 4GB of free hard disk space for a successful installation. As we don't offer custom package selection during the installation, this minimum size constraint applies to all installations.

4.3.2. Aanbevolen eisen

In order to effectively use Pardus Kurumsal 2 on your system without running low on disk space during your daily workflow, we recommend at least 10GB of free hard disk space.

4.3.3. Opmerking over swapruimte

Swapruimte is nodig wanneer het besturingssysteem over onvoldoende systeemgeheugen (RAM) beschikt. Wanneer het aanwezige systeemgeheugen voldoende is voor het dagelijks gebruik, is het aanmaken van swapruimte niet nodig tijdens de installatie.
Let op: een gereserveerde partitie voor swapruimte is noodzakelijk indien de schijf-slaapstand (Suspend to disk) in Pardus gebruikt wordt. Anders is deze functie uitgeschakeld.
