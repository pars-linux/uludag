Pardus Corporate 2 Alpha 3 Release Notes

1. Welcome to Pardus Corporate 2 Alpha 3

Welcome to Pardus Corporate 2 Alpha 3. This release is expected to be the last alpha release, subsequent releases will be beta releases as we move towards the final release.


We kindly invite you to visit http://www.pardus.org.tr/eng for more information.

1.1. What is Pardus?

Pardus is a Linux distribution developed as a product of the Pardus Project, by Turkish National Research Institute of Electronics and Cryptology (UEKAE), which is under the Scientific and Technological Research Council of Turkey (TÜBİTAK) and by volunteered contributors. It is named after the Latin (and scientific) name for the Anatolian leopard, Panthera pardus tulliana.

Pardus consists of a Linux based operating system, open source applications and related data files necessary for desktop and server environments, aiming to give its users an easy to use and feature rich experience.

2. What is new?

2.1. List of important bugs fixed since Pardus Corporate 2 Alpha 2

  • #14602 - Pardus is no longer correctly shutting down after the latest updates
  • #13981 - Intel Pro/Wireless 4965 AGN chipsets are not working correctly

2.2. Updates in Pardus Technologies

With the release of Pardus Corporate 2 Alpha 3, core technologies like PiSi and COMAR are updated to provide new features. YALI is also updated to offer a smoother and more reliable installation experience. The enhancements in Mudur initialization system provide a faster and more robust startup process.

Installation system YALI now supports LVM and RAID installations.

2.3. Graphical Enhancements

Pardus Corporate 2 Alpha 3 comes with a lot of graphical enhancements. The default icon theme is a Tango flavour polished for KDE called BCTango. We also have a wide variety of graphical enhancements, from wallpapers of nature photographs to 3D rendered images, splash images etc.

2.4. KDE Desktop Environment

Pardus Corporate 2 Alpha 3 uses KDE Desktop Environment 3.5.10 which fully satisfies the needs of enterprise users in terms of performance and stability.

3. Installation requirements

You can run Pardus Corporate 2 Alpha 3 on an average hardware. We suggest to use at least 512 MB RAM (memory) and a 1400 MHz processor to get a good performance. Minimum and recommended hardware requirements for Pardus Corporate 2 Alpha 3 are mentioned below:
  • 256 MB memory (1GB or more recommended)
  • 1000 MHz Intel or AMD processor (1400 MHz recommended)
  • At least 4 GB of free hard disk space (10 GB recommended)
