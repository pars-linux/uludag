Notas de Lanzamiento

1. Welcome to Pardus Kurumsal 2

Welcome to Pardus Kurumsal 2, the first release of Pardus Corporate series offering a mature enterprise desktop experience to users.

1.1. Qué es Pardus ?

Pardus es un proyecto de una distribución de GNU/Linux amigable al usuario, creado con código abierto, a libre distribución, y liecnciado pajo la licencia GPL (General Public License) , con varios mejoras en usabilidad, compatibilidad de hardware, estabilidad y seguridad.
El auspiciante (sponsor) del proyecto Pardus es el Centro de Investigación de Tecnologías Avanzadas de Informática y Seguridad Informática (BILGEM), un asociado del Concilio de Investigación Científica y Tecnológica de Turquía (TUBITAK).
La distribución Pardus se esfuerza para brindar a la gente una experiencia de computación estable, confiable y fácil de usar mediante la provisión de software de código abierto y popular. Además los desarrolladores trabajan arduamente para lograr una compatibilidad con una amplia gama de hardware.
El nombre del proyecto viene del nombre en latín (cientifico) del leopardo de Anatolia, Panthera pardus tulliana.

1.2. Como puedo probar Pardus ?

Para usar Pardus puede instalarlo en su computadora o ejecutando el sistema en la versión live (en vivo) sin modificar su disco duro. Puede bootear Pardus con un lector óptico o de un dispositivo de almacenamiento USB como discos externos o Pendrive.
Tanto la versión liva como la versión de instalación superan el tamaño de una CD común, por lo tanto si prefiere utilizar un medio óptico para bootear Pardus, debe grabar las imagenes en DVD.
Tenga en cuenta que debe asegurarse que en el orden de arranque en el BIOS el lector óptico o el dispositivo USB deben estar en primer posición para arrancar de ello tanto para la instalación como en modo vivo. En algunos sistemas se puede seleccionar el orden de dispositivos de arranque apretando una tecla (generalmente F8 para equipos de escritorio) durante el arranque.

2. Vista General

The following sections provides a brief overview of the features offered by Pardus Kurumsal 2.

2.1. Hardware

Pardus Kurumsal 2 supports both 32-bit and 64-bit processor architectures manufactured by Intel and AMD.

2.2. Software

  • Kernel - The Linux kernel shipped with Pardus Kurumsal 2 is 2.6.35 which is the latest kernel branch which will be supported for a long term period.
  • Plymouth - The bootsplash technology used in Pardus Kurumsal 2 is the brand new Plymouth engine.
  • YALI - El asistente de instalación, YALI, ahora viene con soporte para LVM/RAID y UUID.
  • KDE - Pardus Kurumsal 2 comes with KDE 3.5.10, the old KDE stack which has proven its stability and reliability. The base packages also contains numerous fixes which will improve the stability of your desktop experience significantly.
  • NetworkManager - GNOME NetworkManager 0.8.2 is the default networking backend in Pardus Kurumsal 2. Users are now able to set up their HSPA/CDMA/VPN networks together with the already supported Ethernet and 802.11 WLAN networks.
  • LibreOffice - LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is the default Office Suite in Pardus Kurumsal 2.

3. Que novedades trae ?

3.1. Actualizaciones de tecnologías de Pardus

3.1.1. YALI

  • YALI, el instalador de Pardus, viene con soporte LVM/RAID y UUID.
  • La interfaz del usuario fue totalmente rediseñado para mejorar la usabilidad

3.1.2. Pisi

  • Pisi ahora usa internamente el algoritmo XZ predeterminado para su método de compresión.
  • Los nombres de los paquetes binarios ahora reflejan la arquitectura y la versión de la distribución correspondiente.
  • Varias mejoras se aplicaron en componentes internos, para una mejor experiencia con el administrador de paquetes.

3.1.3. Administrador de Paquetes

  • The graphical user interface is redesigned to improve the usability.
  • Various speed enhancements has been done for a better graphical package management experience.

3.2. Actualizaciones en componentes internos

  • Kernel - The Linux kernel shipped with Pardus Kurumsal 2 is 2.6.35 which is the latest kernel branch which will be supported for a long term period.
  • Plymouth - The bootsplash technology used in Pardus Kurumsal 2 is the brand new Plymouth engine.
  • Python - Python 2.6.5 is the default Python interpreter in Pardus Kurumsal 2.
  • Perl - Perl 5.10.1 is the default Perl interpreter in Pardus Kurumsal 2.
  • X.org - The shipped xorg-server version is 1.7.7, improvements to the automatic driver configuration mechanism has been done by our developers.

3.3. Updates in Server Components

  • Web servers - Pardus Kurumsal 2 ships both Apache 2.2.17 and NGINX 0.8.54 HTTP daemon alternatives.
  • Databases - MySQL 5.5.8, PostgreSQL 9.0.3, BerkeleyDB 4.6.21 and SqLite 3.7.3 are among the DBMS alternatives offered in Pardus Kurumsal 2.
  • Mail Servers - Postfix 2.6.5 and Dovecot 1.2.16 are the mail server daemon softwares offered within Pardus Kurumsal 2.

3.4. Experiencia de Escritorio

3.4.1. Mejoras Visuales

The default icon theme of Pardus Kurumsal 2 is BC Tango which is a Tango derivative specially customized for KDE desktop environment.

3.4.2. Configuración de Red

Pardus Kurumsal 2 uses GNOME NetworkManager as the default networking backend.
With the GNOME networking backend, users are able to easily set up their Ethernet (With support for 802.1x authentication), 802.11 Wireless (With support for WEP/WPA/WPA2/WPA-EAP and 802.1x authentication), EDGE/3G/CDMA and VPN networks.

3.4.3. Suite Offimatica

LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is the default Office Suite in Pardus Kurumsal 2.

4. Requirimientos al Sistema

4.1. Procesador (CPU)

You may want to install the 64-bit version of Pardus Kurumsal 2 if your system is equipped with one of the processors listed in the sections below.

4.1.1. Procesadores INTEL con soporte a la arquitectura 64-bit

  • Versiones más reciente de Pentium 4, Celeron D, Xeon y Pentium Dual-Core
  • Atom D5xx, N4xx and N5xx
  • Todas las versiones de Pentium D, Pentium Extreme Edition, Core 2, Core i7, Core i5 y Core i3
  • Probablemente procesadores con fabricación más reciente que este informativo.

4.1.2. Procesadores AMD con soporte a la arquitectura 64-bit

  • AMD Athlon 64, 64 X2, 64 FX
  • AMD Athlon II
  • AMD Opteron
  • AMD Turion 64
  • AMD Turion 64 X2
  • AMD Sempron
  • AMD Phenom
  • AMD Phenom II
  • Probablemente procesadores con fabricación más reciente que este informativo.

4.2. Memoria del sistema (RAM)

System memory (RAM) requirements of Pardus Kurumsal 2 depends heavily on your daily workflow so it is very hard to define upper and lower bounds, but we recommend at least 512MB of memory for a smooth desktop experience.

4.3. DIsco rígido

4.3.1. Requisitos mínimos

Pardus Kurumsal 2 requires at least 4GB of free hard disk space for a successful installation. As we don't offer custom package selection during the installation, this minimum size constraint applies to all installations.

4.3.2. Requirimientos recomendados

In order to effectively use Pardus Kurumsal 2 on your system without running low on disk space during your daily workflow, we recommend at least 10GB of free hard disk space.

4.3.3. Comentario sobre espacio Swap

El sistema usa el espacio de intercambio (Swap), cuando la memoria RAM del sistema ya no da abasto. Si cree que tiene suficiente memoria RAM para su uso diario, puede omitir la creación del espacio de intercambio (swap) durante la instalación.
Atención: Si desea utilizar la funcionalidad de hibernar (Suspend to disk) en Pardus se necesita la existencia del espacio de intercambio Pardus. Esta función queda deshabilitada si no existe el espacio de intercambio.
