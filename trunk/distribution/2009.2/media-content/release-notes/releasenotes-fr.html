Release Notes

1. Bienvenue à Pardus

Bienvenue à Pardus 2009.2, la nouvelle version de Pardus que vous ont concocté les développeurs de Pardus, ses traducteurs, ses artistes et ses rédacteurs de documentation.

Nous vous invitons à visiter http://www.pardus.org.tr/eng pour plus d'informations.

1.1. Qu'est-ce que Pardus ?

Pardus est une distribution Linux développée comme un produit du projet Pardus, par l'Institut National turc de Recherche en Électronique et Cryptologie (UEKAE), qui est affilié au Conseil de Recherche Scientifique et Technologique de Turquie (TUBITAK). Pardus vient du nom latin (et scientifique) du léopard d'Anatolie, Panthera pardus tulliana.

Pardus consists of a Linux based operating system, opensource applications et related data files necessary for desktop and server environments, aiming to provide users stable, reliable, easy to use and feature rich computer experience.

1.2. Comment puis-je utiliser Pardus ?

Pardus est livré avec deux saveurs, Kurulan (qui installe sur votre ordinateur Pardus) ou Çalışan (Media Live, qui fonctionne sur le seul média, plus lent, mais ne s'installe pas sur votre ordinateur) et peut être démarré à partir d'un CD / DVD / USB, etc. Pour commencer votre expérience de Pardus, vous devez insérer le support média choisi dans votre ordinateur, le redémarrez et configurer le BIOS de l'ordinateur pour démarrer à partir de ce support média de Pardus.

Çalışan (live) image and Kurulan (installation) images contain support for English and Turkish languages, and can fit on one CD. International image is a multilanguage image (including Turkish) contains language packages, some applications and artwork that are not included in Kurulan image. International image is about 1GB in size, and can be written on a DVD or harddisk like media like a USB stick.

To start your Pardus experience, you need to insert the related media in your computer, reboot the system and set up your computer BIOS to boot from the Pardus media.

2. Ce qui est nouveau

2.1. Mises à jour des technologies de Pardus

Core Pardus technologies are updated to new versions, including many bugfixes, optimizations and new features.

Tous les gestionnaires de Pardus ont été refondus et optimisé pour KDE 4.4. De nouveaux outils de Pardus, le plasmoïde réseau pour un changement de profil réseau facile, et le plasmoïde des services pour la gestion facile des services, sont maintenant disponibles dans nos dépôts. Ces plasmoïdes peuvent être placés soit sur le bureau, soit sur l'un des panneaux sur le bureau pour un accès plus facile.

Un nouvel outil de KDE plasma-runner-pisi has been added to provide easy package installation in KDE environment. Under KDE environment, when you start up krunner (default by pressing ALT-F2) and write a command that is not available on the running system but provided by a package in Pardus repositories, an entry to install the necessary package will appear. Simply selecting the entry will automatically install the required package.

Le gestionnaire de paquets and PiSi have been optimized for better performance and improved stability. PiSi interface has been refactored for easier use. Package Manager went through heavy refactoring and is now using Services du bureau de Pardus, which allows the GUI to run on desktop environments other than KDE 4 without the necessity of any KDE library. Among performance enhancements on the GUI, Package Manager now supports update types. Type of the updates are shown next to packages and users can select group of updates (security for example) easily from the Package Manager GUI.

Pardus welcome wizard Kaptan has also been refactored for the new Pardus release. With new graphical changes, Kaptan is more user friendly then ever, especially on small displays like netbooks.

2.2. Améliorations graphiques

Like other Pardus releases, this new release comes with the theme and name of species that is in danger of extinction. With new packages added to repositories, GTK and KDE integration is improved from notifications to user dialogs. Milky icon theme of Pardus is also updated with new icons. Older wallpapers released withing Pardus 2009 family can also be installed from package repositories.

2.3. Environnement de bureau KDE 4

Default desktop environment is updated to KDE 4.4.3, the latest stable release of KDE that has been enhanced and modified for Pardus. All the KDE programs in Pardus repositories have been updated to or replaced by their latest KDE 4 versions. With the new KDE desktop users can select between Default Mode, Old KDE Mode and the new Mode pour netbooks, optimized for small displays.

Pardus team has integrated Kosd kded module, an eye candy status and level display for sound volume and monitor brightness with the new KDE release.

With the new version KDE, Pardus provides a new look and feel for users. Plasmoid based desktop system, where every component can be placed on anywhere on the desktop, Strigi desktop search fully integrated to desktop environment, desktop effects coming as a part of KDE, Dolphin the next generation file manager with many remote protocol support are just a few examples of the new desktop experience for our users.

3. Requis pour l'installation

You can run Pardus on an average x86 compatible hardware. We suggest to use at least 512 MB RAM (memory) and a 1400 MHz processor to get a good performance. Minimum and recommended hardware requirements for Pardus 2009.2 are mentioned below:
  • 256 Mo de mémoire (1 Go ou plus recommandé)
  • Processeur Intel ou AMD 450 MHz (1000 MHz recommandé)
  • Au moins 4 Go d'espace libre sur le disque dur (10 Go recommandé)
