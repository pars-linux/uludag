Release Notes

1. Welcome to Pardus 2011.1 Dama Dama

Welcome to Pardus 2011.1 Dama Dama, the latest maintenance release of Pardus, a powerful, KDE-focused Linux distribution.

1.1. What is Pardus?

Pardus is an open-source, freely distributable, GPL (General Public License) licensed user-friendly GNU/Linux distribution with many usability, hardware compatibility, stability and security enhancements.
The Pardus project is sponsored by the Center of Research for Advanced Technologies of Informatics and Information Security (BILGEM), an affiliate of the Scientific and Technological Research Council of Turkey (TUBITAK).
The aim of the Pardus distribution is to offer people a stable, reliable, easy-to-use and feature-rich computer experience by providing modern and popular open-source software. The developers are also putting an additional effort to make Pardus run on a wide range of hardware.
The project is named after the latin (and scientific) name for the Anatolian leopard, Panthera pardus tulliana.

1.2. How Can I Try Pardus?

You can try Pardus either by installing it on your computer or running the Live version without touching your hard disk. You can boot Pardus from an optical drive as well as from USB mass storage devices like external hard drives and USB flash drives.
Both the installation and live version images exceeds the size of a regular CD so if you are willing to use an optical drive for booting Pardus, you currently have to use DVD media to burn the images.
Note that the boot order in your system's BIOS software should be set with the optical drive or USB mass storage device as the first device in order to boot from the installation or the live medium. On newer systems, the boot order can also be changed by pressing a key (generally F8 on desktop systems) while booting.

2. Overview

Pardus 2011.1 contains major improvements over the previous Pardus release 2011. The following sections provide a brief overview of those changes.
  • Kernel - 2.6.37.6 Linux kernel provides updated hardware support together with a hundreth of bug fixes.
  • YALI - YALI, the installer of Pardus, gained Rescue Mode which provides easy password and bootloader recovery.
  • KDE SC - Pardus 2011.1 comes with the latest KDE Software Compilation, KDE SC 4.6.5. The base packages also contains numerous backports and fixes which will improve the stability of your desktop experience significantly.
  • LibreOffice - LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is updated to the much-awaited 3.4 version which brings huge improvements over the older version.
  • Firefox - Firefox 5.0, the latest stable version of the most popular Open Source web browser, is now shipped with Pardus 2011.1.

3. What is New?

3.1. Updates in Pardus Technologies

3.1.1. YALI

  • YALI, the installer of Pardus, gained system rescue menu that provides easy bootloader and password recovery.

3.1.2. Pisi

  • Some of the core components are now parallellized to increase the utilization of multi-core machines which means faster package operations and better package management experience.

3.1.3. QuickFormat

  • QuickFormat is a removable device formatting tool that can integrate to Dolphin. Now, it's very easy to format USB flash drives and card readers.

3.2. Updates in Core Components

  • Kernel - The Linux kernel shipped with Pardus 2011.1 is 2.6.37.6.
  • Plymouth - The bootsplash technology used in Pardus 2011 is updated to its latest version for a much stable experience.
  • X.org - xorg-server is updated to 1.9.5.
  • TeX Live - TeX Live documentation stack is updated to latest 2011 version.

3.3. Desktop Experience

3.3.1. Changes in KDE Desktop Environment

Pardus 2011.1 comes with the latest KDE Software Compilation, KDE SC 4.6.5. The base packages also contains numerous backports and fixes which will improve the stability of your desktop experience significantly.
KDE Software Compilation, versioned 4.6.5, provide many new features in each of KDE's components including Plasma, Solid, Nepomuk, KWin and also introduces a brand new Bluetooth framework called BlueDevil allows easy device pairing and management.

3.3.2. Default Applications

3.3.2.1. Mozilla Firefox
Pardus 2011.1 comes with Mozilla Firefox 5.0 as the default web browser application. This version of Firefox introduces CSS animations, Do-Not-Track, improved support for HTML5, MathML, and SMIL, improved network, memory and JavaScript performance and fixes for several stability and security issues.
3.3.2.2. Clementine
Clementine, a cross-platform music player based on Amarok 1.4 is updated to latest stability and maintenance version 0.7.1.
3.3.2.3. SMPlayer
SMplayer, a Qt based frontend for MPlayer is set as the default Video/VCD/DVD playing application as it is well integrated with KDE desktop environment in terms of usability and graphical user interface and supports playing a wide range of video file types thanks to the MPlayer backend.
3.3.2.4. KsCD
KsCD, is set as the default Audio CD playing application.

3.3.3. Web Browsers

Although not included in the installation images, a wide range of web browsers including Chromium browser, rekonq, Arora and Opera are available for installation through the package manager.

3.3.4. System Settings

The display configuration tool of the graphics card manufacturer is now recommended in the Display and Monitor module of the System Settings, if it suits better to your graphics card driver.

3.3.5. Office Suite

LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is the default Office Suite in Pardus 2011.1. 3.4 version of LibreOffice introduces; improved text rendering, configurable color charts, improved thumbnailed HTML export of presentations, faster Flat ODF import/export and better desktop theme integration.

3.3.6. Other

Most wanted packages like Skype, a P2P VoIP client that allows you to make voice calls, and Wine, a software that allows you to run Microsoft Windows applications, are now included in Pardus 2011.1 package repopsitories.

4. System Requirements

4.1. Processor (CPU)

You may want to install the 64-bit version of Pardus 2011.1 if your system is equipped with one of the processors listed in the sections below.

4.1.1. Intel processors which support 64-bit architecture

  • Newer versions of Pentium 4, Celeron D, Xeon and Pentium Dual-Core
  • Atom D5xx, N4xx and N5xx
  • All versions of Pentium D, Pentium Extreme Edition, Core 2, Core i7, Core i5 and Core i3 processors
  • Probably newer processors manufactured after the time this document was originally written.

4.1.2. AMD processors which support 64-bit architecture

  • AMD Athlon 64, 64 X2, 64 FX
  • AMD Athlon II
  • AMD Opteron
  • AMD Turion 64
  • AMD Turion 64 X2
  • AMD Sempron
  • AMD Phenom
  • AMD Phenom II
  • Probably newer processors manufactured after the time this document was originally written.

4.2. System memory (RAM)

System memory (RAM) requirements of Pardus 2011.1 depends heavily on your daily workflow so it is very hard to define upper and lower bounds, but we recommend at least 1GB of memory for a smooth desktop experience.

4.3. Hard disk space requirements

4.3.1. Minimum requirements

Pardus 2011.1 requires at least 4GB of free hard disk space for a successful installation. As we don't offer custom package selection during the installation, this minimum size constraint applies to all installations.

4.3.2. Recommended requirements

In order to effectively use Pardus 2011.1 on your system without running low on disk space during your daily workflow, we recommend at least 10GB of free hard disk space.

4.3.3. Note on Swap space

Swap space is used whenever your operating system runs out of physical memory. If you think that your physical memory is big enough for your daily usage, you may skip creating a swap space during installation.
Note that a disk partition reserved for swap space is required if you want to hibernate (Suspend to disk) Pardus. Otherwise the functionality will be disabled.
