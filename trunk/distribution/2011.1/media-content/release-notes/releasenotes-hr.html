Bilješke o izadanju

1. Dobrodošli u Pardus 2011.1

Welcome to Pardus 2011.1, the latest maintenance release of Pardus, a powerful, KDE-focused Linux distribution.

1.1. Što je Pardus?

Pardus is an open-source, freely distributable, GPL (General Public License) licensed user-friendly GNU/Linux distribution with many usability, hardware compatibility, stability and security enhancements.
Pardus projekt sponzorira Center of Research for Advanced Technologies of Informatics and Information Security (BILGEM), kojeg je ustanovio Scientific and Technological Research Council of Turkey (TUBITAK).
Cilj Pardus distribucije je ponuditi ljudima stabilano,pouzdano,lagano za korištenje i osobinama bogato računalno iskustvo donoseći moderan i popularan software otvorenog koda.Razvijatelji su uložili dodatni napor da bi se Pardus pokretaqo na širokom spektru hardware-a.
Projakt je nazvan po latinskom (i znanstvenom) nazivu za Anatolijskog leoparda, Panthera pardus tulliana.

1.2. Kako mogu isprobati Pardus?

You can try Pardus either by installing it on your computer or running the Live version without touching your hard disk. You can boot Pardus from an optical drive as well as from USB mass storage devices like external hard drives and USB flash drives.
Instalacijaka i Live verzija premašuju veličinu običnog CD-a,pa ako želite koristiti optički pogon zas podizanje Pardus,trenutno morate koristiti DVDF medij da biste snimili slike.
Note that the boot order in your system's BIOS software should be set with the optical drive or USB mass storage device as the first device in order to boot from the installation or the live medium. On newer systems, the boot order can also be changed by pressing a key (generally F8 on desktop systems) while booting.

2. Pregled

Pardus 2011.1 sadrži značajna poboljšanja u odnosu na prethodna Pardus izadanja 2011. slijedeće sekcije donose kratki pregled tih promjena.

2.1. Pregled hardware-a

Pardus 2011.1 sada podržava 64-bit procesorsku arhitekturu koju podržavaju mnogi novi Intel i AMD procesori.

2.2. Pregled software-a

  • Kernel posljedni Linux kernel 2.6.37.6 donosi dopunjenu podršku za hardware zajedno sa tisućama riješenih problema.
  • Plymouth - The bootsplash technology used in Pardus 2011 is updated to its latest version for a much stable experience.
  • YALI - YALI, instalacijski program za Pardus, dobio je LVM/RAID i UUID podršku.
  • KDE SC - Pardus 2011.1 comes with the latest KDE Software Compilation, KDE SC 4.6.5. The base packages also contains numerous backports and fixes which will improve the stability of your desktop experience significantly.
  • Kaptan - Kaptan, Pardus alat za prilagodbu radne površine, sada po izboru postavlja vašu sliku kao avatar u KDE.
  • NetworkManager - GNOME NetworkManager 0.8.4 is now the default networking backend in Pardus 2011.1. Users are now able to set up their HSPA/CDMA/VPN networks together with the already supported Ethernet and 802.11 WLAN networks.
  • GTK Oxygen style - All GTK applications are now rendered with Oxygen style thanks to the oxygen-gtk project. This brings a huge improvement to the user interface consistency.
  • LibreOffice - LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is now the default Office Suite in Pardus 2011.1.

3. Što je novo?

3.1. Dopune u Pardus tehnologiji

3.1.1. YALI

  • YALI, Pardus instalacijaki program,dobio je LVM/RAID i UUID podršku.
  • Grafičko sučelje je potpuno redizajnirano zbog poboljšanja upotrebljivosti.
  • Korisnici sada mogu odabrati između upravljačkih programa otvorenog koda te vlasničkih upravljačkih programa za svoje NVIDIAgrafičke kartice.

3.1.2. Prvo podizanje

  • Neke od postavki za koje se pitalo tijekom instalacije u Pardusu 2011 sada su premještene u drugi program koji se pokazuje jednom tijekom prvog podizanja nakon instalacije.

3.1.3. Kaptan

  • Kaptan, Pardus alat za prilagodbu radne površine, sada može dodati vaši fotografiju i postaviti je kaop avatar u KDE.
  • Korisnici sada mogu odabrati željene teme ikona koje će biti korištene u njihovom KDE radnom okruženju.
  • Grafičko sučelje je potpuno redizajnirano zbog poboljšanja upotrebljivosti.

3.1.4. Pisi

  • Pisi sada iznutra korisiti XZ kao zadani algoritam kompresije.
  • Imena binarnih pakete sada odražavaju arhitekturu i izdanje distribucije za koju je paket napravljen.
  • Učinjena su razna poboljšanja brzine u komponentama jezgre z bolje iskutvo upravljanja paketima.

3.1.5. Upravitelj paketa

  • Users are now able to vote for a package and see the screenshots of graphical applications shipped within packages.
  • The graphical user interface is redesigned to improve the usability. Various speed enhancements has been done for a better graphical package management experience.
  • A lot of KDE integration work has been done including the usage of KDE system tray.

3.2. Dopune u komponentama jezgre

  • Kernel - The Linux kernel shipped with Pardus 2011.1 is 2.6.37.6.
  • Plymouth - The bootsplash technology used in Pardus 2011 is updated to its latest version for a much stable experience.
  • Python - Python je dopunjen na 2.7.1.
  • Perl - Perl je dopunjen na 5.12.2.
  • X.org - xorg-server is updated to 1.9.5, improvements to the automatic driver configuration mechanism has been done by our developers.
  • TeX Live - TeX Live documentation stack is updated to 2011.

3.3. Iskustvo radne površine

3.3.1. Promjene u KDE radnom okruženju

Pardus 2011.1 comes with the latest KDE Software Compilation, KDE SC 4.6.5. The base packages also contains numerous backports and fixes which will improve the stability of your desktop experience significantly.
KDE SC 4.6.5 brings more performance and stability to the desktop experience. Support for WebKit, improved user experience for netbooks, a new high-speed cache allowing fast access to resources stored on disk are among the new features of this version.

3.3.2. Vizualna poboljšanja

All GTK applications are now rendered with Oxygen style thanks to the oxygen-gtk project. This brings a huge improvement to the user interface consistency.
Milky icon theme especially designed for Pardus is shipped within the installation image. Users can set this icon theme through Kaptan or System Settings.

3.3.3. Upravljanje mrežom

Besides switching to GNOME NetworkManager as the default networking backend, Pardus 2011.1 comes with a heavily improved and modified KDE Network Management tool and plasmoid to give their users a much more consistent networking experience.
With this new networking backend, users are now able to easily set up their Ethernet (With support for 802.1x authentication), 802.11 Wireless (With support for WEP/WPA/WPA2/WPA-EAP and 802.1x authentication), EDGE/3G/CDMA and VPN networks.

3.3.4. Zadani programi

3.3.4.1. Mozilla Firefox
Pardus 2011.1 comes with Mozilla Firefox 5.0 as the default web browser application. New features of this Firefox release include Firefox Panorama, application tabs, a redesigned extension manager, Jetpack extensions support, integration with Firefox Sync, and support for multitouch displays.
3.3.4.2. Clementine
Clementine, a cross-platform music player based on Amarok 1.4 is set as the default audio playing application as it is well maintained and lightweight.
3.3.4.3. SMPlayer
SMplayer, a Qt based frontend for MPlayer is set as the default Video/VCD/DVD playing application as it is well integrated with KDE desktop environment in terms of usability and graphical user interface and supports playing a wide range of video file types thanks to the MPlayer backend.
3.3.4.4. KsCD
KsCD, is set as the default Audio CD playing application.

3.3.5. Web preglednici

Iako nisu uključeni u insatalacijske slike,širok raspon web preglednika uključujući Chromium preglednik,Arora i Opera dostupni su za instalaciju kroz upravitelja paketa.

3.3.6. Postavke sustava

Panda (Pardus alternative driver administration) is a new System Settings module which allows users to change the driver between Open Source and proprietary one for NVIDIA and ATI display adapters.
Users are now able to set up their Bluetooth devices, Wacom tablets and Synaptics touchpads through System Settings.
Zadani jezik koji se koristi u cijelom sustavu sada se može promijeniti kroz sekciju postavki jezika.

3.3.7. Office Suite

LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is now the default Office Suite in Pardus 2011.1.

3.3.8. Igre

Pardus 2011.1 continues the tradition of giving importance to the quality of games offered to their users. The default installation now includes the board and logic games provided by the kdegames package.

3.4. Ostale promjene

Firefox gets new stable release rapidly. 5.0 release presents many new features like the new interface of the browser as well as plugin management new interface, tab grouping, bookmarks between computers, passwords and history matching, a new plug-in interface. HTML5, CSS3 and WebM technology supports are also most important features for the developers.

4.2. Memorija sustava (RAM)

Zahtjevi memorije sustava (RAM) za Pardus 2011.1 ovise o vašem dnevnom radu pa je teško odrediti gornju i donju granicu,ali mi preporučujemo najmanje 1GB memorije za ugodno iskustvo radne površine.

4.3. Zahtjevi prostora tvrdoga diska

4.3.1. Minimalni zahtjevi

Pardus 2011.1 zahtjeva najmanje 4GB slobodnog prostora tvrdog diska za uspješnu instalaciju.Zato jer ne nudimo prolagođeni odabir paketa tijekom instalacije,ova minimalna veličina sa primjenjuje na sve instalacije.

4.3.2. Preporučeni zahtjevi

Da biste učinkovito koristili Pardus 2011.1 da vam ne ponestane prostora na disku tijekom vašeg dnevnog rada,preporučujemo najmanje 10GB slobodnog prostora tvrdog diska.

4.3.3. Bilješka o swap prostoru

swap prostor se koristi uvijek kada vašem operativnom sustavu ponestane fizičke memorije.Ako mislite da je vaša fizička memorija dovoljno velikaza vašu dnevnu uporabu možete preskočiti stvaranje swap prostora tijekom instalacije.
Napominjemo particija diska rezervirana za swap prostor je potrebna ako želite hibernirati (Suspend to disk) Pardus.U suprotnom ta će finkcija biti onemogućena.
