Notas de Lanzamiento

1. Welcome to Pardus 2011.1 Dama Dama

Welcome to Pardus 2011.1 Dama Dama, the latest maintenance release of Pardus, a powerful, KDE-focused Linux distribution.

1.1. Qué es Pardus ?

Pardus es un proyecto de una distribución de GNU/Linux amigable al usuario, creado con código abierto, a libre distribución, y liecnciado pajo la licencia GPL (General Public License) , con varios mejoras en usabilidad, compatibilidad de hardware, estabilidad y seguridad.
El auspiciante (sponsor) del proyecto Pardus es el Centro de Investigación de Tecnologías Avanzadas de Informática y Seguridad Informática (BILGEM), un asociado del Concilio de Investigación Científica y Tecnológica de Turquía (TUBITAK).
La distribución Pardus se esfuerza para brindar a la gente una experiencia de computación estable, confiable y fácil de usar mediante la provisión de software de código abierto y popular. Además los desarrolladores trabajan arduamente para lograr una compatibilidad con una amplia gama de hardware.
El nombre del proyecto viene del nombre en latín (cientifico) del leopardo de Anatolia, Panthera pardus tulliana.

1.2. Como puedo probar Pardus ?

Para usar Pardus puede instalarlo en su computadora o ejecutando el sistema en la versión live (en vivo) sin modificar su disco duro. Puede bootear Pardus con un lector óptico o de un dispositivo de almacenamiento USB como discos externos o Pendrive.
Tanto la versión liva como la versión de instalación superan el tamaño de una CD común, por lo tanto si prefiere utilizar un medio óptico para bootear Pardus, debe grabar las imagenes en DVD.
Tenga en cuenta que debe asegurarse que en el orden de arranque en el BIOS el lector óptico o el dispositivo USB deben estar en primer posición para arrancar de ello tanto para la instalación como en modo vivo. En algunos sistemas se puede seleccionar el orden de dispositivos de arranque apretando una tecla (generalmente F8 para equipos de escritorio) durante el arranque.

2. Vista General

Pardus 2011.1 contiene muchas mejoras en relación a la versión anterior de Pardus 2011. Las siguientes secciones muestran una breve información sobre las modificaciones.
  • Kernel - 2.6.37.6 Linux kernel provides updated hardware support together with a hundreth of bug fixes.
  • YALI - YALI, the installer of Pardus, gained Rescue Mode which provides easy password and bootloader recovery.
  • KDE SC - Pardus 2011.1 le trae la última Compilación de KDE Software, KDE SC 4.6.5. EL paquete base además incluye varios códigos backports y correcciones para mejorar notablemente la estabilidad de su escritorio.
  • LibreOffice - LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is updated to the much-awaited 3.4 version which brings huge improvements over the older version.
  • Firefox - Firefox 5.0, the latest stable version of the most popular Open Source web browser, is now shipped with Pardus 2011.1.

3. Que novedades trae ?

3.1. Actualizaciones de tecnologías de Pardus

3.1.1. YALI

  • YALI, the installer of Pardus, gained system rescue menu that provides easy bootloader and password recovery.

3.1.2. Pisi

  • Some of the core components are now parallellized to increase the utilization of multi-core machines which means faster package operations and better package management experience.

3.1.3. QuickFormat

  • QuickFormat is a removable device formatting tool that can integrate to Dolphin. Now, it's very easy to format USB flash drives and card readers.

3.2. Actualizaciones en componentes internos

  • Kernel - The Linux kernel shipped with Pardus 2011.1 is 2.6.37.6.
  • Plymouth - The bootsplash technology used in Pardus 2011 is updated to its latest version for a much stable experience.
  • X.org - xorg-server is updated to 1.9.5.
  • TeX Live - TeX Live documentation stack is updated to latest 2011 version.

3.3. Experiencia de Escritorio

3.3.1. Modificaciones del entorno de escritorio KDE

Pardus 2011.1 comes with the latest KDE Software Compilation, KDE SC 4.6.5. The base packages also contains numerous backports and fixes which will improve the stability of your desktop experience significantly.
KDE Software Compilation, versioned 4.6.5, provide many new features in each of KDE's components including Plasma, Solid, Nepomuk, KWin and also introduces a brand new Bluetooth framework called BlueDevil allows easy device pairing and management.

3.3.2. Aplicaciones predeterminadas

3.3.2.1. Mozilla Firefox
Pardus 2011.1 comes with Mozilla Firefox 5.0 as the default web browser application. This version of Firefox introduces CSS animations, Do-Not-Track, improved support for HTML5, MathML, and SMIL, improved network, memory and JavaScript performance and fixes for several stability and security issues.
3.3.2.2. Clementine
Clementine, a cross-platform music player based on Amarok 1.4 is updated to latest stability and maintenance version 0.7.1.
3.3.2.3. SMPlayer
SMplayer, a Qt based frontend for MPlayer is set as the default Video/VCD/DVD playing application as it is well integrated with KDE desktop environment in terms of usability and graphical user interface and supports playing a wide range of video file types thanks to the MPlayer backend.
3.3.2.4. KsCD
KsCD, is set as the default Audio CD playing application.

3.3.3. Navegadores Web

Although not included in the installation images, a wide range of web browsers including Chromium browser, rekonq, Arora and Opera are available for installation through the package manager.

3.3.4. Configuración del sistema

The display configuration tool of the graphics card manufacturer is now recommended in the Display and Monitor module of the System Settings, if it suits better to your graphics card driver.

3.3.5. Suite Offimatica

LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is the default Office Suite in Pardus 2011.1. 3.4 version of LibreOffice introduces; improved text rendering, configurable color charts, improved thumbnailed HTML export of presentations, faster Flat ODF import/export and better integration desktop theme integration.

3.3.6. Other

Most wanted packages like Skype, a P2P VoIP client that allows you to make voice calls, and Wine, a software that allows you to run Microsoft Windows applications, are now included in Pardus 2011.1 package repopsitories.

4.2. Memoria del sistema (RAM)

Los requirimientos de memoria RAM de Pardus 2011.1 dependen del tipo de uso que le dan al sistema, y por lo tanto es dificil de determinar límites inferiores o superiores. Sin embargo recomendamos al mínimo 1GB de RAM para una experiencia agradable del escritorio.

4.3. DIsco rígido

4.3.1. Requisitos mínimos

Pardus 2011.1 requiere al menos 4GB de espacio en disco para uns instalación exitosa. Este valor de espacio está predeterminado por la cantidad de paquetes que se van a instalar, ya que durante de la instalación no se tiene la opción de elegir los paquetes a instalar.

4.3.2. Requirimientos recomendados

Para un uso eficiente de Pardus 2011.1 en su sistema sin llegar al límite de espacio en disco para su uso diario, recomendamos al menos 10GB de espacio disponible en disco.

4.3.3. Comentario sobre espacio Swap

El sistema usa el espacio de intercambio (Swap), cuando la memoria RAM del sistema ya no da abasto. Si cree que tiene suficiente memoria RAM para su uso diario, puede omitir la creación del espacio de intercambio (swap) durante la instalación.
Atención: Si desea utilizar la funcionalidad de hibernar (Suspend to disk) en Pardus se necesita la existencia del espacio de intercambio Pardus. Esta función queda deshabilitada si no existe el espacio de intercambio.
