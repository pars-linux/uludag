Release Notes

1. Welcome to Pardus 2011.1 Dama Dama

Welcome to Pardus 2011.1 Dama Dama, the latest maintenance release of Pardus, a powerful, KDE-focused Linux distribution.

1.1. Was ist Pardus?

Pardus ist eine nutzerfreundliche OpenSource GNU/Linux Distribution, die frei verfügbar ist und unter der GPL (General Public License) steht mit vielen nutzerfreundlichen Funktionen, ausgezeichneter Hardwarekompatibilität, herausragender Stabilität und Sicherheit.
Das Pardus Projekt ist finanziert durch das Center of Research for Advanced Technologies of Informatics and Information Security (BILGEM), welches der Scientific and Technological Research Council of Turkey (TUBITAK) angegliedert ist.
Das Ziel von Pardus ist, den Benutzern eine stabile, verlässliche, einfach zu nutzende und funktionsreiche Computererfahrung durch moderne und beliebte OpenSource Software zu bieten. Die Entwickler legen großen Wert darauf, dass Pardus auf einer breiten Basis von Hardware läuft.
Das Projekt ist nach dem wissenschaftlichen lateinischen Namen des Anatolischen Leopards, Panthera pardus tulliana benannt.

1.2. Wie kann ich Pardus probieren?

Sie können Pardus testen indem Sie es auf Ihren Computer installieren, oder Sie lassen die Live Version laufen, ohne dass die Festplatte verändert wird. Sie können Pardus von einem optischen Medium genauso wie von einem USB Speicher (ext. Festplatte oder USB-Stick) booten.
Falls Sie ein optisches Medium verwenden wollen, um Pardus zu booten, müssen Sie eine DVD nutzen um das Image zu brennen, da sowohl die Installations- als auch das Live-Image die Größe einer regulären CD übersteigen.
Um von dem Installations- oder Livemedium zu booten sollte die Bootreihenfolge im BIOS Ihres Computers so eingestellt werden, dass zuerst von dem optischen Laufwerk oder vom USB Gerät gebootet wird. Bei neueren Systemen kann die Bootreihenfolge auch durch Drücken einer Taste (bei Desktop Systemen normalerweise F8) während des Bootvorgangs geändert werden.

2. Übersicht

Pardus 2011.1 enthält gegenüber der vorherigen Pardus Version 2011 zahlreiche Verbesserungen. Die folgenden Abschnitte erlauben einen kurzen Überblick über die Änderungen.
  • Kernel - 2.6.37.6 Linux kernel provides updated hardware support together with a hundreth of bug fixes.
  • YALI - YALI, the installer of Pardus, gained Rescue Mode which provides easy password and bootloader recovery.
  • KDE SC - Pardus 2011.1 wird mit der neuesten KDE Software Compilation, KDE SC 4.6.5 ausgeliefert. Um die Stabilität des Desktops signifikant zu steigern beinhalten die Basispakete zusätzlich eine gewisse Anzahl von Portierungen und Fehlerkorrekturen.
  • LibreOffice - LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is updated to the much-awaited 3.4 version which brings huge improvements over the older version.
  • Firefox - Firefox 5.0, the latest stable version of the most popular Open Source web browser, is now shipped with Pardus 2011.1.

3. Was ist neu?

3.1. Updates der Pardus Technologien

3.1.1. YALI

  • YALI, the installer of Pardus, gained system rescue menu that provides easy bootloader and password recovery.

3.1.2. PiSi

  • Some of the core components are now parallellized to increase the utilization of multi-core machines which means faster package operations and better package management experience.

3.1.3. QuickFormat

  • QuickFormat is a removable device formatting tool that can integrate to Dolphin. Now, it's very easy to format USB flash drives and card readers.

3.2. Aktualisierungen der Kernkomponenten

  • Kernel - The Linux kernel shipped with Pardus 2011.1 is 2.6.37.6.
  • Plymouth - The bootsplash technology used in Pardus 2011 is updated to its latest version for a much stable experience.
  • X.org - xorg-server is updated to 1.9.5.
  • TeX Live - TeX Live documentation stack is updated to latest 2011 version.

3.3. Desktoperlebnis

3.3.1. Änderungen der KDE Desktopumgebung

Pardus 2011.1 wird mit der neuesten KDE Software Compilation, KDE SC 4.6.5 ausgeliefert. Die Basispakete beinhalten zusätzlich eine gewisse Anzahl von Portierungen und Fehlerkorrekturen, um die Stabilität des Desktops signifikant zu steigern.
KDE Software Compilation, versioned 4.6.5, provide many new features in each of KDE's components including Plasma, Solid, Nepomuk, KWin and also introduces a brand new Bluetooth framework called BlueDevil allows easy device pairing and management.

3.3.2. Standard Programme

3.3.2.1. Mozilla Firefox
Pardus 2011.1 comes with Mozilla Firefox 5.0 as the default web browser application. This version of Firefox introduces CSS animations, Do-Not-Track, improved support for HTML5, MathML, and SMIL, improved network, memory and JavaScript performance and fixes for several stability and security issues.
3.3.2.2. Clementine
Clementine, a cross-platform music player based on Amarok 1.4 is updated to latest stability and maintenance version 0.7.1.
3.3.2.3. SMPlayer
SMplayer, eine auf Qt basierende Oberfläche für MPlayer ist das Standardprogramm für das Abspielen von Video/VCD und DVD. Das Programm ist sehr gut in die KDE Desktopumgebung integriert und bietet neben einer nutzerfreudlichen Bedienung dank dem MPlayer Unterbau eine sehr gute Unterstützung für viele Videoformate.
3.3.2.4. KsCD
KsCD ist das Standardprogramm zum Abspielen von Audio CDs.

3.3.3. Web Browser

Obwohl nicht im Installationsmedium integriert, ist eine große Anzahl von Webbrowsern verfügbar. Unter anderem kann mit dem Paket-Manager Chromium, rekonq, Arora und Opera installiert werden.

3.3.4. Systemeinstellungen

The display configuration tool of the graphics card manufacturer is now recommended in the Display and Monitor module of the System Settings, if it suits better to your graphics card driver.

3.3.5. Office Suite

LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is the default Office Suite in Pardus 2011.1. 3.4 version of LibreOffice introduces; improved text rendering, configurable color charts, improved thumbnailed HTML export of presentations, faster Flat ODF import/export and better desktop theme integration.

3.3.6. Other

Most wanted packages like Skype, a P2P VoIP client that allows you to make voice calls, and Wine, a software that allows you to run Microsoft Windows applications, are now included in Pardus 2011.1 package repopsitories.

4.2. Arbeitsspeicher (RAM)

Die Anforderung von Pardus 2011.1 nach Arbeitsspeicher (RAM) hängt stark davon ab, welche arbeiten Sie mit dem System durchführen wollen. Deshalb ist es sehr schwer eine Unter- und Obergrenze zu definieren. Für ein ruckelfreies System empfehlen wir aber wenigstens 1GB Arbeitsspeicher.

4.3. Benötigter Festplattenspeicher

4.3.1. Minimale Anforderung

Für eine erfolgreiche Installation benötigt Pardus 2011.1 zumindest 4GB freien Speicherplatz. Da wir während der Installation keine Paketauswahl anbieten, ist diese Mindestgröße für alle Installationen gültig.

4.3.2. Empfohlene Anforderung

Um mit Pardus 2011.1 effektiv arbeiten zu können, ohne ständig durch mangelnden Speicherplatz in Bedrängnis zu kommen, empfehlen wir mindestens 10GB freien Speicherplatz.

4.3.3. Anmerkung zum Swap Speicher

Der Swap Speicher wird benutzt, wenn das System mehr Arbeitsspeicher nutzen möchte, als vorhanden ist. Falls Sie der Meinung sind, dass der verfügbare Arbeitsspeicher groß genug für die tägliche Arbeit ist, können Sie die Erstellung der Swap Partition während der Installation überspringen.
Beachten Sie, dass eine Swap-Partition notwendig ist, falls Sie bei Pardus den Ruhezustand (Suspend to disk bzw. Hibernate) nutzen wollen. Andernfalls ist die Funktionalität nicht möglich.
