Release Notes

1. Welcome to Pardus

Welcome to Pardus 2011, another great Pardus release from Pardus developers, translators, artists and documentation writers.

We kindly invite you to visit http://www.pardus.org.tr/eng for more information.

1.1. What is Pardus?

Pardus is a Linux distribution developed as a product of the Pardus Project, by National Research Institute of Electronics and Cryptology (UEKAE), which is under the Scientific and Technological Research Council of Turkey (TÜBİTAK) and by volunteered contributors. It is named after the Latin (and scientific) name for the Anatolian leopard, Panthera pardus tulliana.

Pardus consists of a Linux based operating system, opensource applications and related data files necessary for desktop and server environments, aiming to provide users stable, reliable, easy to use and feature rich computer experience.

1.2. How Can I Use Pardus?

Pardus comes with two flavours, Kurulan (which installs Pardus to your computer) or Çalışan (live media, running on the media only, slower but does not install on your computer) and can be started from a DVD / External storage (USB stick for example) or can be installed to internal harddisk depending on your preference.

To start your Pardus experience, you need to insert the related media in your computer, reboot the system and set up your computer BIOS to boot from the Pardus media.

You can use http://bugs.pardus.org.tr, (Pardus bug tracking system) to report a bug, send a bug fix or suggest anything related to Pardus. We would like to thank our users and developers for their bug reports, fixes, new package or feature suggestions, which makes up Pardus what it is.

2. What is New?

2.1. Updates in Pardus Technologies

  • YALI is easy, stable and fast with its renewed interface, LVM/RAID support and disc partitioning screen.
  • The boot splash system has completely renewed. 2011 release offers richer theme support and more capable animations by using Plymouth technology.
  • Package scoring and screenshot features which are the users' ambitions expected for 2011, now supported. Screenshot from the web interface and voting will be done soon.
  • As well as the some innovations and effects on The Package Manager interfaces, faster package updates and installation can be performed with the improvements in Pisi infrastructure.

2.3. KDE 4 Desktop Enviroment

  • Pardus 2011 Beta release, coming with latest release of KDE desktop environment, KDE 4.5.2. With this release the backbone of desktop environment Plasma, the file manager (Dolphin) and desktop effects has become more stable and faster with a lot of fixes.
  • The applications which are frequently used by desktop users such as Amarok (multimedia player), Skanlite (browser interface), KTorrent (torrent interface), SMPlayer (video player) can be installed with Beta version.

2.4. Updated Components

  • Firefox gets new stable release rapidly. 4.0b8pre release presents many new features like the new interface of the browser as well as plugin management new interface, tab grouping, bookmarks between computers, passwords and history matching, a new plug-in interface. HTML5, CSS3 and WebM technology supports are also most important features for the developers.
  • The alternative web browsers such as Chromium and Rekonq are ready for the users which prefer the alternative Internet browsers other than Firefox.
  • Those who use office applications extensively, can try LibreOffice, OpenOffice's continuation project, with improvements like Turkish spell checking, add-ons and compatibility which are prepared by Pardus developers.
  • User can now configure Wacom tablet, Touchpad and Bluetooth devices using System Settings. 2.6.36-rc7 version of Linux kernel provides support for the users with recent hardware.

3. Installation Requirements

You can run Pardus on an average x86 compatible hardware. We suggest to use at least 512 MB RAM (memory) and a 1400 MHz processor to get a good performance. Minimum and recommended hardware requirements for Pardus 2011 are mentioned below:
  • 256 MB memory (1GB or more recommended)
  • 450 MHz Intel or AMD processor (1000 MHz recommended)
  • At least 4 GB of free hard disk space (10 GB recommended)
